## Aspose.Cells для .NET

{{% alert color="primary" %}}

![Логотип продукта Aspose.Cells для .NET](../home_3.png)

Aspose.Cells for .NET — это библиотека для программирования электронных таблиц, которая позволяет разработчикам программного обеспечения создавать, читать, преобразовывать, обновлять и печатать файлы электронных таблиц из своих собственных приложений. Aspose.Cells для .NET, сочетающий в себе API и элементы управления с графическим интерфейсом, ускоряет программирование и преобразование Microsoft Excel®. Aspose.Cells для .NET поддерживает популярные форматы файлов электронных таблиц (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS), которые ваша компания использует каждый день. Он также позволяет экспортировать файлы Excel в PDF, XPS, HTML, MHTML, обычный текст и популярные форматы изображений, включая TIFF, JPG, PNG, BMP и SVG.

## Aspose.Cells для C++

{{% alert color="primary" %}}

![Логотип продукта Aspose.Cells для C++](../home_6.png)

Aspose.Cells for C++ — это нативная библиотека C++, которая позволяет разработчикам C++ программно создавать электронные таблицы, управлять ими и преобразовывать их, не требуя автоматизации Office или приложений Microsoft Excel. Aspose.Cells для C++ поддерживает форматы электронных таблиц, такие как MS Excel 97-2003 (XLS), MS Excel 2007-2013/2016 (XLSX, XLSM, XLSB и т. д.), Open Office XML и другие форматы. Вы также можете отображать изображения из рабочих листов и конвертировать электронные таблицы MS Excel в формат PDF. API также позволяет создавать, манипулировать и отображать диаграммы и фигуры. Кроме того, расчеты по формулам — еще один хороший аспект, предоставляемый компонентом, и вы можете его использовать.
