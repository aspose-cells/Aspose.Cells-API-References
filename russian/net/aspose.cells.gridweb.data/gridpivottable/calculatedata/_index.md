## GridPivotTable.CalculateData method

Вычисляет данные сводной таблицы в ячейки.

```csharp
public void CalculateData()
```

### Примечания

Cell.Value в сводном диапазоне не может вернуть правильный результат, если метод не был вызван . Этот метод вычисляет данные с помощью внутреннего сводного кеша, а не исходного источника данных. Поэтому, если источник данных изменился, сначала вызовите метод RefreshData().
