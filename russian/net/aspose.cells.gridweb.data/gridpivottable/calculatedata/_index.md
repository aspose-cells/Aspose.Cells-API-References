public void CalculateData()

### Примечания

Cell.Value в диапазоне сводки не может вернуть правильный результат, если метод не был вызван. Этот метод вычисляет данные с помощью внутреннего кэша сводки, а не исходного источника данных. Поэтому, если источник данных изменен, вызовите RefreshData () метод first.
