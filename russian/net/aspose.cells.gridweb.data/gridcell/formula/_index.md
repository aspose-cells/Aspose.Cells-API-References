## GridCell.Formula property

Получает или задает формулуCell .

```csharp
public string Formula { get; set; }
```

### Примечания

Строка формулы всегда начинается со знака равенства (=). И, пожалуйста, всегда используйте запятую (,) в качестве разделителя параметров, например "=СУММ(A1, E1, H2)".

Пользователь может установить любую формулу в файле конструктора рабочей книги. Aspose.Cells сохранит все формулы. Если пользователь использует это свойство для установки формулы в ячейку, поддерживается большая часть встроенных функций Workbook . И это еще не все. Если у вас есть особые потребности во встроенных функциях Workbook, сообщите нам об этом.

### Примеры

```csharp
[C#]
	cell.Formula = "=SUM(A1:C3) + E6*2";
[Visual Basic]
	cell.Formula = "=SUM(A1:C3) + E6*2"
```
