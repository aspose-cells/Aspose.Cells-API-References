### Возвращаемое значение

Значение для создания списка этой проверки для указанной ячейки. Если список ссылается на диапазон, то возвращаемое значение будет объектом ReferredArea; В противном случае возвращаемое значение может быть нулевым, объектом[] или простым объектом.
