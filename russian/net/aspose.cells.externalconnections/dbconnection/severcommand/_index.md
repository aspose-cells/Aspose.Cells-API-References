## DBConnection.SeverCommand property

Указывает вторую текстовую строку команды, которая сохраняется, когда используются поля страницы на основе сервера сводной таблицы. Для подключений ODBC serverCommand обычно является более широким запросом, чем команда (в первом нет предложения WHERE). На основе этих двух команд (Command и ServerCommand) можно заполнить пользовательский интерфейс параметров и построить параметризованные запросы

public string SeverCommand { get; set; }
