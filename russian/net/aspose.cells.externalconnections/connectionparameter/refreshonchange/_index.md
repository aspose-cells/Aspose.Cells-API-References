## ConnectionParameter.RefreshOnChange property

Флаг, указывающий, должен ли запрос автоматически обновляться при изменении содержимого ячейки , содержащей значение параметра. Если true, то внешние данные обновляются с использованием нового значения параметра при каждом изменении. Если false, то внешние данные обновляются только по запросу пользователя, или какое-то другое событие запускает обновление (например, открытие рабочей книги).

```csharp
public bool RefreshOnChange { get; set; }
```
