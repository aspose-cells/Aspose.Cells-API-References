## Insert(int) {#insert_1}

Вставляет рабочий лист в коллекцию WorksheetCollection по указанному индексу. Количество строк этого рабочего листа равно 256, количество столбцов равно 32. Имя рабочего листа будет сгенерировано методом Add().

```csharp
public void Insert(int index)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | Int32 | Отсчитываемый от нуля индекс, по которому должен быть вставлен рабочий лист. |

Добавлен объект рабочего листа возврата.
