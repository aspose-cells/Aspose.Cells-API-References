## ImageOrPrintOptions.DefaultFont property

Когда символы в Excel имеют формат Unicode и для них не установлен правильный шрифт в стиле ячейки, Они могут отображаться как блоки в pdf, изображении. Установите DefaultFont, например MingLiu или MS Gothic, для отображения этих символов. Если это свойство не установлено, Aspose.Cells будет использовать системный шрифт по умолчанию для отображения этих символов Юникода.

```csharp
public string DefaultFont { get; set; }
```
