## ImageOrPrintOptions.Quality property

Получает или задает значение, определяющее качество сгенерированных изображений для применения только при сохранении страниц в`JPEG` формат. Значение по умолчанию: 100 .

```csharp
public int Quality { get; set; }
```

### Примечания

Действует только при сохранении в формате JPEG. Значение должно быть от 0 до 100. Значение по умолчанию: 100.
