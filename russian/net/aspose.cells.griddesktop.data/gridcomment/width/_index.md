## GridComment.Width property

Получить или установить ширину комментария. Значение по умолчанию: 140. Вводимое значение должно находиться в диапазоне от 20 до 1000. Будет установлено значение 20, если будет введено значение меньше 20. Будет установлено значение 1000, если будет введено значение больше 1000.

public int Width { get; set; }
