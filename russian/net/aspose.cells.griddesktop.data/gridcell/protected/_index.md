## GridCell.Protected property

Указывает, защищена ли ячейка. Если значение равно "true", пользователь не может изменять ячейку через пользовательский интерфейс. Этот атрибут не имеет ничего общего со свойством Style.CellLocked и не будет сохранен в файл при данные сетки экспортированы. Значение по умолчанию "false".

public bool Protected { get; set; }
