## GridCell.GetFontColor method

Получает цвет шрифта ячейки. При изменении цвета следует вызвать метод "SetFontColor", для установки цвета шрифта ячейки.

```csharp
public Color GetFontColor()
```

### Возвращаемое значение

вернуть цвет шрифта ячейки.

### Примеры

```csharp
[C#]
GridCell cell = gridDesktop1.GetActiveWorksheet().Cells[0, 0];
Font font = new Font("Courier New", 8, FontStyle.Italic);
cell.SetFont(font);
Color color = cell.GetFontColor();
color = Color.Black;
cell.SetFontColor(color);

[Visual Basic]

```
