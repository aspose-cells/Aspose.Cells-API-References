## Chart.PivotSource property

Источником являются данные сводной таблицы. Если PivotSource не пуст, диаграмма является сводной диаграммой.

```csharp
public string PivotSource { get; set; }
```

### Примечания

Если сводная таблица «Сводная таблица1» на рабочем листе «Лист1» в файле «Book1.xls». Сводная таблица может быть «[Книга1.xls]Лист1!Сводная таблица1», если диаграмма и сводная таблица не находятся в одной книге. Если установить это свойство, предыдущая настройка источника данных будет потеряна.
