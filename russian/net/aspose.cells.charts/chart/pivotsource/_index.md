## Chart.PivotSource property

Источником являются данные сводной таблицы. Если PivotSource не пуст, диаграмма является PivotChart.

```csharp
public string PivotSource { get; set; }
```

### Примечания

Если сводная таблица "Сводная таблица1" на рабочем листе "Лист1" в файле "Book1.xls". PivotSource может быть "[Book1.xls]Sheet1!PivotTable1", если диаграмма и сводная таблица не находятся в одной книге. Если установить это свойство, предыдущая настройка источника данных будет потеряна.
