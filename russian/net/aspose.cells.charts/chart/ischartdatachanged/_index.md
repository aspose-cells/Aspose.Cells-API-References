## Chart.IsChartDataChanged method

Определяет, изменился ли источник данных диаграммы.

```csharp
public bool IsChartDataChanged()
```

### Возвращаемое значение

Возвращает true, если диаграмма изменилась, в противном случае возвращает false

### Примечания

Метод обнаруживает изменения в источнике данных диаграммы перед преобразованием диаграммы в формат изображения. При первом вызове Chart.toImage исходные данные диаграммы (например, XValuesParseData, ValuesParseData) будут записаны. Перед повторным вызовом метода Chart.toImage , вызовите метод IsChartDataChanged, чтобы проверить, нуждается ли Chart в повторном рендеринге.
