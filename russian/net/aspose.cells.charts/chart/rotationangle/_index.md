## Chart.RotationAngle property

Представляет поворот представления трехмерной диаграммы (поворот области графика вокруг оси Z в градусах).

```csharp
public int RotationAngle { get; set; }
```

### Примечания

Значение этого свойства должно быть от 0 до 360, за исключением трехмерных гистограмм, где значение должно быть от 0 до 44. Значение по умолчанию — 20. Применяется только к трехмерным диаграммам.
