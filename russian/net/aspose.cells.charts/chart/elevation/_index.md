## Chart.Elevation property

Представляет высоту трехмерной карты в градусах.

```csharp
public int Elevation { get; set; }
```

### Примечания

Высота карты — это высота, на которой вы просматриваете карту, в градусах. По умолчанию для большинства типов диаграмм установлено значение 15. Значение этого свойства должно быть между -90 и 90, за исключением трехмерных гистограмм, где оно должно быть между 0 и 44.
