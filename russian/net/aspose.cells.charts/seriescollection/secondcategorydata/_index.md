## SeriesCollection.SecondCategoryData property

Получает или задает диапазон значений оси второй категории. Это может быть диапазон ячеек (например, "d1:e10"), или последовательность значений (например, "{2,6,8,10}"). Действует только тогда, когда некоторые серии AS строятся по второй оси.

public string SecondCategoryData { get; set; }
