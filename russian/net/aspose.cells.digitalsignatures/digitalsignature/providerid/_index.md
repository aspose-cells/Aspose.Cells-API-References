## DigitalSignature.ProviderId property

Указывает идентификатор класса поставщика подписи. Значение по умолчанию — Пусто (все нули) Guid.

```csharp
public Guid ProviderId { get; set; }
```

### Примечания

Поставщик криптографических услуг (CSP) представляет собой независимый программный модуль, который фактически выполняет криптографические алгоритмы для аутентификации, кодирование и шифрование. Microsoft Office резервирует значение {00000000-0000-0000-0000-0000000000000} для поставщика подписи по умолчанию, и {000CD6A4-0000-0000-C000-000000000046} для поставщик подписи в Восточной Азии. GUID дополнительно установленного провайдера следует получить из документации, поставляемой с провайдером.
