## Config.IslimitShapeOrImage property

Устанавливает, следует ли ограничивать общее количество отображаемых форм/изображений внутри одного рабочего листа, если установлено значение true, GridJs будет ограничивать общее отображаемое количество фигур/ размер изображения внутри одного рабочего листа для MaxShapeOrImageCount значение по умолчанию равно true
