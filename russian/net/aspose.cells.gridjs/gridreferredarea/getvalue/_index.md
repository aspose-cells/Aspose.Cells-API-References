## GridReferredArea.GetValue method

Получает значение ячейки с заданным смещением от левого верхнего угла этой области.

```csharp
public object GetValue(int rowOffset, int colOffset)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| rowOffset | Int32 | смещение строки от начальной строки этой области |
