## PivotPageFields.AddIdentify method

Устанавливает, какую метку элемента в каждом поле страницы использовать для идентификации диапазона данных. Значение pageItemIndex.Length должно быть равно PageFieldCount, поэтому сначала добавьте поле страницы.

```csharp
public void AddIdentify(int rangeIndex, int[] pageItemIndex)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| rangeIndex | Int32 | Индекс диапазона данных консолидации. |
| pageItemIndex | Int32[] | Индекс элемента страницы в каждом поле страницы. pageItemIndex[2] = 1 означает, что второй элемент в третьем поле используется для идентификации этого диапазона. pageItemIndex[1] = -1 означает отсутствие элемента во втором поле для идентификации этого диапазона и MS автоматически создаст "пустой" элемент во втором поле для идентификации этого диапазон. |
