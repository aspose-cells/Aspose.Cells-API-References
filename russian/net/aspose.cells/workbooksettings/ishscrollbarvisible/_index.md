## WorkbookSettings.IsHScrollBarVisible property

Получает или задает значение, указывающее, будет ли сгенерированная электронная таблица содержать горизонтальную полосу прокрутки.

```csharp
public bool IsHScrollBarVisible { get; set; }
```

### Примечания

Значение по умолчанию верно.

### Примеры

Следующий код делает горизонтальную полосу прокрутки невидимой для электронной таблицы.

```csharp
[C#]
// Скрыть горизонтальную полосу прокрутки файла Excel.
settings.IsHScrollBarVisible = false;

[Visual Basic]
' Скрыть горизонтальную полосу прокрутки файла Excel.
settings.IsHScrollBarVisible = False
```
