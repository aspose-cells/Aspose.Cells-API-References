## WorkbookSettings.IsVScrollBarVisible property

Получает или задает значение, указывающее, будет ли сгенерированная электронная таблица содержать вертикальную полосу прокрутки.

```csharp
public bool IsVScrollBarVisible { get; set; }
```

### Примечания

Значение по умолчанию — true.

### Примеры

Следующий код делает вертикальную полосу прокрутки невидимой для электронной таблицы.

```csharp
[C#]
  // Скрыть вертикальную полосу прокрутки файла Excel.
settings.IsVScrollBarVisible = false;

[Visual Basic]
' Hide the vertical scroll bar of the Excel file.
settings.IsVScrollBarVisible = False
```
