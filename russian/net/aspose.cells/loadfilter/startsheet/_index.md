## LoadFilter.StartSheet method

Подготавливает параметры фильтра перед загрузкой данного листа. Пользовательская реализация LoadFilter может изменить LoadDataFilterOptions здесь , чтобы указать, как загружать данные для этого листа.

```csharp
public virtual void StartSheet(Worksheet sheet)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| sheet | Worksheet | Загружаемый рабочий лист. Для данного объекта рабочего листа здесь можно использовать только несколько свойств, поскольку большинство данных и свойств не загружены. Доступны следующие свойства: Name, Index, VisibilityType. |
