## LoadFilter.StartSheet method

Подготавливает параметры фильтра перед загрузкой данного рабочего листа. Пользовательская реализация LoadFilter может изменить здесь LoadDataFilterOptions , чтобы указать, как загружать данные для этого рабочего листа.

```csharp
public virtual void StartSheet(Worksheet sheet)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| sheet | Worksheet | Загружаемый лист. Здесь можно использовать только несколько свойств для данного объекта рабочего листа потому что большинство данных и свойств не загружены. Доступны следующие свойства: Name, Index, VisibilityType |
