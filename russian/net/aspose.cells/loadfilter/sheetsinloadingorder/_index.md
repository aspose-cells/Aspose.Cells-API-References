## LoadFilter.SheetsInLoadingOrder property

Указывает листы (индексы) и порядок загрузки. Значение по умолчанию равно null, что означает загрузку всех листов в порядке по умолчанию в файле шаблона. Если не null и в возвращаемом массиве нет индекса листа, то лист не будет загружен.
