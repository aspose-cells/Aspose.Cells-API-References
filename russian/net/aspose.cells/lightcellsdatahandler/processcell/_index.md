## LightCellsDataHandler.ProcessCell method

Начинает обрабатывать одну ячейку.

```csharp
public bool ProcessCell(Cell cell)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| cell | Cell | Объект ячейки, который обрабатывается в данный момент |

### Возвращаемое значение

нужно ли эту ячейку сохранять в модели ячеек текущего листа. Обычно должно быть ложно, чтобы все ячейки не оставались в памяти после обработки, а затем память сохранялась. Для некоторых специальных целей, таких как пользователь должен получить доступ к некоторым ячейкам позже после обработки всей книги, пользователь может заставить этот метод возвращать значение true, чтобы сохранить эти специальные ячейки в модели ячеек и получить к ним доступ позже с помощью API, таких как Cells [строка, столбец]. Однако для хранения данных ячеек в модели ячеек потребуется больше памяти, и если все ячейки будут сохранены, то чтение файла шаблона в режиме LightCells станет таким же, как и чтение в обычном режиме.

### Примечания

Он будет вызываться после того, как будут прочитаны данные одной ячейки.
