## LightCellsDataHandler.ProcessCell method

Начинает обрабатывать одну ячейку.

```csharp
public bool ProcessCell(Cell cell)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| cell | Cell | Объект ячейки, который обрабатывается в данный момент |

### Возвращаемое значение

нужно ли хранить эту ячейку в модели ячеек текущего листа. Обычно это должно быть ложно, чтобы все ячейки не сохранялись в памяти после обработки, а затем память сохранялась. Для какой-то специальной цели, такой как доступ пользователя некоторые ячейки позже, после обработки всей рабочей книги, пользователь может заставить этот метод возвращать значение true, чтобы сохранить эти специальные ячейки в модели ячеек и получить к ним доступ позже с помощью API, таких как ячейки [строка, столбец]. Однако сохранение данных ячеек в ячейках модели потребуется больше памяти, и если все ячейки будут сохранены, то чтение шаблона file в режиме LightCells станет таким же, как и чтение в обычном режиме.

### Примечания

Будет вызываться после того, как будут прочитаны данные одной ячейки.
