## Style.IndentLevel property

Представляет уровень отступа для ячейки или диапазона. Может быть только целым числом от 0 до 250.

```csharp
public int IndentLevel { get; set; }
```

### Примечания

Если для типа горизонтального выравнивания текста установлено значение, отличное от левого или правого, уровень отступа будет сброшен на ноль.
