## Style.CultureCustom property

Получает и задает строку шаблона, зависящую от языка и региональных параметров, для числового формата. Если для этого объекта не задан числовой формат, будет возвращено значение null. Если числовой формат является встроенным, будет возвращена строка шаблона, соответствующая встроенному номеру.

```csharp
public string CultureCustom { get; set; }
```

### Примечания

Для встроенного числового формата как содержимое шаблона (например, один встроенный формат даты "m/d/y" для некоторых языков, , но для некоторых других языков он становится "d/m/y"), так и спецификатор формата (например, некоторые локали используют символ, отличный от «y», для представления части года для форматирования даты) зависят от культуры; для пользовательского пользовательского формата изменяются только спецификаторы формата в соответствии с культурой, другие части шаблона форматирования не будет изменено.
