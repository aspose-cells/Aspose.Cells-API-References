## Cells.MultiThreadReading property

Получает или задает, должна ли модель данных ячеек поддерживать многопоточное чтение. Значение по умолчанию этого свойства - false.

```csharp
public bool MultiThreadReading { get; set; }
```

### Примечания

Если имеется несколько потоков для одновременного чтения объектов Row/Cell в этой коллекции, это свойство должно быть установлено как true, иначе может быть получен неожиданный результат. Поддержка многопоточного чтения может снизить производительность доступа к объектам Row/Cell из этой коллекции. Обратите внимание, что некоторые функции не могут поддерживать многопоточное чтение, например, значения форматирования (от[`StringValue`](../../cell/stringvalue),[`DisplayStringValue`](../../cell/displaystringvalue), .и т.д.). Таким образом, даже если для этого свойства задано значение true, эти API все равно могут давать неожиданный результат для многопоточного чтения.
