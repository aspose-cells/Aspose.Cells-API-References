## Protection.AllowDeletingColumn property

Указывает, разрешено ли удаление столбцов на защищенном листе.

```csharp
public bool AllowDeletingColumn { get; set; }
```

### Примечания

Столбцы, содержащие удаляемые ячейки, должны быть разблокированы, когда лист защищен, и опция «Выбрать разблокированные ячейки» должна быть включена.
