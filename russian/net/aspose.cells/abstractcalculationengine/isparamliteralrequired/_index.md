## AbstractCalculationEngine.IsParamLiteralRequired property

Указывает, нужен ли этому движку буквальный текст параметра при выполнении вычислений. Значение по умолчанию — false.

```csharp
public virtual bool IsParamLiteralRequired { get; }
```

### Примечания

Если для этого пользовательского механизма вычислений требуется буквальный текст параметра, потребуется больше стеков для кэширования буквального текста параметров, а метод Calculate() может вызываться рекурсивно для вычисления значения параметра. Обычно буквальный текст не требуется для вычисления формул и этот метод должен возвращать false для большинства реализаций, чтобы повысить производительность.
