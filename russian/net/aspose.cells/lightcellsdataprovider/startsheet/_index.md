## LightCellsDataProvider.StartSheet method

Начинает сохранять рабочий лист.

```csharp
public bool StartSheet(int sheetIndex)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| sheetIndex | Int32 | индекс текущего листа для сохранения. |

### Возвращаемое значение

true, если этот провайдер предоставит данные для данного листа; false, если данный лист должен использовать свою обычную модель данных (ячейки).

### Примечания

Будет вызываться в начале сохранения листа при сохранении книги. Если провайдеру нужно обратиться к

```csharp
sheetIndex
```

Later в методе startRow(Row) или startCell(Cell), то есть, если процессу необходимо знать, какой рабочий лист обрабатывается, реализация должна сохранить

```csharp
sheetIndex
```

значение здесь.
