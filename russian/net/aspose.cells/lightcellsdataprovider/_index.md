### Примечания

При сохранении книги в этом режиме[`StartSheet`](./startsheet)будет проверяться при сохранении каждого листа в книге. Для одного листа, если[`StartSheet`](./startsheet)дает true, то все данные и свойства строк/ячеек этого листа быть сохраненным будет обеспечен реализацией этого интерфейса. Во-первых,[`NextRow`](./nextrow)будет вызываться для получения индекса следующей строки для сохранения. Если возвращается допустимый индекс строки (индекс строки должен быть в порядке возрастания, чтобы строки были сохранены), тогда объект Row, представляющий эту строку, будет предоставлен для реализации чтобы установить его свойства с помощью[`StartRow`](./startrow). Для одной строки сначала будет проверяться[`NextCell`](./nextcell). Если будет возвращен допустимый индекс столбца (индекс столбца должен быть в порядке возрастания для сохранения всех ячеек одной строки), , то объект Cell, представляющий эту ячейку, будет предоставлен для реализации, чтобы установить ее данные и свойства[`StartCell`](./startcell). После того, как данные этой ячейки будут установлены, эта ячейка будет сохранена непосредственно в сгенерированном файле электронной таблицы, а следующая ячейка будет проверена и обработана.
