## LightCellsDataProvider.StartRow method

Начинает сохранять данные одной строки.

```csharp
public void StartRow(Row row)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| row | Row | Объект строки для реализации для заполнения данных. Его индекс строки — это возвращаемое значение последнего вызова[`NextRow`](../nextrow). Если строка была инициализирована во внутренней модели ячеек, будет использован существующий объект строки. В противном случае для реализации для заполнения данных будет использоваться временный объект Row. |

### Примечания

Будет вызываться в начале сохранения данных строки и ее ячеек. Если текущая строка имеет некоторые пользовательские свойства, такие как высота, стиль и т. д., реализация должна установить эти свойства для данного объекта Row здесь.
