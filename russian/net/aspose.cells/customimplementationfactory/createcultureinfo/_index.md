### Возвращаемое значение

### Примечания

Эта реализация полезна для ситуаций: 1. Некоторые культуры могут не поддерживаться пользовательской средой, и создание необходимой информации о культуре с заданным идентификатором может вызвать Исключение. Чтобы избежать исключения, пользователь может переопределить этот метод, чтобы предоставить действительный экземпляр CultureInfo для неподдерживаемого экземпляра. 2. Пользователь может захотеть указать некоторые пользовательские свойства для некоторых культур, чтобы получить ожидаемый результат форматирования. Для этой цели пользователь может переопределить этот метод, чтобы предоставить экземпляру CultureInfo определенные пользователем свойства. Обратите внимание, что свойство UseUserOverride возвращаемого экземпляра CultureInfo может повлиять на отформатированный результат. Если оно равно false, некоторые свойства возвращенного экземпляра CultureInfo могут быть переопределены нашим встроенным механизмом форматирования в соответствии с требованиями форматирования различных сценариев. Если это правда, мы не будем изменять какие-либо его свойства и будем использовать его для непосредственного форматирования значений. Итак, если пользователь указал пользовательские свойства для возвращенного экземпляра CultureInfo, убедитесь, что его UseUserOverride имеет значение true.
