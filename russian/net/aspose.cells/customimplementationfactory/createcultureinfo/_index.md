### Примечания

Эта реализация полезна в следующих ситуациях: 1. Некоторые культуры могут не поддерживаться пользовательской средой, и создание требуемой информации о культуре с заданным идентификатором может вызвать исключение. неподдерживаемый. 2. Пользователь может захотеть указать некоторые пользовательские свойства для некоторых культур, чтобы получить ожидаемый результат форматирования. Для этой цели пользователь может переопределить этот метод, чтобы предоставить экземпляру CultureInfo указанные пользователем свойства. Обратите внимание на свойство UseUserOverride возвращенный экземпляр CultureInfo может повлиять на отформатированный результат. Если это значение равно false, некоторые свойства возвращенного экземпляра CultureInfo могут быть переопределены нашим встроенным механизмом форматирования в соответствии с требованиями форматирования различных сценариев. Если это правда, мы не будем изменить любые его свойства и использовать для форматирования значений напрямую. Итак, если у пользователя есть спец. cified настраиваемые свойства для возвращаемого экземпляра CultureInfo, убедитесь, что для параметра UseUserOverride установлено значение true.
