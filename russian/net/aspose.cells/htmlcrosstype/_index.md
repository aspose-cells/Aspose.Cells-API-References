Представляет пять типов перекрестной строки HTML.

public enum HtmlCrossType

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| Default | `0` | Отображение как в MS Excel, зависит от следующей ячейки. Если следующая ячейка пуста, строка будет пересекаться или будет усечена |
| MSExport | `1` | Отобразить строку, как экспорт MS Excel html. |
| Cross | `2` | Отображать перекрестную строку HTML, эта производительность для создания больших файлов html будет более чем в десять раз быстрее, чем установка значения по умолчанию или FitToCell. |
| CrossHideRight | `3` | Отображать перекрестную строку HTML и скрывать правую строку, когда тексты перекрываются. |
| FitToCell | `4` | Отображает только строку в пределах ширины ячейки. |
