## Cell.StringValue property

Получает строковое значение, содержащееся в ячейке. Если тип этой ячейки — строка, верните само строковое значение. Для других типов ячеек будет возвращено форматированное строковое значение (отформатированное в соответствии с указанным стилем этой ячейки). Отформатированное значение ячейки совпадает с тем, что вы можете получить из Excel при копировании ячейки в виде текста (например, копирование ячейки в текстовый редактор или экспорт в csv).

public string StringValue { get; }
