## AbstractCalculationMonitor.OnCircular method

Реализуйте этот метод для ведения бизнеса при расчете формул с циклическими ссылками.

```csharp
public virtual bool OnCircular(IEnumerator circularCellsData)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| circularCellsData | IEnumerator | IEnumerator с элементами CalculationCell, представляющими ячейки, которые зависят от циклических ссылок. |

### Возвращаемое значение

Нужно ли обработчику формул вычислять эти ячейки по кругу после этого вызова. Значение true, чтобы обработчик формул продолжал выполнять вычисления для них. False, чтобы обработчик формул просто помечал эти ячейки как рассчитанные.

### Примечания

В реализации пользователь также может установить ожидаемое значение как вычисленное значение result для части/всех этих ячеек, чтобы обработчик формул не вычислял их рекурсивно.
