## RemoveAt(int, bool) {#removeat_1}

Удаляет указанную внешнюю ссылку из книги.

```csharp
public void RemoveAt(int index, bool updateReferencesAsLocal)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | Int32 | индекс внешней ссылки, которую необходимо удалить. |
| updateReferencesAsLocal | Boolean | Обновить ли все ссылки данной внешней ссылки на ссылку самой текущей рабочей книги. |

### Примечания

Если необходимо обновить ссылки, ссылки на внешние ссылки в формулах будут заменены на текущую книгу. исходная формула одной ячейки - "='externalsource.xlam'!customfunction()", после удаления формулы станет "=customfunction()". Если ссылку не требуется обновлять, все формулы со ссылкой на эту внешнюю link также будет удален, поскольку эти ссылки станут недействительными.
