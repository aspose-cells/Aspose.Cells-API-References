## TxtValueQuoteType enumeration

Указывает тип использования кавычек для значений в файлах текстового формата.

```csharp
public enum TxtValueQuoteType
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| Normal | `0` | Все значения, содержащие специальные символы, такие как кавычки, символы-разделители, будут заключены в кавычки. То же, что и поведение MS Excel при экспорте текстового файла. |
| Always | `1` | Все значения всегда будут заключаться в кавычки. |
| Minimum | `2` | Закавычивайте значения только при необходимости. Например, если одно значение содержит кавычку, но кавычка не стоит в начале этого значения, это значение не будет заключено в кавычки. |
| Never | `3` | Все значения не будут заключаться в кавычки. Экспортированный текстовый файл этого типа может быть неправильно прочитан из-за отсутствия необходимых кавычек. |
