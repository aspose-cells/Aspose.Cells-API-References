## DataBarAxisPosition enumeration

Указывает положение оси для диапазона ячеек с условным форматированием в виде гистограмм.

```csharp
public enum DataBarAxisPosition
```

### Ценности

| Имя | Ценность | Описание |
| --- | --- | --- |
| Automatic | `0` | Отображение оси в переменном положении на основе отношения минимального отрицательного значения к максимальному положительному значению в диапазоне. Положительные значения отображаются слева направо. Отрицательные значения отображаются справа налево. Когда все значения положительные или все значения отрицательные, ось не отображается. |
| Midpoint | `1` | Отображать ось в середине ячейки независимо от набора значений в диапазоне. Положительные значения отображаются слева направо. Отрицательные значения отображаются справа налево. |
| None | `2` | Ось не отображается, и положительные, и отрицательные значения отображаются в направлении слева направо. |
