## Workbook.ChangePalette method

Изменяет палитру электронной таблицы в указанном индексе.

```csharp
public void ChangePalette(Color color, int index)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| color | Color | Структура цвета. |
| index | Int32 | Индекс палитры, 0–55. |

### Примечания

Палитра содержит 56 элементов, каждый из которых представлен значение RGB. Если вы установите цвет, которого нет в палитре, он не вступит в силу. Поэтому, если вы хотите установить собственный цвет, сначала измените палитру.
