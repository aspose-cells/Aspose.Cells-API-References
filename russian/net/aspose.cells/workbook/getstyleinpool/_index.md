## Workbook.GetStyleInPool method

Получает стиль из пула стилей. Все стили в книге будут собраны в пул. В ячейках есть только простой ссылочный индекс.

```csharp
public Style GetStyleInPool(int index)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| index | Int32 | Индекс. |

### Возвращаемое значение

Стиль в пуле соответствует заданному индексу, может быть нулевым.

### Примечания

Если возвращенный стиль изменен, стиль всех ячеек (который относится к этому стилю) будет быть изменен.
