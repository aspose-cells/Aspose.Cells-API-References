## Workbook.UpdateLinkedDataSource method

Если эта книга содержит внешние ссылки на другой источник данных, Aspose.Cells попытается получить последние данные.

```csharp
public void UpdateLinkedDataSource(Workbook[] externalWorkbooks)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| externalWorkbooks | Workbook[] | Эта книга ссылается на внешние книги. снова внешние связанные файлы. |

### Примечания

Если метод не вызывается перед вычислением формул, Aspose.Cells будет использовать предыдущую информацию (кэшированную в файле); Пожалуйста, установите CellsHelper.StartupPath,CellsHelper.AltStartPath,CellsHelper.LibraryPath. И, пожалуйста, установите Workbook.FilePath, если эта книга из потока, в противном случае Aspose.Cells иногда не может получить полный путь внешней ссылки.
