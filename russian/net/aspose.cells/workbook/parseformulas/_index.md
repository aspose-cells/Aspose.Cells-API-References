## Workbook.ParseFormulas method

Разбирает все формулы, которые не были проанализированы, когда они были загружены из файла шаблона или установлены в ячейку.

```csharp
public void ParseFormulas(bool ignoreError)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| ignoreError | Boolean | игнорировать ли ошибку для недопустимой формулы. Для одной недопустимой формулы, если игнорировать ошибку, то эта формула будет проигнорирована , и процесс продолжит анализ других формул, в противном случае будет выдано исключение. |
