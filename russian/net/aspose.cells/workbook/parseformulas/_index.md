## Workbook.ParseFormulas method

Разбирает все формулы, которые не были проанализированы, когда они были загружены из файла шаблона или установлены в ячейку.

```csharp
public void ParseFormulas(bool ignoreError)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| ignoreError | Boolean | следует ли игнорировать ошибку для неверной формулы. Для одной недопустимой формулы, если игнорировать ошибку, эта формула будет проигнорирована и процесс продолжит разбор других формул, иначе будет выдано исключение. |
