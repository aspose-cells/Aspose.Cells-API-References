## FontConfigs.PreferSystemFontSubstitutes property

Указывает, следует ли сначала использовать замены системного шрифта, когда шрифт не представлен и замена этого шрифта не установлена. Например, в Ubuntu шрифт «Arial» обычно заменяется на «Liberation Sans». Значение по умолчанию — false .

public static bool PreferSystemFontSubstitutes { get; set; }
