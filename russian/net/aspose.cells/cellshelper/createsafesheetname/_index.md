## CreateSafeSheetName(string) {#createsafesheetname}

Проверяет заданное имя листа и при необходимости создает допустимое. Если заданное имя листа соответствует правилам имени листа Excel, вернуть его. В противном случае строка будет усечена, если длина превысит лимит и недопустимые символы будут заменены на ' ', после чего будет возвращено перестроенное строковое значение.

```csharp
public static string CreateSafeSheetName(string nameProposal)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| nameProposal | String | имя используемого листа |

Проверяет заданное имя листа и при необходимости создает допустимое. Если заданное имя листа соответствует правилам имени листа Excel, вернуть его.
