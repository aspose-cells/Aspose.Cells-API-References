## ExportTableOptions.SkipErrorValue property

Указывает, пропускать ли недопустимое значение для столбца. Например, если тип столбца десятичный, значение больше десятичного.MaxValue и это свойство истинно, мы больше не будем выдавать исключение. Значение по умолчанию — false.
