## LoadOptions.KeepUnparsedData property

Сохранять ли непроанализированные данные в памяти для книги, когда она загружается из файла шаблона. Значение по умолчанию — true.

```csharp
public bool KeepUnparsedData { get; set; }
```

### Примечания

Для сценариев, когда пользователю нужно только прочитать некоторое содержимое из файла шаблона и не нужно сохранять книгу обратно, установка для этого свойства значения false может повысить производительность, особенно при использовании его вместе с каким-либо LoadFilter,
