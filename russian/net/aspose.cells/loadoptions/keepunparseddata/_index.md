## LoadOptions.KeepUnparsedData property

Сохранять ли непроанализированные данные в памяти для рабочей книги, когда она загружается из файла шаблона. Значение по умолчанию верно.

```csharp
public bool KeepUnparsedData { get; set; }
```

### Примечания

Для сценариев, в которых пользователю нужно только прочитать некоторое содержимое из файла шаблона и не нужно сохранять workbook back, установка этого свойства как false может повысить производительность, особенно при использовании вместе с каким-либо LoadFilter,
