## CopyOptions.ReferToDestinationSheet property

При копировании диапазона в тот же файл, а диаграмма ссылается на исходный лист, False означает, что источник данных скопированной диаграммы не будет изменен . True означает, что источник данных скопированного графика относится к целевому листу.

```csharp
public bool ReferToDestinationSheet { get; set; }
```

### Примечания

Значение по умолчанию - false, работает как MS Excel.
