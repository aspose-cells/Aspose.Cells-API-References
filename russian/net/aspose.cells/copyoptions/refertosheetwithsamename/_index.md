## CopyOptions.ReferToSheetWithSameName property

При копировании рабочего листа в другую рабочую книгу, если рабочий лист содержит формулы, которые ссылаются на другие рабочие листы в MS Excel, скопированные формулы должны ссылаться на исходная рабочая тетрадь. Но иногда мы копировали другие рабочие листы, и мы надеемся, что скопированные формулы ссылаются на другие рабочие листы с именем в той же рабочей книге, установите для этого свойства значение true.

```csharp
public bool ReferToSheetWithSameName { get; set; }
```

### Примечания

Значение по умолчанию - true.
