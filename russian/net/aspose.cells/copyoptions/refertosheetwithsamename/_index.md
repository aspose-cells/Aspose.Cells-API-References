## CopyOptions.ReferToSheetWithSameName property

При копировании листа в другую книгу, если лист содержит формулы, которые ссылаются на другие листы в MS Excel, скопированные формулы должны ссылаться на исходную книгу. Но иногда мы копировали другие листы, и мы надеемся, что скопированные формулы ссылаются на другие листы с таким же именем в той же книге, установите для этого свойства значение true.

```csharp
public bool ReferToSheetWithSameName { get; set; }
```

### Примечания

Значение по умолчанию — true.
