## PdfSaveOptions.EmfRenderSetting property

Настройка для рендеринга метафайла EMF.

```csharp
public EmfRenderSetting EmfRenderSetting { get; set; }
```

### Примечания

Метафайлы EMF, обозначенные как «EMF+ Dual», могут содержать как записи EMF+, так и записи EMF. Для рендеринга изображения можно использовать записи любого типа, только записи EMF+ или только записи EMF. КогдаEmfPlusPrefer установлен, то записи EMF+ будут анализироваться при рендеринге в pdf, в противном случае будут анализироваться только записи EMF. Значение по умолчанию:EmfOnly .
