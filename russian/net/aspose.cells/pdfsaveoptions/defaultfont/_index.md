## PdfSaveOptions.DefaultFont property

Если символы в Excel имеют формат Unicode и для них не задан правильный шрифт в стиле ячейки, Они могут отображаться в виде блока в pdf, image. Установите шрифт по умолчанию, например MingLiu или MS Gothic, для отображения этих символов. Если это свойство не задано, Aspose.Cells будет использовать системный шрифт по умолчанию для отображения этих символов Юникода.

```csharp
public string DefaultFont { get; set; }
```
