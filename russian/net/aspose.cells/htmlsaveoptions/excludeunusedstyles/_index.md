## HtmlSaveOptions.ExcludeUnusedStyles property

Указывает, исключаются ли неиспользуемые стили. Для сгенерированных html-файлов исключение неиспользуемых стилей может уменьшить размер файла без влияния на визуальные эффекты. Таким образом, значение этого свойства по умолчанию равно true. Если пользователю необходимо сохранить все стили в рабочей книге для сгенерированного html (например, сценарий, в котором пользователю необходимо позже восстановить рабочую книгу из сгенерированного html), установите этот свойство как ложное.

```csharp
public bool ExcludeUnusedStyles { get; set; }
```
