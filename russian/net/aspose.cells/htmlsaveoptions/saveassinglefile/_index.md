## HtmlSaveOptions.SaveAsSingleFile property

Указывает, следует ли сохранять html как один файл. Значение по умолчанию — false.

```csharp
public bool SaveAsSingleFile { get; set; }
```

### Примечания

Если в книге есть несколько рабочих листов или других необходимых ресурсов, таких как изображения, обычно эти рабочие листы и другие ресурсы необходимо сохранять в отдельные файлы. В некоторых сценариях пользователю может потребоваться получить только один результирующий файл, например, для удобства передачи. Если это так, пользователь может установить для этого свойства значение true.
