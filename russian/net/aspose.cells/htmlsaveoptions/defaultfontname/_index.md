## HtmlSaveOptions.DefaultFontName property

Укажите имя шрифта по умолчанию для экспорта html, шрифт по умолчанию будет использоваться, когда шрифт стиля не существует, Если это свойство null, Aspose.Cells будет использовать универсальный шрифт того же семейства, что и исходный шрифт, значение по умолчанию равно null.

public string DefaultFontName { get; set; }
