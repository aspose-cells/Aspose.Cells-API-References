## HtmlSaveOptions.ExportArea property

Получает или задает экспортируемую CellArea текущего активного рабочего листа. Если установить этот атрибут, область печати текущего активного рабочего листа будет опущена. Только указанная область будет экспортирована при сохранении файла в формате html.

public CellArea ExportArea { get; set; }
