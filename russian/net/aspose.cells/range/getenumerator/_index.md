## Range.GetEnumerator method

Получает перечислитель для ячеек в этом диапазоне.

```csharp
public IEnumerator GetEnumerator()
```

### Возвращаемое значение

Перечислитель ячеек

### Примечания

При обходе элементов возвращенным перечислителем коллекция ячеек не должна изменяться (например, операции, которые вызовут создание экземпляра новой ячейки/строки или удаление существующей ячейки/строки). В противном случае перечислитель не сможет правильно пройти все ячейки (некоторые элементы могут быть пройдены повторно или пропущены).

### Примеры

```csharp
[C#]
Workbook workbook = new Workbook("template.xlsx");
Cells cells = workbook.Worksheets[0].Cells;

IEnumerator en = cells.CreateRange("B2:C3").GetEnumerator();
while (en.MoveNext())
{
    Cell cell = (Cell)en.Current;
    Console.WriteLine(cell.Name + ": " + cell.Value);
}
```
