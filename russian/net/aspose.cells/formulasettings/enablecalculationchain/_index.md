## FormulaSettings.EnableCalculationChain property

Включить ли цепочку вычислений для формул. По умолчанию false.

```csharp
public bool EnableCalculationChain { get; set; }
```

### Примечания

Когда в рабочей книге много формул и пользователю необходимо вычислять их повторно с изменением лишь небольшой их части, для производительности может быть полезно включить цепочку вычислений. С другой стороны, если цепочка включена, поддержание модели цепочки требует дополнительной памяти, , а также требует немного больше процессорного времени для некоторых других операций, таких как изменение значения ячейки или формул. время первого расчета для рабочей книги, поэтому требуемое время для первого расчета может быть больше, чем обычный расчет без цепочки.
