## FormulaSettings.ForceFullCalculation property

Указывает, вычисляются ли все формулы каждый раз при запуске вычисления.

```csharp
public bool ForceFullCalculation { get; set; }
```

### Примечания

Это свойство предназначено только для сохранения настроек в итоговый файл электронной таблицы так что другие приложения (например, MS Excel) могут действовать соответствующим образом при загрузке и манипулировании результирующим файлом. Из соображений производительности для большинства пользовательских приложений мы не вычисляем никакие формулы в книге автоматически, независимо от того, какое значение было установлено для этого свойства.
