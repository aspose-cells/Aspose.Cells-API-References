## FormulaSettings.CalculationId property

Указывает версию механизма расчета, используемого для расчета значений в рабочей книге.

```csharp
public string CalculationId { get; set; }
```

### Примечания

Это свойство предназначено только для сохранения настроек в итоговый файл электронной таблицы так что другие приложения (например, MS Excel) могут действовать соответствующим образом при загрузке и манипулировании результирующим файлом. В случае MS Excel, если значение этого свойства меньше, чем идентификатор механизма пересчета, связанный с приложением, открывающим результирующий файл, приложение пересчитывает результаты всех формул в этой книге сразу после загрузки файла. Из соображений производительности для большинства пользовательских приложений мы не вычисляем никакие формулы в книге автоматически, независимо от того, какое значение было установлено для этого свойства.
