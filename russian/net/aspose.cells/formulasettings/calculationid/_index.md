## FormulaSettings.CalculationId property

Указывает версию механизма расчета, используемого для расчета значений в рабочей книге.

```csharp
public string CalculationId { get; set; }
```

### Примечания

Это свойство предназначено только для сохранения настроек в итоговом файле электронной таблицы , чтобы другие приложения (такие как MS Excel) могли действовать соответствующим образом при загрузке результирующего файла и управлении им. В случае MS Excel, если значение этого свойства меньше чем идентификатор механизма пересчета, связанный с приложением, которое открывает результирующий файл, приложение будет пересчитывать результаты всех формул в этой книге сразу после загрузки файла. Из соображений производительности для приложений большинства пользователей мы не вычисляем никакие формулы для книга автоматически, независимо от того, какое значение было установлено для этого свойства.
