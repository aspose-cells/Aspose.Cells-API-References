## FormulaSettings.CalculateOnOpen property

Указывает, требуется ли приложению выполнять полный расчет при открытии рабочей книги.

```csharp
public bool CalculateOnOpen { get; set; }
```

### Примечания

Это свойство предназначено только для сохранения настроек в итоговый файл электронной таблицы так что другие приложения (например, MS Excel) могут действовать соответствующим образом при загрузке результирующего файла. Из соображений производительности для большинства пользовательских приложений мы не вычисляем никакие формулы в книге автоматически, независимо от того, какое значение было установлено для этого свойства.
