## FormatCondition.SetFormulas method

Устанавливает значение или выражение, связанное с этим условием формата.

```csharp
public void SetFormulas(string formula1, string formula2, bool isR1C1, bool isLocal)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| formula1 | String | Значение или выражение, связанное с этим условием формата. Если входное значение начинается с '=', то оно будет принято как формула. В противном случае оно будет принято как простое значение (текст, число, логическое значение). Для текстового значения, которое начинается с '=', пользователь может ввести его как формулу в формате:"=\"=...\"". |
| formula2 | String | Значение или выражение, связанное с этим условием формата. Формат ввода аналогичен формуле1 |
| isR1C1 | Boolean | Является ли формула формулой R1C1. |
| isLocal | Boolean | Отформатирована ли формула по локали. |
