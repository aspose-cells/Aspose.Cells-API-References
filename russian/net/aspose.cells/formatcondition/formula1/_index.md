### Примечания

Перед установкой формулы добавьте все области. Для установки формулы для этого условия, если входное значение начинается с '=', то оно будет принято как формула. В противном случае будет принято простое значение (текст, число, логическое значение). Для текстового значения, которое начинается с '=', пользователь может ввести его как формулу в формате:"=\"=...\"".
