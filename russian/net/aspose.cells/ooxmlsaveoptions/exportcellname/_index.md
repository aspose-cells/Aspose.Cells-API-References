## OoxmlSaveOptions.ExportCellName property

Указывает, следует ли экспортировать имя ячейки в файл Excel2007 .xlsx (.xlsm, .xltx, .xltm). Если к выходному файлу может обращаться SQL Server DTS, это значение должно быть истинным. Установка значения false значительно повысит производительность и уменьшит размер файла при создании большого файла. Значение по умолчанию — true.

public bool ExportCellName { get; set; }
