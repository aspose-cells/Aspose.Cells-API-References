## FormulaParseOptions.Parse property

Разбирать ли заданную формулу. Значение по умолчанию — true. Если значение равно false, то заданная строка формулы будет храниться в ячейке как есть до тех пор, пока пользователь не вызовет другие методы для их анализа или проанализированные данные формулы не потребуются для других операций, таких как вычисление формул.

public bool Parse { get; set; }
