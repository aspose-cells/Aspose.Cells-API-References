## FormulaParseOptions.Parse property

Разбирать ли заданную формулу. Значение по умолчанию верно. Если оно ложно, то заданная строка формулы будет сохранена как есть для ячейки до тех пор, пока пользователь не вызовет другие методы для их разбора или данные проанализированной формулы не потребуются другими операциями, такими как как расчетные формулы.

public bool Parse { get; set; }
