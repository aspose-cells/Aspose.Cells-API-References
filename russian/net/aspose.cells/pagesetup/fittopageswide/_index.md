## PageSetup.FitToPagesWide property

Представляет количество страниц в ширину, до которых рабочий лист будет масштабирован при печати. Значение по умолчанию: 1.

```csharp
public int FitToPagesWide { get; set; }
```

### Примечания

Вы должны установить FitToPagesTall равным нулю, если хотите разместить все столбцы на одной странице.
