## PageSetup.FitToPagesTall property

Представляет количество страниц в высоту, до которых рабочий лист будет масштабирован при печати. Значение по умолчанию равно 1.

```csharp
public int FitToPagesTall { get; set; }
```

### Примечания

Вы должны установить FitToPagesWide равным нулю если вы хотите разместить все строки на одной странице.
