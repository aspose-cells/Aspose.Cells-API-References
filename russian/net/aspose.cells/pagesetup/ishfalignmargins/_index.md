## PageSetup.IsHFAlignMargins property

Указывает, выровнены ли поля верхнего и нижнего колонтитула с полями страницы. Если это свойство имеет значение true, левый верхний и нижний колонтитулы будут выровнены по левому полю, , а правый верхний и нижний колонтитулы будут выровнены по правому полю. Эта опция включена по умолчанию.

```csharp
public bool IsHFAlignMargins { get; set; }
```
