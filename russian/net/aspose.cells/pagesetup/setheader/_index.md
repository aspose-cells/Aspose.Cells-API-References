## PageSetup.SetHeader method

Устанавливает скрипт, форматирующий заголовок файла Excel.

```csharp
public void SetHeader(int section, string headerScript)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| section | Int32 | 0: левая часть, 1: центральная часть, 2: правая часть. |
| headerScript | String | Скрипт формата заголовка. |

### Примечания

Команды скрипта:

**Команда**

**Описание**

&amp;П

Текущий номер страницы

&amp;N

Количество страниц

&amp;D

Текущая дата

&amp;Т

Текущее время

&amp;А

Имя листа

&amp;F

Имя файла без пути

&amp;"&lt;ИмяШрифта&gt;"

Название шрифта, например: &amp;"Arial"

&amp;"&lt;ИмяШрифта&gt;, &lt;СтильШрифта&gt;"

Название и стиль шрифта, например: &amp;"Arial,Bold"

&amp;&lt;размер шрифта&gt;

Размер шрифта. Если за этой командой следует простое число, которое будет напечатано в заголовке, оно будет отделено от высоты шрифта символом пробела.

&amp;K&lt;RRGGBB&gt;

Цвет шрифта, например (КРАСНЫЙ): &amp;KFF0000

&amp;ГРАММ

Сценарий изображения

Например: "&amp;Arial,Bold&amp;8Примечание в заголовке"
