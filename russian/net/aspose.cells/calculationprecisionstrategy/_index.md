## CalculationPrecisionStrategy enumeration

Перечисляет стратегии обработки точности вычислений. Из-за проблем с точностью арифметики с плавающей запятой IEEE 754 некоторые "кажущиеся простыми" формулы могут не рассчитываться как ожидаемый результат. Например, формула "=-0,45+0,43+0,02", при прямом вычислении операндов с помощью оператора "+", результат не равен нулю. Для такой проблемы с точностью некоторые специальные стратегии могут дать ожидаемый результат.

| Round | `1` | Округляет результат вычисления по значащим цифрам. |
