## TxtLoadOptions.PreferredParsers property

Получает и устанавливает анализаторы предпочтительных значений для загрузки текстового файла.

```csharp
public ICustomParser[] PreferredParsers { get; set; }
```

### Примечания

parsers[0] — парсер, который будет использоваться для первого столбца в текстовом файле шаблона, parsers[1] — парсер, который будет использоваться для второго столбца, ... и т.д. Последний (parsers[parsers.length-1]) будет использоваться для всех остальных столбцов, начиная с parsers.length-1. Если один элемент имеет значение null, соответствующий столбец будет проанализирован синтаксическим анализатором Aspose.Cells по умолчанию.
