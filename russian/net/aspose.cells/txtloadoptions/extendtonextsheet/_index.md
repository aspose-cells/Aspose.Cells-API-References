## TxtLoadOptions.ExtendToNextSheet property

Расширяет ли данные на следующий лист, когда строки или столбцы данных превышают предел. Если это свойство истинно, дополнительные данные будут расширены на следующий лист за текущим (если текущий лист является последним, будет добавлен новый лист в текущую книгу). Если для этого свойства установлено значение false, данные, превышающие лимит, будут игнорироваться. Значение по умолчанию — false;

```csharp
public bool ExtendToNextSheet { get; set; }
```
