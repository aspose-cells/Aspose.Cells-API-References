## TxtLoadOptions.ExtendToNextSheet property

Продлевает ли данные на следующий лист, когда строки или столбцы данных превышают лимит. Если это свойство истинно, дополнительные данные будут расширены на следующий лист за текущим (если текущий лист является последним, новый лист будет добавлен к текущей книге). Если это свойство равно false, данные превышают лимит, будут игнорироваться. По умолчанию false;

```csharp
public bool ExtendToNextSheet { get; set; }
```
