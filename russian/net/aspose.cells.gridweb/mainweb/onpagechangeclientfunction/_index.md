## MainWeb.OnPageChangeClientFunction property

Получает или задает функцию на стороне клиента, которая будет вызываться после изменения индекса страницы. Действует, только если EnablePaging имеет значение true. Функция клиента должна быть объявлена следующим образом: функция MyOnPageChange (индекс) { console.log("текущая страница:"+index); } Примечание. Вы можете использовать указатель «this» в клиентской функции, чтобы указать элемент управления сеткой, который запускает событие.

```csharp
public string OnPageChangeClientFunction { get; set; }
```
