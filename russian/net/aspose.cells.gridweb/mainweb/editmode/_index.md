## MainWeb.EditMode property

Получает или задает режим редактирования элемента управления.

```csharp
public bool EditMode { get; set; }
```

### Примечания

Когда установлено значение true, ячейки элемента управления доступны для редактирования, и отображаются значки «сохранить» и «отменить» . Если задано значение false, ячейки элемента управления доступны только для чтения, а значок «сохранить» и значок «отменить» не отображаются.
