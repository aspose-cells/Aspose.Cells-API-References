## MainWeb.OnSubmitClientFunction property

Получает или задает клиентскую функцию, которая будет вызываться перед отправкой элемента управления на стороне клиента. Клиентская функция должна быть объявлена следующим образом:&lt;br /&gt; function MyOnSubmit(arg, cancelEdit)&lt;br /&gt; {&lt;br /&gt; return true;&lt;br /&gt;}&lt;br /&gt; arg — это аргумент отправки, содержащий команду для отправки на сервер. CancelEdit является логическим значением, указывающим, отменил ли элемент управления пользовательский ввод перед отправкой. Элемент управления продолжит отправку, если функция возвращает значение true. &lt;br /&gt;Примечание. Вы можете использовать указатель "this" в клиентской функции, чтобы указать элемент управления сетки, который запускает событие.
