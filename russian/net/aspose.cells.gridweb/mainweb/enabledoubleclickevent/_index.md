## MainWeb.EnableDoubleClickEvent property

Получает или задает, следует ли включить событие двойного щелчка на стороне клиента.

```csharp
public bool EnableDoubleClickEvent { get; set; }
```

### Примечания

Когда установлено значение true, элемент управления будет получать события двойного щелчка на стороне клиента, и вы можете обрабатывать эти события на стороне сервера. В противном случае элемент управления будет игнорировать любые события двойного щелчка .
