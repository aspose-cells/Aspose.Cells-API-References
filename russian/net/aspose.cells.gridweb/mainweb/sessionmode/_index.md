## MainWeb.SessionMode property

Получает или устанавливает режим сеанса сетки. Существует 4 типа режима сеанса: 1. Сеанс (по умолчанию): Используйте системный сеанс для хранения данных листа. Обычно asp.net использует состояние сеанса InProc. Сетка также поддерживает состояние сеанса State процесса StateServer и состояние сеанса SQLServer. 2. ViewState: используйте состояние просмотра страницы для хранения данных листа. 3. Пользовательский: используйте события LoadCustomData и SheetDataUpdated для сохранения/восстановления данных листа. 4. Файл: сохранение/восстановление данных листа в SessionStorePath.

```csharp
public SessionMode SessionMode { get; set; }
```

### Примечания

При использовании SessionMode.ViewState сетка будет хранить данные листов в состоянии просмотра страницы. Это уменьшит использование памяти сервером, но размер страницы будет больше, и это повлияет на общую производительность.

### Примеры

```csharp
[C#]
	GridWeb1.SessionMode = SessionMode.ViewState;
	
[Visual Basic]
	GridWeb1.SessionMode = SessionMode.ViewState
```
