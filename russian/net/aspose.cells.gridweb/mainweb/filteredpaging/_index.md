## MainWeb.FilteredPaging property

Получает или задает, следует ли включить разбиение по страницам после фильтрации данных, вступит в силу, когда EnablePaging имеет значение true.

```csharp
public bool FilteredPaging { get; set; }
```

### Примечания

Когда установлено значение true, элемент управления будет выполнять пейджинг на основе отфильтрованных данных, значение по умолчанию — false.
