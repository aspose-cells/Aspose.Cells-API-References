## FillFormat.GradientVariant property

Возвращает вариант градиента для указанной заливки. Применяется только для Excel 2007.

```csharp
public int GradientVariant { get; }
```

### Примечания

Может быть значением только от 1 до 4, соответствующим одному из четыре варианта на вкладке «Градиент» в диалоговом окне «Эффекты заливки». Если стиль — GradientStyle.FromCenter, аргумент Variant может быть только 1 или 2.
