## ShapeCollection.Group method

Сгруппируйте фигуры.

```csharp
public GroupShape Group(Shape[] groupItems)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| groupItems | Shape[] | элементы группы. |

### Возвращаемое значение

Вернуть форму группы.

### Примечания

Фигуры в groupItems не должны группироваться. Форма должна быть в этой коллекции Shapes.

### Примеры

```csharp

[C#]
  // добавляем первый shape
shapes.AddRectangle(2, 0, 2, 0, 50, 50);
  //добавляем второй shape
shapes.AddRectangle(6, 0, 2, 0, 30, 30);

Shape[] shapesArr = new Shape[] { shapes[0], shapes[1] };
GroupShape groupShape = shapes.Group(shapesArr);

```
