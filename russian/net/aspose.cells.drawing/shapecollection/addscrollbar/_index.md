## ShapeCollection.AddScrollBar method

Добавляет полосу прокрутки на рабочий лист.

```csharp
public ScrollBar AddScrollBar(int upperLeftRow, int top, int upperLeftColumn, int left, int height, 
    int width)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| upperLeftRow | Int32 | Индекс верхней левой строки. |
| top | Int32 | Представляет вертикальное смещение полосы прокрутки от ее левой строки в пикселях. |
| upperLeftColumn | Int32 | Индекс левого верхнего столбца. |
| left | Int32 | Представляет горизонтальное смещение полосы прокрутки от ее левого столбца в пикселях. |
| height | Int32 | Представляет высоту полосы прокрутки в пикселях. |
| width | Int32 | Представляет ширину полосы прокрутки в пикселях. |

### Возвращаемое значение

Объект полосы прокрутки.
