## ShapeCollection.AddButton method

Добавляет кнопку на рабочий лист.

```csharp
public Button AddButton(int upperLeftRow, int top, int upperLeftColumn, int left, int height, 
    int width)
```

| Параметр | Тип | Описание |
| --- | --- | --- |
| upperLeftRow | Int32 | Индекс верхней левой строки. |
| top | Int32 | Представляет вертикальное смещение кнопки относительно ее левой строки в единицах пикселей. |
| upperLeftColumn | Int32 | Индекс верхнего левого столбца. |
| left | Int32 | Представляет горизонтальное смещение Button от его левого столбца в единицах пикселей. |
| height | Int32 | Представляет высоту кнопки в пикселях. |
| width | Int32 | Представляет ширину кнопки в пикселях. |

Объект Button.
