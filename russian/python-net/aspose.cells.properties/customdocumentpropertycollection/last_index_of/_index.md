##  last_index_of(item) {#DocumentProperty}
Выполняет поиск указанного объекта и возвращает отсчитываемый от нуля индекс последнего вхождения во всем списке массивов.


###  Возвращает

Отсчитываемый от нуля индекс последнего вхождения значения во всем списке массивов, если он найден, иначе -1.


```python
def last_index_of(self, item):
    ...
```

##  last_index_of(item, index) {#DocumentProperty-int}
Выполняет поиск указанного объекта и возвращает отсчитываемый от нуля индекс последнего вхождения в диапазоне элементов в списке массивов, который простирается от первого элемента до указанного индекса.


###  Возвращает

Отсчитываемый от нуля индекс последнего вхождения значения в диапазоне элементов в списке массивов, простирающемся от первого элемента до startIndex, если он найден; в противном случае -1.


```python
def last_index_of(self, item, index):
    ...
```


| Параметры| Тип| Описание|
| :- | :- | :- |
| item | [DocumentProperty](/cells/ru/python-net/aspose.cells.properties/documentproperty) | Объект, который нужно найти в списке массивов.|
| index | int | Отсчитываемый от нуля начальный индекс обратного поиска.|


##  last_index_of(item, index, count) {#DocumentProperty-int-int}
Выполняет поиск указанного объекта и возвращает отсчитываемый от нуля индекс последнего вхождения в диапазоне элементов в списке массивов, который содержит указанное количество элементов и заканчивается указанным индексом.


###  Возвращает

Отсчитываемый от нуля индекс последнего вхождения значения в диапазоне элементов в списке массивов System.Collections., который содержит число элементов count и заканчивается на startIndex, если он найден, иначе -1.


```python
def last_index_of(self, item, index, count):
    ...
```


| Параметры| Тип| Описание|
| :- | :- | :- |
| item | [DocumentProperty](/cells/ru/python-net/aspose.cells.properties/documentproperty) | Объект, который нужно найти в списке массивов.|
| index | int | Отсчитываемый от нуля начальный индекс обратного поиска.|
| count | int | Количество элементов в разделе для поиска.|
