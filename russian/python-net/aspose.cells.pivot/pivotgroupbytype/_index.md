##  PivotGroupByType перечисление
Представляет тип сводной таблицы groupby.



Тип PivotGroupByType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| RANGE_OF_VALUES | Представлен диапазон значений, сгруппированных по типу.|
| SECONDS | Подарки Секунды сгруппированы по типу.|
| MINUTES | Подарки Минуты сгруппированы по типам.|
| HOURS | Подарки Часы сгруппированы по типу.|
| DAYS | Подарки Дни группируются по типу.|
| MONTHS | Подарки Месяцы сгруппированы по типу.|
| QUARTERS | Представленные кварталы группируются по типам.|
| YEARS | Подарки Годы сгруппированы по типу.|
