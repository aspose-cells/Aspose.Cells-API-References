##  PivotOptions класс
Представляет сложный тип, указывающий элементы управления сводкой, отображаемые на диаграмме.



Тип PivotOptions предоставляет следующие члены:

###  Характеристики
| Свойство| Описание|
| :- | :- |
| [drop_zone_filter](/cells/ru/python-net/aspose.cells.charts/pivotoptions/drop_zone_filter) | Указывает, должен ли элемент управления для каждого поля сводной таблицы на оси страницы сводной таблицы<br/> исходной сводной таблицы отображается на диаграмме, когда для параметра dropZonesVisible установлено значение true.|
| [drop_zone_categories](/cells/ru/python-net/aspose.cells.charts/pivotoptions/drop_zone_categories) | Указывает, должен ли элемент управления для каждого поля сводной таблицы на оси строки сводной таблицы<br/> исходной сводной таблицы отображается на диаграмме, когда для параметра dropZonesVisible установлено значение true.|
| [drop_zone_data](/cells/ru/python-net/aspose.cells.charts/pivotoptions/drop_zone_data) | Указывает, должен ли элемент управления для каждого поля сводной таблицы на оси данных сводной таблицы<br/> исходной сводной таблицы отображается на диаграмме, когда для параметра dropZonesVisible установлено значение true.|
| [drop_zone_series](/cells/ru/python-net/aspose.cells.charts/pivotoptions/drop_zone_series) | Указывает, должен ли элемент управления для каждого поля сводной таблицы на оси столбца сводной таблицы<br/> исходной сводной таблицы отображается на диаграмме, когда для параметра dropZonesVisible установлено значение true.|
| [drop_zones_visible](/cells/ru/python-net/aspose.cells.charts/pivotoptions/drop_zones_visible) |Указывает, могут ли на сводной диаграмме отображаться какие-либо элементы управления сводкой.|
