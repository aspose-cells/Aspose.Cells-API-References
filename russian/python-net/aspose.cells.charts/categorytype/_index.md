##  CategoryType перечисление
Представляет тип оси категорий.



Тип CategoryType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| AUTOMATIC_SCALE | Автоматическая шкала|
| CATEGORY_SCALE | Шкала категории|
| TIME_SCALE | Шкала времени|
