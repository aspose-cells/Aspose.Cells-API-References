##  TickMarkType перечисление
Представляет тип деления для указанной оси.



Тип TickMarkType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| CROSS | Тип галочки — Крест.|
| INSIDE | Тип отметки «Внутри».|
| NONE |Тип галочки — None.|
| OUTSIDE | Тип галочки — Снаружи|
