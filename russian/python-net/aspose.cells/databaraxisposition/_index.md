##  DataBarAxisPosition перечисление
Задает положение оси для диапазона ячеек с условным форматированием в виде гистограмм.



Тип DataBarAxisPosition предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| AUTOMATIC | Отобразите ось в переменном положении на основе отношения минимального отрицательного значения к максимальному положительному значению в диапазоне.<br/> Положительные значения отображаются слева направо.<br/> Отрицательные значения отображаются в направлении справа налево.<br/>Когда все значения положительные или все значения отрицательные, ось не отображается.|
| MIDPOINT | Отображать ось в середине ячейки вне зависимости от набора значений в диапазоне.<br/> Положительные значения отображаются слева направо.<br/> Отрицательные значения отображаются в направлении справа налево.|
| NONE | Ось не отображается, а положительные и отрицательные значения отображаются в направлении слева направо.|
