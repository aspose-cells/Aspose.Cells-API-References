##  LookInType перечисление
Представляет вид в типе.



Тип LookInType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| FORMULAS |Если ячейка содержит формулу, найти объект по формуле, иначе найти его по значению.|
| VALUES | Находить объекты только из форматированных значений.|
| VALUES_EXCLUDE_FORMULA_CELL | Найдите объект только из значений ячеек, которые не содержат формулы.|
| COMMENTS | Находить объекты только из комментариев.|
| ONLY_FORMULAS | Находить объекты только по формулам.|
| ORIGINAL_VALUES | Находить объекты только из исходных значений.|
