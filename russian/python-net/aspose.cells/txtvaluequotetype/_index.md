##  TxtValueQuoteType перечисление
Задает тип использования кавычек для значений в файлах текстового формата.



Тип TxtValueQuoteType предоставляет следующие члены:

###  Поля
| Поле| Описание|
| :- | :- |
| NORMAL | Все значения, содержащие специальные символы, такие как кавычки, символы-разделители, будут заключены в кавычки.<br/> То же самое с поведением MS Excel для экспорта текстовых файлов.|
| ALWAYS | Все значения всегда будут указаны в кавычках.|
| MINIMUM | Заключайте значения в кавычки только при необходимости. Например, если одно значение содержит кавычки, но кавычки нет в начале этого значения, это значение не будет заключено в кавычки.|
| NEVER | Все значения не будут заключаться в кавычки, экспортированный текстовый файл с таким типом может быть неправильно прочитан из-за отсутствия нужных кавычек.|
