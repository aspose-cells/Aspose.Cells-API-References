##  start_sheet(sheet) {#Worksheet}
Подготавливает параметры фильтра перед загрузкой данного рабочего листа.
Пользовательская реализация LoadFilter может изменить LoadDataFilterOptions здесь
для обозначения того, как загружать данные для этого рабочего листа.



```python
def start_sheet(self, sheet):
    ...
```


| Параметры| Тип| Описание|
| :- | :- | :- |
| sheet | [Worksheet](/cells/ru/python-net/aspose.cells/worksheet) | Рабочий лист для загрузки.<br/>Здесь можно использовать только несколько свойств для данного объекта рабочего листа.<br/>потому что большинство данных и свойств не были загружены.<br/> Имя, индекс, тип видимости|
