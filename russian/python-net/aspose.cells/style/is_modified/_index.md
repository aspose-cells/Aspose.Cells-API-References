##  is_modified(modify_flag) {#StyleModifyFlag}
Проверяет, были ли изменены указанные свойства стиля.
Используется для стиля ConditionalFormattings, чтобы проверить, следует ли использовать указанные свойства этого стиля при применении ConditionalFormattings к ячейке.


###  Возвращает

true, если указанные свойства были изменены


```python
def is_modified(self, modify_flag):
    ...
```

| :- | :- | :- |
| modify_flag | [StyleModifyFlag](/cells/ru/python-net/aspose.cells/stylemodifyflag) | Флаги изменения стиля|
