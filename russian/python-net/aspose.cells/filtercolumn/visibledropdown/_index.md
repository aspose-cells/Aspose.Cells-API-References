##  visibledropdown недвижимость

Указывает, видна ли кнопка автофильтра для этого столбца.

###  Примечания

 ПРИМЕЧАНИЕ. Этот элемент устарел.
 Вместо этого используйте FilterColumn.IsDropdownVisible, чтобы проверить, видна ли кнопка автофильтра для этого столбца.
Это свойство будет удалено через 12 месяцев, начиная с сентября 2020 года.
Aspose приносит извинения за возможные неудобства.
