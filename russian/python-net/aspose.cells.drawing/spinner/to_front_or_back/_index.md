##  to_front_or_back(orders) {#int}
Выводит фигуру на передний план или отправляет фигуру на задний план.



```python
def to_front_or_back(self, orders):
    ...
```


| Параметры| Тип| Описание|
| :- | :- | :- |
| orders | int | Если меньше нуля, возвращает фигуру обратно.<br/>Если он больше нуля, он выводит фигуру на передний план.|

###  Примеры

```python

shape.to_front_or_back(2)
```
