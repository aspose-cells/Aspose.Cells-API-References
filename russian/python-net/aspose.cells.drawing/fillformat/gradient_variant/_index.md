##  gradient_variant недвижимость

Возвращает вариант градиента для указанной заливки.
Применяется только для Excel 2007.

###  Примечания

Может принимать значение только от 1 до 4, соответствующее одному из четырех вариантов на вкладке «Градиент» в диалоговом окне «Эффекты заливки».
