##  id недвижимость

Указывает идентификатор GUID, который можно сопоставить с идентификатором GUID строки подписи, хранящейся в содержимом документа.
Значение по умолчанию — Пусто (все нули) Guid.

###  Примечания

Когда он установлен, он связывает [SignatureLine](/cells/ru/python-net/aspose.cells.drawing/signatureline) с соответствующим [DigitalSignature](/cells/ru/python-net/aspose.cells.digitalsignatures/digitalsignature).
###  Определение:
```python
@property
def id(self):
    ...
@id.setter
def id(self, value):
    ...
```
