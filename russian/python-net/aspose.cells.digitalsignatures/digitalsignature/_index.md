##  DigitalSignature класс
Подпись в файле.



Тип DigitalSignature предоставляет следующие члены:

###  Конструкторы
| Конструктор| Описание|
| :- | :- |
| [DigitalSignature(raw_data, password, comments, sign_time)](/cells/ru/python-net/aspose.cells.digitalsignatures/digitalsignature/__init__/#bytes-str-str-DateTime) | Конструктор цифровой подписи, использующий реализацию Bouncy Castle.|


###  Характеристики
| Свойство| Описание|
| :- | :- |
| [comments](/cells/ru/python-net/aspose.cells.digitalsignatures/digitalsignature/comments) | Цель подписи.|
| [sign_time](/cells/ru/python-net/aspose.cells.digitalsignatures/digitalsignature/sign_time) | Время подписания документа.|
| [id](/cells/ru/python-net/aspose.cells.digitalsignatures/digitalsignature/id) |Указывает идентификатор GUID, который можно сопоставить с идентификатором GUID строки подписи, хранящейся в содержимом документа.<br/> Значение по умолчанию — Пусто (все нули) Guid.|
| [text](/cells/ru/python-net/aspose.cells.digitalsignatures/digitalsignature/text) | Задает текст фактической подписи в цифровой подписи.<br/> Значение по умолчанию — Пусто.|
| [image](/cells/ru/python-net/aspose.cells.digitalsignatures/digitalsignature/image) | Задает изображение для цифровой подписи.<br/> Значение по умолчанию равно нулю.|
| [provider_id](/cells/ru/python-net/aspose.cells.digitalsignatures/digitalsignature/provider_id) | Указывает идентификатор класса поставщика подписи.<br/> Значение по умолчанию — Пусто (все нули) Guid.|
| [is_valid](/cells/ru/python-net/aspose.cells.digitalsignatures/digitalsignature/is_valid) | Если эта цифровая подпись действительна и документ не был подделан,<br/> это значение будет истинным.|
| [x_ad_es_type](/cells/ru/python-net/aspose.cells.digitalsignatures/digitalsignature/x_ad_es_type) | Тип XAdES.<br/> Значение по умолчанию — None (XAdES отключен).|
