Questo metodo imposta solo lo stile specificato come stile predefinito per questa colonna,
senza modificare le impostazioni di stile per le celle esistenti in questa colonna.
Per aggiornare contemporaneamente le impostazioni di stile delle celle esistenti allo stile specificato,
si prega di utilizzare [Column.apply_style(style, flag)](/cells/it/python-net/aspose.cells/column/apply_style)
