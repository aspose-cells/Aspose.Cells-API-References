##  id proprietà

Specifica un GUID che può essere incrociato con il GUID della riga della firma archiviata nel contenuto del documento.
Il valore predefinito è Guid vuoto (tutti zeri).

###  Osservazioni

Quando impostato associa [SignatureLine](/cells/it/python-net/aspose.cells.drawing/signatureline) al corrispondente [DigitalSignature](/cells/it/python-net/aspose.cells.digitalsignatures/digitalsignature).
