## Aspose.Cells per .NET

{{% alert color="primary" %}}

![Logo del prodotto Aspose.Cells per .NET](../home_3.png)

Aspose.Cells for .NET è una libreria di programmazione di fogli di calcolo che consente agli sviluppatori di software di creare, leggere, convertire, aggiornare e stampare file di fogli di calcolo dalle proprie applicazioni. Una combinazione di API e controlli GUI, Aspose.Cells per .NET velocizza la programmazione e la conversione di Microsoft Excel®. Aspose.Cells for .NET supporta i popolari formati di file di fogli di calcolo (XLS, XLSX, XLSM, XLSB, XLTX, XLTM, CSV, SpreadsheetML, ODS) utilizzati quotidianamente dalla tua azienda. Consente inoltre di esportare file Excel in PDF, XPS, HTML, MHTML, testo normale e formati di immagine popolari tra cui TIFF, JPG, PNG, BMP e SVG.

## Aspose.Cells per C++

{{% alert color="primary" %}}

![Logo del prodotto Aspose.Cells per C++](../home_6.png)

Aspose.Cells per C++ è una libreria C++ nativa che consente agli sviluppatori C++ di creare, manipolare e convertire fogli di calcolo a livello di codice senza richiedere l'automazione di Office o l'applicazione Microsoft Excel. Aspose.Cells per C++ supporta formati di fogli di calcolo come MS Excel 97-2003 (XLS), MS Excel 2007-2013/2016 (XLSX, XLSM, XLSB, ecc.), Open Office XML e altri formati. Puoi anche eseguire il rendering di immagini da fogli di lavoro e convertire un foglio di calcolo MS Excel in formato PDF. L'API consente inoltre di creare, manipolare e renderizzare grafici e forme. Inoltre, i calcoli delle formule sono un altro aspetto positivo fornito dal componente e puoi utilizzarlo.

## Aspose.Cells per Node.js tramite Java
