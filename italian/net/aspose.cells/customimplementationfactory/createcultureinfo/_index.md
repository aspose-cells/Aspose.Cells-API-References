## CustomImplementationFactory.CreateCultureInfo method

Crea un CultureInfo in base all'id specificato.

```csharp
public virtual CultureInfo CreateCultureInfo(int lcid)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| lcid | Int32 |  |

### Valore di ritorno

L'istanza CultureInfo.

### Osservazioni

Questa implementazione è utile per le situazioni: 1. Alcune culture potrebbero non essere supportate dall'ambiente dell'utente e la creazione del CultureInfo richiesto con l'identificatore specificato potrebbe causare un'eccezione. Per evitare l'eccezione, l'utente può sovrascrivere questo metodo per fornire un'istanza CultureInfo valida per il uno non supportato. 2. L'utente potrebbe voler specificare alcune proprietà personalizzate per alcune impostazioni cultura per ottenere il risultato previsto per la formattazione. A questo scopo l'utente può sovrascrivere questo metodo per fornire all'istanza CultureInfo le proprietà specificate dall'utente. Si prega di notare la proprietà UseUserOverride del l'istanza di CultureInfo restituita può influenzare il risultato formattato. Se è falso, alcune proprietà dell'istanza di CultureInfo restituita potrebbero essere sovrascritte dal nostro motore di formattazione integrato in base ai requisiti di formattazione di diversi scenari. Se è vero, non lo faremo cambia qualsiasi proprietà di esso e usalo per formattare i valori direttamente. Quindi, se l'utente ha spe cified proprietà personalizzate per l'istanza CultureInfo restituita, assicurati che UseUserOverride sia true.
