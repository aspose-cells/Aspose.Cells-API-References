## TxtValueQuoteType enumeration

Specifica il tipo di utilizzo delle virgolette per i valori nei file in formato testo.

```csharp
public enum TxtValueQuoteType
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Normal | `0` | Verranno virgolette tutti i valori che contengono caratteri speciali come virgolette, carattere separatore. Lo stesso vale per il comportamento di ms excel per l'esportazione di file di testo. |
| Always | `1` | Tutti i valori saranno sempre quotati. |
| Minimum | `2` | Citare i valori solo quando necessario. Ad esempio, se un valore contiene virgolette ma le virgolette non si trovano all'inizio di questo valore, questo valore non sarà tra virgolette. |
| Never | `3` | Tutti i valori non verranno quotati. Il file di testo esportato con questo tipo potrebbe non essere riletto correttamente perché le virgolette necessarie sono assenti. |
