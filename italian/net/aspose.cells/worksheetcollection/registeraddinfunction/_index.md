## RegisterAddInFunction(string, string, bool) {#registeraddinfunction}

Aggiunge la funzione di aggiunta nella cartella di lavoro

```csharp
public int RegisterAddInFunction(string addInFile, string functionName, bool lib)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| addInFile | String | il file contiene le funzioni di aggiunta |
| functionName | String | il nome della funzione aggiuntiva |
| lib | Boolean | se il file addin specificato si trova nella directory o nella sottodirectory della libreria del componente aggiuntivo cartella di lavoro. Questo flag ha effetto e fa la differenza quando addInFile specificato è di percorso relativo: true indica che il percorso è relativo alla libreria del componente aggiuntivo e false indica che il percorso è relativo a questa cartella di lavoro. |

### Valore di ritorno

ID dei dati che contengono una data funzione di aggiunta
