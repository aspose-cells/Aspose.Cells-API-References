## WorkbookSettings.IsHScrollBarVisible property

Ottiene o imposta un valore che indica se il foglio di calcolo generato conterrà una barra di scorrimento orizzontale.

```csharp
public bool IsHScrollBarVisible { get; set; }
```

### Osservazioni

Il valore predefinito è vero.

### Esempi

Il codice seguente rende invisibile la barra di scorrimento orizzontale per il foglio di calcolo.

```csharp
[C#]
// Nasconde la barra di scorrimento orizzontale del file Excel.
settings.IsHScrollBarVisible = false;

[Visual Basic]
' Nascondi la barra di scorrimento orizzontale del file Excel.
```
