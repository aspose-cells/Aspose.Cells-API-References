## Range.GetEnumerator method

Ottiene l'enumeratore per le celle in questo intervallo.

```csharp
public IEnumerator GetEnumerator()
```

### Valore di ritorno

L'enumeratore di celle

### Osservazioni

Quando si attraversano gli elementi dall'enumeratore restituito, le celle collection non devono essere modificate (come le operazioni che causeranno la creazione di un'istanza di una nuova cella/riga o l'eliminazione di una cella/riga esistente). Altrimenti l'enumeratore potrebbe non essere in grado di attraversare correttamente tutte le celle (alcuni elementi possono essere attraversati ripetutamente o saltati).

### Esempi

```csharp
[C#]
Workbook workbook = new Workbook("template.xlsx");
Cells cells = workbook.Worksheets[0].Cells;

IEnumerator en = cells.CreateRange("B2:C3").GetEnumerator();
while (en.MoveNext())
{
    Cell cell = (Cell)en.Current;
    Console.WriteLine(cell.Name + ": " + cell.Value);
}
```
