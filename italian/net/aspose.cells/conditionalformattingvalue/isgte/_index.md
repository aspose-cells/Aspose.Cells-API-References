## ConditionalFormattingValue.IsGTE property

Ottieni o imposta il flag Maggiore di o uguale. Usa solo per set di icone, determina se questo valore di soglia utilizza l'operatore maggiore o uguale a. 'false' indica che viene utilizzato 'maggiore di' invece di 'maggiore o uguale a'. Il valore predefinito è true.
