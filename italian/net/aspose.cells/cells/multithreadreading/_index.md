## Cells.MultiThreadReading property

Ottiene o imposta se il modello di dati delle celle deve supportare la lettura multi-thread. Il valore predefinito di questa proprietà è false.

```csharp
public bool MultiThreadReading { get; set; }
```

### Osservazioni

Se sono presenti più thread per leggere contemporaneamente oggetti Riga/Cella in questa raccolta, questa proprietà deve essere impostata su true, altrimenti potrebbe essere prodotto un risultato imprevisto. Il supporto della lettura multi-thread può ridurre le prestazioni per l'accesso agli oggetti Riga/Cella da questa raccolta. Nota, alcune funzionalità non possono supportare la lettura multi-thread, come i valori di formattazione (per[`StringValue`](../../cell/stringvalue) ,[`DisplayStringValue`](../../cell/displaystringvalue) .ecc.). Quindi, anche con questa proprietà impostata su true, tali API potrebbero comunque fornire risultati imprevisti per la lettura multi-thread.
