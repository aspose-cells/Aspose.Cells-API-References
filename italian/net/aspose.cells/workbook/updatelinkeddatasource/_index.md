## Workbook.UpdateLinkedDataSource method

Se questa cartella di lavoro contiene collegamenti esterni ad altre origini dati, Aspose.Cells tenterà di recuperare i dati più recenti.

```csharp
public void UpdateLinkedDataSource(Workbook[] externalWorkbooks)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| externalWorkbooks | Workbook[] | Le cartelle di lavoro esterne sono referenziate da questa cartella di lavoro. Se è null, apriremo direttamente i file collegati esterni.. Se non è null, verificheremo prima se il collegamento esterno nell'array; in caso contrario, apriremo di nuovo i file collegati esterni. |

### Osservazioni

Se il metodo non viene chiamato prima di calcolare le formule, Aspose.Cells utilizzerà le informazioni precedenti (cache nel file); Impostare CellsHelper.StartupPath,CellsHelper.AltStartPath,CellsHelper.LibraryPath. E per favore imposta Workbook.FilePath se questa cartella di lavoro proviene da un flusso, altrimenti Aspose.Cells non potrebbe ottenere il percorso completo del collegamento esterno a volte.
