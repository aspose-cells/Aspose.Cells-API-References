### Esempi

Il codice seguente ha esportato i dati collegati dalla prima XmlMap.

```csharp
Workbook wb = new Workbook("Book1.xlsx");

//Assicurati che il file xlsx di origine contenga una XmlMap.
XmlMap xmlMap = wb.Worksheets.XmlMaps[0];

wb.ExportXml(xmlMap.Name, "output.xml");
```
