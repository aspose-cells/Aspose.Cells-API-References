## PdfSaveOptions.EmfRenderSetting property

Impostazione per il rendering del metafile Emf.

```csharp
public EmfRenderSetting EmfRenderSetting { get; set; }
```

### Osservazioni

I metafile EMF identificati come "EMF+ Dual" possono contenere sia record EMF+ che record EMF. È possibile utilizzare entrambi i tipi di record per eseguire il rendering dell'immagine, solo record EMF+ o solo record EMF. QuandoEmfPlusPrefer è impostato, i record EMF+ verranno analizzati durante il rendering in pdf, altrimenti verranno analizzati solo i record EMF. Il valore predefinito èEmfOnly .
