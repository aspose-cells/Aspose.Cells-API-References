## PdfSaveOptions.PrintingPageType property

Indica quali pagine non verranno stampate.

```csharp
public PrintingPageType PrintingPageType { get; set; }
```

### Osservazioni

Se il contenuto del foglio è scarso, ci saranno alcune pagine completamente vuote nel file pdf di output. Se non vuoi queste pagine vuote, puoi usare questa opzione per ometterle.

### Esempi

Il codice seguente omette le pagine vuote o le pagine che contengono solo alcuni contenuti di stile come lo sfondo della cella, i bordi.

```csharp
Workbook wb = new Workbook("Book1.xlsx");

PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();

//ignora le pagine vuote
pdfSaveOptions.PrintingPageType = PrintingPageType.IgnoreBlank;

wb.Save("output_ignore_blank_page.pdf", pdfSaveOptions);


//ignora le pagine vuote and pages which only contains some style content like cell background
pdfSaveOptions.PrintingPageType = PrintingPageType.IgnoreStyle;

wb.Save("output_ignore_blank_and_style_page.pdf", pdfSaveOptions);
```
