## PdfSaveOptions.SetImageResample method

Imposta i PPI (pixel per pollice) desiderati per ricampionare le immagini e la qualità jpeg. Tutte le immagini verranno convertite in JPEG con l'impostazione di qualità specificata, e le immagini che sono maggiori del PPI specificato (pixel per pollice) verranno ricampionate.

```csharp
public void SetImageResample(int desiredPPI, int jpegQuality)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| desiredPPI | Int32 | Pixel desiderati per pollice. 220 di alta qualità. 150 qualità dello schermo. 96 qualità della posta elettronica. |
| jpegQuality | Int32 | 0 - 100% di qualità JPEG. |

### Esempi

Il codice seguente imposta il PPI desiderato come 96 e la qualità jpeg come 80 per le immagini nell'output pdf.

```csharp
//carica il file sorgente con le immagini.
Workbook wb = new Workbook("Book1.xlsx");

PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();

//imposta il PPI desiderato su 96 e la qualità jpeg su 80.
pdfSaveOptions.SetImageResample(96, 80);

wb.Save("output.pdf", pdfSaveOptions);
```
