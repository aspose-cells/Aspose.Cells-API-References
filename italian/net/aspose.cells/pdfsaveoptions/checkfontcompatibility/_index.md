## PdfSaveOptions.CheckFontCompatibility property

Indica se controllare la compatibilità dei font per ogni carattere del testo.

```csharp
public bool CheckFontCompatibility { get; set; }
```

### Osservazioni

Il valore predefinito è true. Disabilitare questa proprietà può fornire prestazioni migliori. Ma quando il carattere predefinito o specificato di testo/carattere non può essere utilizzato per renderizzarlo, caratteri illeggibili (come il blocco) potrebbero verificarsi nel pdf generato. Per tale situazione l'utente dovrebbe mantenere questa proprietà come true in modo che sia possibile cercare un carattere alternativo e utilizzarlo invece per eseguire il rendering del testo;
