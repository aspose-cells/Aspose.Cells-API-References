## Style.CultureCustom property

Ottiene e imposta la stringa del modello dipendente dalle impostazioni cultura per il formato numerico. Se non è stato impostato alcun formato numerico per questo oggetto, verrà restituito null. Se il formato numerico è incorporato, verrà restituita la stringa del modello corrispondente al numero incorporato.

```csharp
public string CultureCustom { get; set; }
```

### Osservazioni

Per il formato numerico incorporato, sia il contenuto del modello (ad esempio, un formato di data incorporato è "m/g/y" per alcune impostazioni locali, ma per alcune altre impostazioni locali diventa "d/m/a") e l'identificatore di formato (ad esempio, alcune impostazioni locali utilizzano un carattere diverso da 'y' per rappresentare la parte dell'anno per la formattazione della data) dipendono dalle impostazioni cultura; Per il formato personalizzato specificato dall'utente, solo gli identificatori di formato vengono modificati in base alle impostazioni cultura, altre parti del modello di formattazione non verrà modificato.
