## Cell.GetPrecedentsInCalculation method

Ottiene tutti i precedenti (riferimento alle celle nella cartella di lavoro corrente) utilizzati dalla formula di questa cella durante il calcolo.

```csharp
public IEnumerator GetPrecedentsInCalculation()
```

### Valore di ritorno

Enumeratore per enumerare tutti i riferimenti (ReferredArea)

### Osservazioni

Questo metodo può funzionare solo con la situazione che[`EnableCalculationChain`](../../formulasettings/enablecalculationchain) è true per la cartella di lavoro e la cartella di lavoro è stata completamente calcolata. Se questa cella non è una formula o non fa riferimento ad altre celle, verrà restituito null.

### Esempi

```csharp
[C#]

Workbook workbook = new Workbook();
Cells cells = workbook.Worksheets[0].Cells;
cells["A2"].Formula = "=IF(TRUE,B2,B1)";
workbook.Settings.FormulaSettings.EnableCalculationChain = true;
workbook.CalculateFormula();
IEnumerator en = cells["A2"].GetPrecedentsInCalculation();
Console.WriteLine("A2's calculation precedents:");
while(en.MoveNext())
{
    ReferredArea r = (ReferredArea)en.Current;
    Console.WriteLine(r);
}
```
