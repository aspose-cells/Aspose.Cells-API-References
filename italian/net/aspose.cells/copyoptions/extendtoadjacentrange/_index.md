## CopyOptions.ExtendToAdjacentRange property

Indica se estendere gli intervalli quando si copia l'intervallo in un intervallo adiacente.

```csharp
public bool ExtendToAdjacentRange { get; set; }
```

### Osservazioni

Se è vero, estende solo l'intervallo del collegamento ipertestuale, non aggiungendo un nuovo collegamento ipertestuale quando si copiano collegamenti ipertestuali su righe adiacenti.
