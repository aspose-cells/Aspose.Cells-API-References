## CopyOptions.ReferToSheetWithSameName property

Quando si copia un foglio di lavoro in un'altra cartella di lavoro e il foglio di lavoro contiene le formule che fanno riferimento ad altri fogli di lavoro in MS Excel, le formule copiate devono fare riferimento alla cartella di lavoro di origine. Ma a volte abbiamo copiato altri fogli di lavoro e speriamo che le formule copiate si riferiscano ad altri fogli di lavoro con il nome nella stessa cartella di lavoro, imposta questa proprietà come true.

```csharp
public bool ReferToSheetWithSameName { get; set; }
```

### Osservazioni

Il valore predefinito è true.
