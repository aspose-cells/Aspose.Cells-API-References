## CopyOptions.ReferToDestinationSheet property

Quando si copia l'intervallo nello stesso file e il grafico fa riferimento al foglio di origine, False significa che l'origine dati del grafico copiato non verrà modificata. True significa che l'origine dati del grafico copiato si riferisce al foglio di destinazione.

```csharp
public bool ReferToDestinationSheet { get; set; }
```

### Osservazioni

Il valore predefinito è false, funziona come MS Excel.
