## FormulaSettings.EnableCalculationChain property

Se abilitare la catena di calcolo per le formule. L'impostazione predefinita è false.

```csharp
public bool EnableCalculationChain { get; set; }
```

### Osservazioni

Quando sono presenti molte formule nella cartella di lavoro e l'utente deve calcolarle ripetutamente modificandone solo una piccola parte, può essere utile per le prestazioni abilitare la catena di calcolo. D'altra parte, se la catena è abilitata, il mantenimento del modello di catena richiede memoria aggiuntiva, e richiede anche un po' più di tempo della CPU per alcune altre operazioni come la modifica del valore della cella o delle formule. Dopo aver modificato questa proprietà da false a true, la catena di calcolo verrà analizzata e costruita in il tempo del primo calcolo per la cartella di lavoro, quindi il tempo richiesto per il primo calcolo potrebbe essere maggiore del normale calcolo senza catena.
