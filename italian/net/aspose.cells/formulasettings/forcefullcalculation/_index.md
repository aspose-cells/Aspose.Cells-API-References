## FormulaSettings.ForceFullCalculation property

Indica se calcola tutte le formule ogni volta che viene attivato un calcolo.

```csharp
public bool ForceFullCalculation { get; set; }
```

### Osservazioni

Questa proprietà serve solo per salvare le impostazioni nel file di foglio di calcolo risultante in modo che altre applicazioni (come ms excel) possano agire di conseguenza durante il caricamento e la manipolazione del file risultante. Per considerazioni sulle prestazioni per la maggior parte delle applicazioni degli utenti, non calcoliamo alcuna formula automaticamente nella cartella di lavoro, indipendentemente dal valore impostato per questa proprietà.
