## StreamProviderOptions.CustomPath property

Il percorso personalizzato dell'utente (URL) salvato nel file html generato per l'origine di riferimento. Se non definito dall'utente, verrà utilizzato DefaultPath. Ad esempio, i dati del foglio verranno salvati dall'utente in d:/sheet001.htm, l'url utilizzato nel file html principale dovrebbe essere "d:/sheet001.htm" o altro percorso relativo valido a cui si può accedere dal file html principale.

public string CustomPath { set; }
