## Protection.Password property

Rappresenta la password per proteggere il foglio di lavoro.

```csharp
public string Password { get; set; }
```

### Osservazioni

Se la password è impostata su una stringa nulla o vuota, è possibile annullare la protezione del foglio di lavoro o della cartella di lavoro senza utilizzare una password. In caso contrario, è necessario specificare la password per annullare la protezione del foglio di lavoro o della cartella di lavoro.
