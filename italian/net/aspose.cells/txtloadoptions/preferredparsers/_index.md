## TxtLoadOptions.PreferredParsers property

Ottiene e imposta i parser di valori preferiti per il caricamento del file di testo.

```csharp
public ICustomParser[] PreferredParsers { get; set; }
```

### Osservazioni

parser[0] è il parser verrà utilizzato per la prima colonna nel file modello di testo, parser[1] è il parser verrà utilizzato per la seconda colonna, ... ecc. L'ultimo(parser[parser. length-1]) verrà utilizzato per tutte le altre colonne a partire da parser.length-1. Se un elemento è nullo, la colonna corrispondente verrà analizzata dal parser predefinito di Aspose.Cells.
