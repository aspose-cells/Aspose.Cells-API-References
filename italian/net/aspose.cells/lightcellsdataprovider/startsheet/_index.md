## LightCellsDataProvider.StartSheet method

Inizia a salvare un foglio di lavoro.

```csharp
public bool StartSheet(int sheetIndex)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| sheetIndex | Int32 | indice del foglio corrente da salvare. |

### Valore di ritorno

true se questo provider fornirà i dati per il foglio specificato; false se il foglio specificato deve utilizzare il suo modello di dati normale (celle).

### Osservazioni

Verrà chiamato all'inizio del salvataggio di un foglio di lavoro durante il salvataggio di una cartella di lavoro. Se il provider deve fare riferimento a

```csharp
sheetIndex
```

later nel metodo startRow(Row) o startCell(Cell), ovvero, se il processo ha bisogno di sapere quale foglio di lavoro viene elaborato, l'implementazione dovrebbe mantenere

```csharp
sheetIndex
```

valore qui.
