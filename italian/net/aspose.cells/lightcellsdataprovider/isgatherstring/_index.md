## LightCellsDataProvider.IsGatherString method

Verifica se il valore della stringa corrente della cella deve essere raccolto in un pool globale.

```csharp
public bool IsGatherString()
```

### Valore di ritorno

true se il valore della stringa deve essere raccolto in un pool globale per il file risultante.

### Osservazioni

La raccolta di valori di stringa trarrà vantaggio solo quando ci sono molti valori di stringa duplicati per le celle fornite da questa implementazione. In questa situazione la raccolta di stringhe farà risparmiare molta memoria e genererà un file risultante più piccolo. Se sono presenti molti valori di stringa per le celle fornite di LightCellsDataProvider ma pochi sono uguali, la raccolta di stringhe costerà più memoria e tempo e non ha alcun vantaggio per il file risultante.
