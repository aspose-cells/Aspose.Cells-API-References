### Osservazioni

Quando si legge una cartella di lavoro in questa modalità,[`StartSheet`](./startsheet) verrà controllato durante la lettura di ogni foglio di lavoro nella cartella di lavoro. Per un foglio, se[`StartSheet`](./startsheet) restituisce true, quindi tutti i dati e le proprietà delle righe/celle di questo foglio verranno controllati ed elaborati dall'implementazione di questa interfaccia. Per ogni riga,[`StartRow`](./startrow) verrà chiamato per verificare se deve essere elaborata. Se una riga deve essere elaborata, le proprietà di questa riga verranno prima lette e l'utente può accedere alle sue proprietà tramite[`ProcessRow`](./processrow). se è necessario elaborare anche le celle della riga, allora[`ProcessRow`](./processrow) dovrebbe restituire true e poi[`StartCell`](./startcell) verrà chiamato per ogni cella esistente in questa riga per verificare se una cella deve essere elaborata. Se è necessario elaborare una cella, allora[`ProcessCell`](./processcell) verrà chiamato per elaborare la cella dall'implementazione di questa interfaccia.
