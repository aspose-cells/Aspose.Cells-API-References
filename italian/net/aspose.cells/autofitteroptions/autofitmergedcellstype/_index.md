## AutoFitterOptions.AutoFitMergedCellsType property

Ottiene e imposta il tipo di adattamento automatico dell'altezza della riga delle celle unite.

```csharp
public AutoFitMergedCellsType AutoFitMergedCellsType { get; set; }
```

### Osservazioni

Per impostazione predefinita Excel ignora le celle unite quando si adatta l'altezza della riga, quindi Aspose.Cells funziona come predefinito di MS Excel. Imposta questo tipo per modificare il modo di adattare automaticamente l'altezza della riga delle celle unite.
