## CalculationData.CalculatedValue property

Ottiene o imposta il valore calcolato per questa funzione.

```csharp
public object CalculatedValue { get; set; }
```

### Osservazioni

L'utente deve impostare questa proprietà nel suo motore di calcolo personalizzato per quelle funzioni supportate dal motore, e il valore impostato verrà restituito quando si ottiene questa proprietà in un secondo momento. Il valore impostato può essere qualsiasi valore di quegli oggetti che possono essere impostati su una cella( Cell.Value). E può anche essere un array di questo tipo di valori, o un Range, Name, ReferredArea. Ottenere questa proprietà prima dell'impostazione farà sì che la funzione venga calcolata dal motore di calcolo predefinito di Aspose.Cells e calcolata il valore verrà restituito.
