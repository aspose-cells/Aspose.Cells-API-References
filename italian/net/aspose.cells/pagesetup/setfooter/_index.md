## PageSetup.SetFooter method

Imposta uno script per la formattazione del piè di pagina di un file Excel.

```csharp
public void SetFooter(int section, string footerScript)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| section | Int32 | 0: Sezione sinistra, 1: Sezione centrale, 2: Sezione destra. |
| footerScript | String | Script in formato piè di pagina. |

### Osservazioni

Comandi di script:

**Comando**

**Descrizione**

&amp;P

Numero di pagina corrente

&amp;N

Conteggio pagine

&amp;D

Data odierna

&amp;T

Ora attuale

&amp;UN

Nome foglio

&amp;F

Nome file senza percorso

&amp;"&lt;Nome carattere&gt;"

Nome del carattere, ad esempio: &amp;"Arial"

&amp;"&lt;Nome carattere&gt;, &lt;Stile carattere&gt;"

Nome del carattere e stile del carattere, ad esempio: &amp;"Arial,Grassetto"

&amp;&lt;Dimensione carattere&gt;

Dimensione del font. Se questo comando è seguito da un numero semplice da stampare nell'intestazione, sarà separato dall'altezza del carattere con uno spazio.

&amp;K&lt;RRGGBB&gt;

Colore del carattere, ad esempio (ROSSO): &amp;KFF0000

&amp;G

Sceneggiatura immagine

Ad esempio: "&amp;Arial, Grassetto&amp;8 Nota a piè di pagina"
