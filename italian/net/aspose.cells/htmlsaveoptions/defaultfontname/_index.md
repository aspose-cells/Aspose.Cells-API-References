## HtmlSaveOptions.DefaultFontName property

Specificare il nome del carattere predefinito per l'esportazione html, il carattere predefinito verrà utilizzato quando il carattere dello stile non è esistente, Se questa proprietà è nulla, Aspose.Cells utilizzerà il carattere universale che ha la stessa famiglia del carattere originale, il valore predefinito è null.

public string DefaultFontName { get; set; }
