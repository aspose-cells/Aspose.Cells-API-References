## HtmlSaveOptions.SaveAsSingleFile property

Indica se salvare l'html come file singolo. Il valore predefinito è false.

```csharp
public bool SaveAsSingleFile { get; set; }
```

### Osservazioni

Se sono presenti più fogli di lavoro o altre risorse necessarie come immagini nella cartella di lavoro, comunemente tali fogli di lavoro e altre risorse devono essere salvati in file separati. Per alcuni scenari, l'utente potrebbe aver bisogno di ottenere solo un file risultante, ad esempio per comodità del trasferimento. In tal caso, l'utente può impostare questa proprietà su true.
