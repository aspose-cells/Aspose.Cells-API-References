## CalculationOptions.CalcStackSize property

Specifica la dimensione dello stack per il calcolo ricorsivo delle celle.

```csharp
public int CalcStackSize { get; set; }
```

### Osservazioni

Quando è necessario calcolare una grande quantità di celle in modo ricorsivo nell'albero delle dipendenze, StackOverflowException potrebbe essere causato nel processo di calcolo. In tal caso, l'utente deve specificare un valore inferiore per questa proprietà. Per tale situazione, l'utente deve determinare il valore corretto per questa proprietà in base alle formule e ai dati effettivi. Un valore troppo piccolo può causare una riduzione delle prestazioni per il calcolo della formula.
