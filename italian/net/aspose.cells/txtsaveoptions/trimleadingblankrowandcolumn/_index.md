## TxtSaveOptions.TrimLeadingBlankRowAndColumn property

Indica se le righe e le colonne vuote iniziali devono essere tagliate come fa ms excel. L'impostazione predefinita è true.

```csharp
public bool TrimLeadingBlankRowAndColumn { get; set; }
```

### Osservazioni

Come per la regola in ms excel, una riga/colonna non verrà presa come vuota se ha uno stile personalizzato, anche se non contiene dati di cella. Quando si salva con la modalità LightCells, questa opzione non ha effetto. L'utente dovrebbe controllare la gamma di uscita mediante l'implementazione di[`LightCellsDataProvider`](../lightcellsdataprovider) o specificando[`ExportArea`](../exportarea)
