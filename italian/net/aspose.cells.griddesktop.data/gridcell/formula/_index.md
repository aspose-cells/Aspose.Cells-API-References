## GridCell.Formula property

Ottiene o imposta una formula diCell .

```csharp
public string Formula { get; set; }
```

### Osservazioni

Una stringa di formula inizia sempre con un segno di uguale (=). E utilizza sempre virgola(,) come delimitatore di parametri, ad esempio "=SOMMA(A1, E1, H2)".

L'utente può impostare qualsiasi formula nel file Designer cartella di lavoro. Aspose.Cells manterrà tutte le formule. Se l'utente utilizza questa proprietà per impostare una formula su una cella, la maggior parte delle funzioni integrate nella cartella di lavoro è supportata. E altro sta arrivando. Se hai esigenze particolari per le funzioni integrate di Workbook, faccelo sapere.

### Esempi

```csharp
[C#]
	cell.Formula = "=SUM(A1:C3) + E6*2";
[Visual Basic]
	cell.Formula = "=SUM(A1:C3) + E6*2"
```
