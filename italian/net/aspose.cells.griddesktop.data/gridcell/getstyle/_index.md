## GridCell.GetStyle method

Ottiene lo stile della cella. Quando si cambia lo stile, è necessario invocare il metodo "SetStyle", per impostare lo stile sulla cella.

```csharp
public Style GetStyle()
```

### Valore di ritorno

restituire la copia dello stile della cella.

### Esempi

```csharp
[C#]
Style style = sheet.GetCell(0, 0).GetStyle();
style.CellLocked = true;
style.VAlignment = VerticalAlignmentType.Top;
...
sheet.GetCell(0, 0).SetStyle(style);

[Visual Basic]
Dim style As Style =  sheet.GetCell(0,0).GetStyle() 
style.CellLocked = True
style.VAlignment = VerticalAlignmentType.Top
...
sheet.GetCell(0, 0).SetStyle(style)

```
