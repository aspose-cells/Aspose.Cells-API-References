## GridCell.Protected property

Indica se la cella è protetta. Se il valore è "true", l'utente non può modificare la cella tramite l'interfaccia utente. Questo attributo non ha nulla a che fare con la proprietà Style.CellLocked e non verrà salvato su file quando dati griglia esportati. Il valore predefinito è "false".

public bool Protected { get; set; }
