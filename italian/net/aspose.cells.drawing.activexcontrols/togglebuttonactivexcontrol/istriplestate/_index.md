## ToggleButtonActiveXControl.IsTripleState property

Indica come il controllo specificato visualizzerà i valori Null.

```csharp
public bool IsTripleState { get; set; }
```

### Osservazioni

**Ambientazione**

**Descrizione**

Vero

Il controllo scorrerà gli stati per i valori Sì, No e Null. Il controllo viene visualizzato in grigio (grigio) quando la relativa proprietà Value è impostata su Null.

Falso

(Predefinito) Il controllo scorrerà gli stati per i valori Sì e No. I valori Null vengono visualizzati come se fossero Nessun valore.
