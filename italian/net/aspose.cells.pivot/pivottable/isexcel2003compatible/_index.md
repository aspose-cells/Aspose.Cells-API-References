## PivotTable.IsExcel2003Compatible property

Specifica se la tabella pivot è compatibile con Excel2003 durante l'aggiornamento della tabella pivot, se true, una stringa deve essere minore o uguale a 255 caratteri, quindi se la stringa è maggiore di 255 caratteri, verrà troncata. se false, una stringa non avrà la suddetta limitazione. Il valore predefinito è true.

public bool IsExcel2003Compatible { get; set; }
