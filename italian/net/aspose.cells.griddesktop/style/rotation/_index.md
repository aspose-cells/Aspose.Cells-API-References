## Style.Rotation property

Rappresenta l'angolo di rotazione del testo.

```csharp
public int Rotation { get; set; }
```

### Osservazioni

0: non ruotato.

255: dall'alto verso il basso.

-90: Verso il basso.

90: Verso l'alto.

È possibile impostare 255 o un valore compreso tra -90 e 90.
