## GridDesktop.StartFormatPainter method

Notifica a GridDesktop di avviare FormatPainter.

```csharp
public void StartFormatPainter(bool formatOnce)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| formatOnce | Boolean | se formatOnce è true, FormatPainter può utilizzare una volta, se formatOnce è false, FormatPainter può utilizzare più volte, finché non invoca EndFormatPainter(). |

### Esempi

```csharp
[C#]
gridDesktop1.StartFormatPainter(true);

[Visual Basic]
gridDesktop1.StartFormatPainter(True)

```
