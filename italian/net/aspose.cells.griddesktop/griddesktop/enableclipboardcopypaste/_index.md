## GridDesktop.EnableClipboardCopyPaste property

Indica se copiare/incollare in base agli appunti, in modo che possa copiare/incollare con MS-EXCEL. Copia/incolla solo il valore della cella, Non copia nessun'altra impostazione della cella come il formato, lo stile del bordo e così via Il valore predefinito è false.

public bool EnableClipboardCopyPaste { get; set; }
