## BuiltInDocumentPropertyCollection.LastPrinted property

Ottiene o imposta la data dell'ultima stampa del documento nel fuso orario locale.

```csharp
public DateTime LastPrinted { get; set; }
```

### Osservazioni

Se il documento non è mai stato stampato, questa proprietà restituirà DateTime.MinValue.

Aspose.Cells non aggiorna questa proprietà quando si modifica il documento.
