## WebQueryConnection.EditWebPage property

L'URL della pagina Web rivolta all'utente che mostra i dati della query Web. Questo URL è persistente nel caso in cui sourceData="true" e url siano stati reindirizzati per fare riferimento a un file XML. Quindi la pagina rivolta all'utente può essere mostrata nell'interfaccia utente e i dati XML possono essere recuperati dietro le quinte.
