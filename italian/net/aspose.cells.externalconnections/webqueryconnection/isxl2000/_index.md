## WebQueryConnection.IsXl2000 property

Questo flag esiste per la compatibilità con le versioni precedenti con file di fogli di calcolo esistenti precedenti ed è impostato su true se questa query Web è stata aggiornata in un'applicazione per fogli di calcolo più recente o uguale a a Microsoft Excel 2000. Questo è un attributo facoltativo che può essere ignorato.
