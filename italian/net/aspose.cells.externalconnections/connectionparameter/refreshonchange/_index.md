## ConnectionParameter.RefreshOnChange property

Flag che indica se la query deve essere aggiornata automaticamente quando il contenuto di una cella che fornisce il valore del parametro cambia. Se true, i dati esterni vengono aggiornati utilizzando il nuovo valore del parametro ogni volta che viene apportata una modifica. Se false, i dati esterni vengono aggiornati solo su richiesta dell'utente o qualche altro evento attiva l'aggiornamento (ad esempio, cartella di lavoro aperta).

public bool RefreshOnChange { get; set; }
