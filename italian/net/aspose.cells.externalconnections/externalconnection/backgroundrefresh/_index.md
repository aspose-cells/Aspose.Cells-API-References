## ExternalConnection.BackgroundRefresh property

Indica se la connessione può essere aggiornata in background (in modo asincrono). true se l'utilizzo preferito della connessione prevede l'aggiornamento asincrono in background; false se l'utilizzo preferito della connessione prevede l'aggiornamento sincrono in primo piano.
