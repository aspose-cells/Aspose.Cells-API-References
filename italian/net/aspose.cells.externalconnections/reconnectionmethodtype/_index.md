## ReConnectionMethodType enumeration

Specifica cosa deve fare l'applicazione del foglio di calcolo quando una connessione non riesce.

```csharp
public enum ReConnectionMethodType
```

### I valori

| Nome | Valore | Descrizione |
| --- | --- | --- |
| Required | `1` | Durante l'aggiornamento, utilizzare le informazioni di connessione esistenti e, se risulta non essere valido , ottenere le informazioni di connessione aggiornate, se disponibili dal file di connessione esterno. |
| Always | `2` | Ad ogni aggiornamento, ottieni informazioni di connessione aggiornate dal file di connessione esterno, se disponibile, e usalo invece delle informazioni di connessione esistenti. In questo caso l'aggiornamento dei dati avrà esito negativo se il file di connessione esterno non è disponibile. |
| Never | `3` | Non ottenere mai informazioni di connessione aggiornate dal file di connessione esterno anche se è disponibile e anche se le informazioni di connessione esistenti non sono valide |
