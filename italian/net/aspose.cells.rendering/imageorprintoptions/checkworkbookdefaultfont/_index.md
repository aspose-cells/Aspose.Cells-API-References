## ImageOrPrintOptions.CheckWorkbookDefaultFont property

Quando i caratteri in Excel sono Unicode e non possono essere impostati con il carattere corretto nello stile della cella, Possono apparire come blocchi in pdf, immagine. Impostalo su true per provare a utilizzare il carattere predefinito della cartella di lavoro per mostrare prima questi caratteri.

```csharp
public bool CheckWorkbookDefaultFont { get; set; }
```

### Osservazioni

L'impostazione predefinita è true.
