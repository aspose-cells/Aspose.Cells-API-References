## ImageOrPrintOptions.TiffColorDepth property

Ottiene o imposta la profondità di bit da applicare solo quando si salvano le pagine nel file`Tiff` formato.

```csharp
public ColorDepth TiffColorDepth { get; set; }
```

### Osservazioni

Ha effetto solo durante il salvataggio in TIFF. Se TiffCompression è impostato su CCITT3, CCITT4, ciò non avrà effetto, la profondità di bit dell'immagine tiff generata sarà sempre 1.
