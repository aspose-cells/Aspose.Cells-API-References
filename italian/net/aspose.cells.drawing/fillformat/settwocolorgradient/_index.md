## SetTwoColorGradient(Color, Color, GradientStyleType, int) {#settwocolorgradient_1}


| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| color1 | Color | Un colore sfumato. |
| color2 | Color | Due sfumature di colore. |
| style | GradientStyleType | Stile di ombreggiatura sfumata. |
| variant | Int32 | La variante sfumata. Può essere un valore compreso tra 1 e 4, corrispondente a una delle quattro varianti nella scheda Sfumatura nella finestra di dialogo Effetti di riempimento. Se style è GradientStyle.FromCenter, l'argomento Variant può essere solo 1 o 2. |
