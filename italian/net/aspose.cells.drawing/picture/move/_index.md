## Picture.Move method

Sposta l'immagine in una posizione specificata.

```csharp
public void Move(int upperLeftRow, int upperLeftColumn)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| upperLeftRow | Int32 | Indice della riga in alto a sinistra. |
| upperLeftColumn | Int32 | Indice della colonna in alto a sinistra. |

### Esempi

```csharp

[C#]
//Creazione di un'istanza di un oggetto cartella di lavoro
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
//Aggiunta di un'immagine nella posizione di una cella i cui indici di riga e colonna sono 1 nel foglio di lavoro. È la cella "B2".
int imgIndex = worksheet.Pictures.Add(1, 1, "example.jpeg");
//Ottieni l'oggetto immagine inserito
Picture pic = worksheet.Pictures[imgIndex];
//Imposta la nuova posizione dell'immagine
pic.Move(2, 4);
//Salva il file excel.
workbook.Save("result.xlsx");
```
