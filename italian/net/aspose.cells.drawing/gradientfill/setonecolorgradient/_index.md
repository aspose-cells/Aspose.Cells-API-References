## GradientFill.SetOneColorGradient method

Imposta il riempimento specificato su una sfumatura di un colore. Si applica solo a Excel 2007.

```csharp
public void SetOneColorGradient(Color color, double degree, GradientStyleType style, int variant)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| color | Color | Un colore sfumato. |
| degree | Double | Il grado di pendenza. Può essere un valore compreso tra 0,0 (scuro) e 1,0 (chiaro). |
| style | GradientStyleType | Stile di ombreggiatura sfumata. |
| variant | Int32 | La variante sfumata. Può essere un valore compreso tra 1 e 4, corrispondente a una delle quattro varianti nella scheda Sfumatura nella finestra di dialogo Effetti di riempimento. Se style è GradientStyle.FromCenter, l'argomento Variant può essere solo 1 o 2. |
