## ShapeCollection.AddAutoShape method

Aggiunge una forma al foglio di lavoro.

```csharp
public Shape AddAutoShape(AutoShapeType type, int upperLeftRow, int top, int upperLeftColumn, 
    int left, int height, int width)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| type | AutoShapeType | Tipo di forma automatica. |
| upperLeftRow | Int32 | Indice della riga in alto a sinistra. |
| top | Int32 | Rappresenta l'offset verticale di Shape dalla riga sinistra, in unità di pixel. |
| upperLeftColumn | Int32 | Indice della colonna in alto a sinistra. |
| left | Int32 | Rappresenta l'offset orizzontale di Shape dalla colonna di sinistra, in unità di pixel. |
| height | Int32 | Rappresenta l'altezza di Shape, in unità di pixel. |
| width | Int32 | Rappresenta la larghezza di Shape, in unità di pixel. |

### Valore di ritorno

Un oggetto Forma.

### Osservazioni

Il tipo non può essere Chart/Comment/Picture/OleObject/Polygon/DialogBox

### Esempi

```csharp

[C#]
//Aggiunge una forma al foglio di lavoro.
Shape autoShape = shapes.AddAutoShape(AutoShapeType.Cube, 1, 0, 1, 0, 100, 50);
```
