## ShapeCollection.AddComboBox method

Aggiunge un ComboBox al foglio di lavoro.

```csharp
public ComboBox AddComboBox(int upperLeftRow, int top, int upperLeftColumn, int left, int height, 
    int width)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| upperLeftRow | Int32 | Indice della riga in alto a sinistra. |
| top | Int32 | Rappresenta l'offset verticale di ComboBox dalla riga sinistra, in unità di pixel. |
| upperLeftColumn | Int32 | Indice della colonna in alto a sinistra. |
| left | Int32 | Rappresenta l'offset orizzontale di ComboBox dalla colonna di sinistra, in unità di pixel. |
| height | Int32 | Rappresenta l'altezza di ComboBox, in unità di pixel. |
| width | Int32 | Rappresenta la larghezza di ComboBox, in unità di pixel. |

### Valore di ritorno

Un oggetto ComboBox.

### Esempi

```csharp

[C#]
//aggiungi una casella combinata
ComboBox comboBox = shapes.AddComboBox(1, 0, 1, 0, 100, 50);
```
