## ShapeCollection.AddGroupBox method

Aggiunge un GroupBox al foglio di lavoro.

```csharp
public GroupBox AddGroupBox(int upperLeftRow, int top, int upperLeftColumn, int left, int height, 
    int width)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| upperLeftRow | Int32 | Indice della riga in alto a sinistra. |
| top | Int32 | Rappresenta l'offset verticale di GroupBox dalla riga sinistra, in unità di pixel. |
| upperLeftColumn | Int32 | Indice della colonna in alto a sinistra. |
| left | Int32 | Rappresenta l'offset orizzontale di GroupBox dalla colonna di sinistra, in unità di pixel. |
| height | Int32 | Rappresenta l'altezza di GroupBox, in unità di pixel. |
| width | Int32 | Rappresenta la larghezza di GroupBox, in unità di pixel. |

### Valore di ritorno

Un oggetto GroupBox.

### Esempi

```csharp

[C#]
//aggiungi una casella di gruppo
GroupBox groupBox = shapes.AddGroupBox(1, 0, 1, 0, 100, 50);
```
