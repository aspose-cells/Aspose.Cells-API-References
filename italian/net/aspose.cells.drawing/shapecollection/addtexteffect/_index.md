## ShapeCollection.AddTextEffect method

Inserisce un oggetto WordArt.

```csharp
public Shape AddTextEffect(MsoPresetTextEffect effect, string text, string fontName, int size, 
    bool fontBold, bool fontItalic, int upperLeftRow, int top, int upperLeftColumn, int left, 
    int height, int width)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| effect | MsoPresetTextEffect | Il tipo di effetto testo preimpostato per mso. |
| text | String | Il testo WordArt. |
| fontName | String | Il nome del carattere. |
| size | Int32 | La dimensione del carattere |
| fontBold | Boolean | Indica se il carattere è in grassetto. |
| fontItalic | Boolean | Indica se il carattere è in corsivo. |
| upperLeftRow | Int32 | Indice della riga in alto a sinistra. |
| top | Int32 | Rappresenta l'offset verticale della forma dalla riga sinistra, in unità di pixel. |
| upperLeftColumn | Int32 | Indice della colonna in alto a sinistra. |
| left | Int32 | Rappresenta l'offset orizzontale della forma dalla colonna di sinistra, in unità di pixel. |
| height | Int32 | Rappresenta l'altezza della forma, in unità di pixel. |
| width | Int32 | Rappresenta la larghezza della forma, in unità di pixel. |

### Valore di ritorno

Restituisce un oggetto Shape che rappresenta il nuovo oggetto WordArt.

### Esempi

```csharp

[C#]
//aggiungi una WordArt
Shape wordArt1 = shapes.AddTextEffect(MsoPresetTextEffect.TextEffect10, "WordArt", "arial", 18, false, false, 3, 0, 3, 0, 200, 50);
```
