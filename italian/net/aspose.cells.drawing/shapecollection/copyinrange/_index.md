## ShapeCollection.CopyInRange method

Copia le forme nell'intervallo nell'intervallo di destinazione.

```csharp
public void CopyInRange(ShapeCollection sourceShapes, CellArea ca, int destRow, int destColumn, 
    bool isContained)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| sourceShapes | ShapeCollection | Forme di origine. |
| ca | CellArea | La gamma di sorgenti. |
| destRow | Int32 | L'indice di riga dest dell'intervallo dest. |
| destColumn | Int32 | La colonna dest dell'intervallo dest. |
| isContained | Boolean | Se copia solo le forme contenute nell'intervallo. Se true, copia solo le forme nell'intervallo. Altrimenti funziona come MS Office. |

### Esempi

```csharp

[C#]
//aggiungi una forma
shapes.AddRectangle(2, 0, 2, 0, 130, 130);
CellArea area2 = new CellArea();
area2.StartColumn = 1;
area2.StartRow = 1;
area2.EndColumn = 5;
area2.EndRow = 11;

//copia
shapes.CopyInRange(shapes, area2, 12, 1, false);

```
