```csharp
public TextBox AddTextBox(int upperLeftRow, int top, int upperLeftColumn, int left, int height,

```

| upperLeftRow | Int32 | Indice della riga in alto a sinistra. |
| top | Int32 | Rappresenta l'offset verticale della casella di testo dalla riga sinistra, in unità di pixel. |
| upperLeftColumn | Int32 | Indice della colonna in alto a sinistra. |
| left | Int32 | Rappresenta l'offset orizzontale della casella di testo dalla colonna di sinistra, in unità di pixel. |
| height | Int32 | Rappresenta l'altezza della casella di testo, in unità di pixel. |
