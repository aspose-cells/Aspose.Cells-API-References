## ShapeCollection.AddIcons method

Aggiunge immagine svg.

```csharp
public Picture AddIcons(int upperLeftRow, int top, int upperLeftColumn, int left, int height, 
    int width, byte[] imageByteData, byte[] compatibleImageData)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| upperLeftRow | Int32 | Indice della riga in alto a sinistra. |
| top | Int32 | Rappresenta l'offset verticale della forma dalla riga sinistra, in unità di pixel. |
| upperLeftColumn | Int32 | Indice della colonna in alto a sinistra. |
| left | Int32 | L'offset orizzontale della forma dalla colonna di sinistra, in unità di pixel. |
| height | Int32 | L'altezza della forma, in unità di pixel. |
| width | Int32 | La larghezza della forma, in unità di pixel. |
| imageByteData | Byte[] | I dati del byte dell'immagine. |
| compatibleImageData | Byte[] | Dati immagine convertiti da svg per essere compatibili con Excel 2016 o versioni precedenti. |

### Esempi

```csharp

[C#]
//aggiungi icona
using (FileStream fs = new FileStream("icon.svg", FileMode.Open))

    int len = (int)fs.Length;
    byte[] imageData = new byte[len];

}
```
