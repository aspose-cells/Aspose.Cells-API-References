## SparklineGroup.ResetRanges method

Reimposta l'intervallo di dati e l'intervallo di posizioni del gruppo sparkline. Questo metodo cancellerà gli elementi sparkline originali nel gruppo e creerà nuovi elementi sparkline per i nuovi intervalli.

```csharp
public void ResetRanges(string dataRange, bool isVertical, CellArea locationRange)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| dataRange | String | Specifica il nuovo intervallo di dati del gruppo sparkline. |
| isVertical | Boolean | Specifica se tracciare i grafici sparkline dal nuovo intervallo di dati per riga o per colonna. |
| locationRange | CellArea | Specifica dove posizionare i grafici sparkline. |
