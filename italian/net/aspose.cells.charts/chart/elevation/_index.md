## Chart.Elevation property

Rappresenta l'elevazione della vista mappa 3D, in gradi.

```csharp
public int Elevation { get; set; }
```

### Osservazioni

L'elevazione della carta è l'altezza alla quale si visualizza la carta, in gradi. Il valore predefinito è 15 per la maggior parte dei tipi di grafici. Il valore di questa proprietà deve essere compreso tra -90 e 90, ad eccezione dei grafici a barre 3D, dove deve essere compreso tra 0 e 44.
