## Chart.IsChartDataChanged method

Rileva se l'origine dati di un grafico è cambiata.

```csharp
public bool IsChartDataChanged()
```

### Valore di ritorno

Restituisce vero se il grafico è cambiato altrimenti restituisce falso

### Osservazioni

Il metodo rileva le modifiche nell'origine dati del grafico prima di eseguire il rendering del grafico in formato immagine. Alla prima chiamata a Chart.toImage, i dati dell'origine del grafico (ad es. XValuesParseData, ValuesParseData) verranno registrati. Prima di richiamare nuovamente il metodo Chart.toImage , chiama il metodo IsChartDataChanged per verificare se il grafico necessita di un nuovo rendering.
