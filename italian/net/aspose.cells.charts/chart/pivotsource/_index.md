## Chart.PivotSource property

L'origine sono i dati della tabella pivot. Se PivotSource non è vuota, il grafico è Grafico pivot.

public string PivotSource { get; set; }

### Osservazioni

Se la tabella pivot "Tabella pivot1" nel foglio di lavoro "Foglio1" nel file "Book1.xls". L'origine pivot potrebbe essere "[Book1.xls]Sheet1!PivotTable1" se il grafico e la tabella pivot non si trovano nella stessa cartella di lavoro. Se si imposta questa proprietà, l'impostazione dell'origine dati precedente andrà persa.
