## Chart.RotationAngle property

Rappresenta la rotazione della vista del grafico 3D (la rotazione dell'area del tracciato attorno all'asse z, in gradi).

```csharp
public int RotationAngle { get; set; }
```

### Osservazioni

Il valore di questa proprietà deve essere compreso tra 0 e 360, ad eccezione dei grafici a barre 3D, dove il valore deve essere compreso tra 0 e 44. Il valore predefinito è 20. Si applica solo ai grafici 3D.
