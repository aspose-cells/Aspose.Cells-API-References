## TickLabels.Offset property

Ottiene e imposta la distanza delle etichette dall'asse.

```csharp
public int Offset { get; set; }
```

### Osservazioni

La distanza predefinita è 100 percento, che rappresenta la spaziatura predefinita tra le etichette dell'asse e la linea dell'asse. Il valore può essere una percentuale intera da 0 a 1000, rispetto alla dimensione del carattere dell'etichetta dell'asse.
