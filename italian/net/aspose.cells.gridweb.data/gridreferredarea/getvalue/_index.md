## GridReferredArea.GetValue method

Ottiene il valore della cella con un determinato offset dall'angolo in alto a sinistra di quest'area.

```csharp
public object GetValue(int rowOffset, int colOffset)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| rowOffset | Int32 | offset di riga dalla riga iniziale di quest'area |
| colOffset | Int32 | offset di colonna dalla riga iniziale di quest'area |
