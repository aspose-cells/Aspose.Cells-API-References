## GridValidation.GetListValue method

Ottieni il valore per l'elenco della convalida per la cella specificata.

```csharp
public object GetListValue(int row, int column)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| row | Int32 | L'indice di riga. |
| column | Int32 | L'indice di colonna. |

### Valore di ritorno

Il valore per produrre l'elenco di questa convalida per la cella specificata. Se l'elenco fa riferimento a un intervallo, il valore restituito sarà un oggetto ReferredArea; Altrimenti il valore restituito potrebbe essere null, oggetto[] o oggetto semplice.

### Osservazioni

Solo per la convalida il cui tipo è List ed è stato applicato alla cella data, altrimenti verrà restituito null.
