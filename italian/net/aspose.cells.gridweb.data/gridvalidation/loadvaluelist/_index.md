## GridValidation.LoadValueList method

Carica la lista valori da un oggetto DataView. È possibile specificare il campo del valore e il campo di testo di DataView. Ciascun valore e coppia di testo verranno combinati in una stringa, con una virgola tra di loro.

```csharp
public void LoadValueList(DataView view, string valueField, string textField, bool valuePrefixText)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| view | DataView | L'oggetto DataView. |
| valueField | String | Il nome del campo del valore. |
| textField | String | Il nome del campo di testo. |
| valuePrefixText | Boolean | Indica se aggiungere un prefisso di valore a ogni testo visualizzato. |
