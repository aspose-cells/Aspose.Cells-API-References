## Sort(int, int, int, int, int, bool, bool, bool) {#sort}

Ordina i dati ascendente/decrescente dall'alto verso il basso in un intervallo di un foglio di lavoro in base all'indice di colonna specificato. Ordina i dati ascendente/decrescente da sinistra a destra in un intervallo di un foglio di lavoro in base all'indice di riga specificato.

```csharp
public void Sort(int startRow, int startColumn, int rows, int columns, int index, bool isAsending, 
    bool isCaseSensitive, bool islefttoright)
```

| Parametro | Tipo | Descrizione |
| --- | --- | --- |
| startRow | Int32 | Il numero di riga della prima cella da ordinare. |
| startColumn | Int32 | Il numero di colonna della prima cella da ordinare. |
| rows | Int32 | Numero di righe da importare. |
| columns | Int32 | Numero di colonne da importare. |
| index | Int32 | L'indice della colonna che specifica la colonna di ordinamento. se l'Orientamento è dall'alto verso il basso, sta per l'indice di colonna che specifica la colonna di ordinamento. se l'Orientamento è da sinistra a destra, sta per l'indice di riga che specifica l'ordinamento riga. |
| isAsending | Boolean | se l'ordine di ordinamento è ascendente. |
| isCaseSensitive | Boolean | se l'ordinamento fa distinzione tra maiuscole e minuscole. |
| islefttoright | Boolean | se l'orientamento dell'ordinamento è da sinistra a destra |
