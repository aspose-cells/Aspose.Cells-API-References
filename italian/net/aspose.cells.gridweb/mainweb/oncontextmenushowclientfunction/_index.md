## MainWeb.OnContextMenuShowClientFunction property

Ottiene o imposta la funzione lato client da chiamare quando verrà visualizzato il menu contestuale. La funzione client dovrebbe essere dichiarata in questo modo: funzione suContextMenuShow() { var menu = event.srcElement; menu.setItemVisibility("Elimina", "Blocca"); menu.setItemVisibility("Aggiorna", "nessuno"); } Nota: puoi utilizzare il puntatore "questo" nella funzione client per puntare il controllo della griglia che attiva l'evento.
