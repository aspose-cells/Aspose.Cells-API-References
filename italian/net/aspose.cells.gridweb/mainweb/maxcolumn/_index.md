## MainWeb.MaxColumn property

Ottiene o imposta l'indice massimo della colonna di visualizzazione (in base zero) del foglio Web. Il controllo utilizza il valore maggiore di MaxColumn e la colonna massima dei dati del foglio.

```csharp
public int MaxColumn { get; set; }
```

```csharp
[C#]
```
