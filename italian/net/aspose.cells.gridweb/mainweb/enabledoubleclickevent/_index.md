## MainWeb.EnableDoubleClickEvent property

Ottiene o imposta se abilitare l'evento di doppio clic lato cliente.

```csharp
public bool EnableDoubleClickEvent { get; set; }
```

### Osservazioni

Quando è impostato su true, il controllo riceverà eventi di doppio clic sul lato cliente e tu potrai gestire questi eventi sul lato server. In caso contrario, il controllo ignorerà qualsiasi evento di doppio clic .
