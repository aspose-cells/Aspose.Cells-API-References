## MainWeb.MaxRow property

Ottiene o imposta l'indice massimo della riga di visualizzazione (in base zero) del foglio Web. Il controllo utilizza il valore maggiore di MaxRow e la riga massima dei dati del foglio.

```csharp
public int MaxRow { get; set; }
```

### Esempi

```csharp
[C#]
```
