## MainWeb.SessionMode property

الحصول على أو تعيين وضع جلسة الشبكة . هناك 4 أنواع من وضع الجلسة: 1. الجلسة (افتراضي): استخدم جلسة النظام لتخزين بيانات الورقة. بشكل عام ، يستخدم asp.net حالة جلسة InProc. تدعم الشبكة أيضًا "StateServer" خارج جلسة المعالجة state وحالة جلسة SQLServer. 2. حالة العرض: استخدم حالة عرض الصفحة لتخزين بيانات الورقة. 3. مخصص: استخدم أحداث LoadCustomData و SheetDataUpdated لتخزين / استرداد بيانات الورقة. 4. الملف: تخزين / استعادة بيانات الورقة في SessionStorePath.

```csharp
public SessionMode SessionMode { get; set; }
```

### ملاحظات

عند استخدام SessionMode.ViewState ، ستقوم الشبكة بتخزين بيانات الأوراق في حالة عرض الصفحة. سيؤدي هذا إلى تقليل استخدام ذاكرة الخادم ، ولكن حجم الصفحة سيكون أكبر وسيؤثر على الأداء العام.

### أمثلة

```csharp
[C#]
	GridWeb1.SessionMode = SessionMode.ViewState;
	
[Visual Basic]
	GridWeb1.SessionMode = SessionMode.ViewState
```
