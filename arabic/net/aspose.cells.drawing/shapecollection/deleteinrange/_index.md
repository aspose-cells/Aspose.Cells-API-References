## ShapeCollection.DeleteInRange method

احذف الأشكال الموجودة في النطاق. لن يتم حذف أشكال التعليقات.

```csharp
public void DeleteInRange(CellArea ca)
```

| معامل | يكتب | وصف |
| --- | --- | --- |
| ca | CellArea | النطاق: إذا كانت الأشكال موجودة في النطاق ، فسيتم إزالتها. |

### أمثلة

```csharp

[C#]
// إضافة الشكل الأول
shapes.AddRectangle(2, 0, 2, 0, 50, 50);
// أضف الشكل الثاني
shapes.AddRectangle(6, 0, 2, 0, 30, 30);

CellArea area3 = new CellArea();
area3.StartColumn = 0;
area3.StartRow = 5;
area3.EndColumn = 5;
area3.EndRow = 8;

// ديل
shapes.DeleteInRange(area3);
```
