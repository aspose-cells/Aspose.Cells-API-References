## Cell.GetPrecedentsInCalculation method

الحصول على كافة السابقات (مرجع للخلايا في المصنف الحالي) المستخدمة بواسطة صيغة هذه الخلية أثناء حسابها.

```csharp
public IEnumerator GetPrecedentsInCalculation()
```

### قيمة الإرجاع

العداد لتعداد جميع المراجع (RefifiedArea)

### ملاحظات

هذه الطريقة يمكن أن تعمل فقط مع الحالة التي[`EnableCalculationChain`](../../formulasettings/enablecalculationchain) صحيح للمصنف وتم حساب المصنف بالكامل. إذا لم تكن هذه الخلية صيغة أو لا تشير إلى أي خلايا أخرى ، فسيتم إرجاع القيمة الخالية.

### أمثلة

```csharp
[C#]

Workbook workbook = new Workbook();
Cells cells = workbook.Worksheets[0].Cells;
cells["A2"].Formula = "=IF(TRUE,B2,B1)";
workbook.Settings.FormulaSettings.EnableCalculationChain = true;
workbook.CalculateFormula();
IEnumerator en = cells["A2"].GetPrecedentsInCalculation();
Console.WriteLine("A2's calculation precedents:");
while(en.MoveNext())
{
    ReferredArea r = (ReferredArea)en.Current;
    Console.WriteLine(r);
}
```
