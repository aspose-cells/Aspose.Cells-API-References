## PdfSaveOptions.SetImageResample method

يضبط PPI المطلوب (بكسل لكل بوصة) للصور المعاد تشكيلها وجودة jpeg. سيتم تحويل جميع الصور إلى JPEG بإعداد الجودة المحدد ، وستتم إعادة تشكيل الصور الأكبر من PPI المحدد (بكسل لكل بوصة).

```csharp
public void SetImageResample(int desiredPPI, int jpegQuality)
```

| معامل | يكتب | وصف |
| --- | --- | --- |
| desiredPPI | Int32 | بكسل المطلوب في البوصة. 220 جودة عالية. 150 شاشة بجودة. 96 جودة البريد الإلكتروني. |
| jpegQuality | Int32 | 0-100٪ جودة JPEG. |

### أمثلة

تعيّن الكود التالي PPI المطلوب بجودة 96 و jpeg كـ 80 للصور في pdf.

```csharp
// تحميل الملف المصدر بالصور.
Workbook wb = new Workbook("Book1.xlsx");

PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();

// قم بتعيين PPI المطلوب على 96 وجودة jpeg على 80.
pdfSaveOptions.SetImageResample(96, 80);

wb.Save("output.pdf", pdfSaveOptions);
```
