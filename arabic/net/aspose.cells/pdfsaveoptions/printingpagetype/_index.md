## PdfSaveOptions.PrintingPageType property

يشير إلى الصفحات التي لن تتم طباعتها .

```csharp
public PrintingPageType PrintingPageType { get; set; }
```

### ملاحظات

إذا كان المحتوى في الورقة متناثرًا ، فستكون هناك بعض الصفحات فارغة تمامًا في ملف pdf الناتج . إذا كنت لا تريد هذه الصفحات الفارغة ، فيمكنك استخدام هذا الخيار لحذفها.

### أمثلة

يحذف الكود التالي الصفحات أو الصفحات الفارغة التي تحتوي فقط على بعض محتوى النمط مثل خلفية الخلية والحدود.

```csharp
Workbook wb = new Workbook("Book1.xlsx");

PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();

// تجاهل الصفحات الفارغة
pdfSaveOptions.PrintingPageType = PrintingPageType.IgnoreBlank;

wb.Save("output_ignore_blank_page.pdf", pdfSaveOptions);


// تجاهل الصفحات الفارغة and pages which only contains some style content like cell background
pdfSaveOptions.PrintingPageType = PrintingPageType.IgnoreStyle;

wb.Save("output_ignore_blank_and_style_page.pdf", pdfSaveOptions);
```
