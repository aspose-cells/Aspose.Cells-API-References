| معامل | يكتب | وصف |

| type | ProtectionType | نوع الحماية. |
| password | String | كلمة المرور. |
| oldPassword | String | إذا كانت ورقة العمل محمية بالفعل بكلمة مرور ، فالرجاء توفير كلمة المرور القديمة. وإلا ، يمكنك تعيين قيمة فارغة أو سلسلة فارغة لهذه المعلمة. |

### ملاحظات

يمكن لهذه الطريقة حماية ورقة العمل في جميع إصدارات ملف Excel.

### أمثلة

```csharp

[C#]

// إنشاء كائن مصنف
Workbook excel = new Workbook("template.xlsx");
// الوصول إلى ورقة العمل الأولى في ملف Excel
Worksheet worksheet = excel.Worksheets[0];
// حماية ورقة العمل بكلمة مرور
worksheet.Protect(ProtectionType.All, "aspose", null);
// حفظ ملف Excel المعدل بالتنسيق الافتراضي (أي Excel 20003)
excel.Save("output.xls");
// إغلاق دفق الملف لتحرير جميع الموارد

[Visual Basic]

'إنشاء دفق ملف يحتوي على ملف Excel ليتم فتحه
```
