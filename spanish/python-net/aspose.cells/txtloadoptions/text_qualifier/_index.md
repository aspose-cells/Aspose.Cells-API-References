##  text_qualifier propiedad

Especifica el calificador de texto para los valores de celda. El calificador predeterminado es '"'.

###  Observaciones

Al configurar esta propiedad, [TxtLoadOptions.has_text_qualifier](/cells/es/python-net/aspose.cells/txtloadoptions#has_text_qualifier) se hará realidad automáticamente.
