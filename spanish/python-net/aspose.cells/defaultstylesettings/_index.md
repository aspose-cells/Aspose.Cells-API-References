##  DefaultStyleSettings clase

El tipo DefaultStyleSettings expone los siguientes miembros:

###  Propiedades
| Propiedad| Descripción|
| :- | :- |
| [font_name](/cells/es/python-net/aspose.cells/defaultstylesettings/font_name) | Obtiene/establece el nombre de fuente predeterminado para el libro de trabajo|
| [font_size](/cells/es/python-net/aspose.cells/defaultstylesettings/font_size) | Obtiene/establece el tamaño de fuente estándar predeterminado para el libro de trabajo.|
| [horizontal_alignment](/cells/es/python-net/aspose.cells/defaultstylesettings/horizontal_alignment) | Obtiene/establece el valor predeterminado para la alineación horizontal|
| [vertical_alignment](/cells/es/python-net/aspose.cells/defaultstylesettings/vertical_alignment) | Obtiene/Establece el valor predeterminado para la alineación vertical|
