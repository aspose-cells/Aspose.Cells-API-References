##  FormulaSettings clase
Configuración de fórmulas y cálculos.



El tipo FormulaSettings expone los siguientes miembros:

###  Propiedades
| Propiedad| Descripción|
| :- | :- |
| [calculate_on_open](/cells/es/python-net/aspose.cells/formulasettings/calculate_on_open) | Indica si se requiere que la aplicación realice un cálculo completo cuando se abre el libro.|
| [calculate_on_save](/cells/es/python-net/aspose.cells/formulasettings/calculate_on_save) | Indica si se debe recalcular el libro de trabajo antes de guardar el documento, cuando se encuentra en el modo de cálculo manual.|
| [force_full_calculation](/cells/es/python-net/aspose.cells/formulasettings/force_full_calculation) | Indica si todas las fórmulas se calculan cada vez que se activa un cálculo.|
| [calculation_mode](/cells/es/python-net/aspose.cells/formulasettings/calculation_mode) | Obtiene o establece el modo para el cálculo del libro de trabajo en MS Excel.|
| [calculation_id](/cells/es/python-net/aspose.cells/formulasettings/calculation_id) |Especifica la versión del motor de cálculo utilizado para calcular valores en el libro.|
| [enable_iterative_calculation](/cells/es/python-net/aspose.cells/formulasettings/enable_iterative_calculation) | Indica si habilitar el cálculo iterativo para resolver referencias circulares.|
| [max_iteration](/cells/es/python-net/aspose.cells/formulasettings/max_iteration) | El máximo de iteraciones para resolver una referencia circular.|
| [max_change](/cells/es/python-net/aspose.cells/formulasettings/max_change) | El cambio máximo para resolver una referencia circular.|
| [precision_as_displayed](/cells/es/python-net/aspose.cells/formulasettings/precision_as_displayed) | Si la precisión del resultado calculado se establece tal como se muestra al calcular fórmulas|
| [enable_calculation_chain](/cells/es/python-net/aspose.cells/formulasettings/enable_calculation_chain) | Si habilita la cadena de cálculo para fórmulas.|
