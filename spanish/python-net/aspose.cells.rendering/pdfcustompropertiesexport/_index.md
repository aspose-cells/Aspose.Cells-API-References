##  PdfCustomPropertiesExport enumeración
Especifica la forma en que se exportan [CustomDocumentPropertyCollection](/cells/es/python-net/aspose.cells.properties/customdocumentpropertycollection) al archivo PDF.



El tipo PdfCustomPropertiesExport expone los siguientes miembros:

###  Campos
| Campo| Descripción|
| :- | :- |
| NONE | No se exportan propiedades personalizadas.|
| STANDARD | Las propiedades personalizadas se exportan como entradas en el diccionario de información.|
