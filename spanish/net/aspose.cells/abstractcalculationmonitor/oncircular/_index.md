## AbstractCalculationMonitor.OnCircular method

Implemente este método para hacer negocios al calcular fórmulas con referencias circulares.

```csharp
public virtual bool OnCircular(IEnumerator circularCellsData)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| circularCellsData | IEnumerator | IEnumerator con elementos CalculationCell que representan celdas que dependen de referencias circulares. |

### Valor_devuelto

Si el motor de fórmulas necesita calcular esas celdas en forma circular después de esta llamada. True para permitir que el motor de fórmulas continúe haciendo cálculos por ellas. False para permitir que el motor de fórmulas simplemente marque esas celdas como Calculadas.

### Observaciones

En la implementación, el usuario también puede establecer el valor esperado como resultado calculado para parte o todas esas celdas, de modo que el motor de fórmulas no las calcule recursivamente.
