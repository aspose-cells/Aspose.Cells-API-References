## PdfSaveOptions.PrintingPageType property

Indica qué páginas no se imprimirán.

```csharp
public PrintingPageType PrintingPageType { get; set; }
```

### Observaciones

Si el contenido de la hoja es escaso, habrá algunas páginas totalmente en blanco en el archivo pdf de salida. Si no desea estas páginas en blanco, puede usar esta opción para omitirlas.

### Ejemplos

El siguiente código omite páginas en blanco o páginas que solo contienen algún contenido de estilo como fondo de celda, bordes.

```csharp
Workbook wb = new Workbook("Book1.xlsx");

PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();

//ignorar paginas en blanco
pdfSaveOptions.PrintingPageType = PrintingPageType.IgnoreBlank;

wb.Save("output_ignore_blank_page.pdf", pdfSaveOptions);


//ignorar paginas en blanco and pages which only contains some style content like cell background
pdfSaveOptions.PrintingPageType = PrintingPageType.IgnoreStyle;

wb.Save("output_ignore_blank_and_style_page.pdf", pdfSaveOptions);
```
