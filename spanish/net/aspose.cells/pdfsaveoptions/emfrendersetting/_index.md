## PdfSaveOptions.EmfRenderSetting property

Configuración para representar el metarchivo Emf.

```csharp
public EmfRenderSetting EmfRenderSetting { get; set; }
```

### Observaciones

Los metarchivos EMF identificados como "EMF+ Dual" pueden contener tanto registros EMF+ como registros EMF. Se puede usar cualquier tipo de registro para renderizar la imagen, solo registros EMF+ o solo registros EMF. CuandoEmfPlusPrefer está configurado, los registros EMF+ se analizarán mientras se procesan en pdf; de lo contrario, solo se analizarán los registros EMF. El valor predeterminado esEmfOnly .
