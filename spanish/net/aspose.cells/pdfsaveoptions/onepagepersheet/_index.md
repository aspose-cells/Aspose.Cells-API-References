## PdfSaveOptions.OnePagePerSheet property

Si OnePagePerSheet es verdadero, todo el contenido de una hoja se generará en una sola página como resultado. El tamaño de papel de la configuración de la página no será válido y las demás configuraciones de la configuración de la página seguirán teniendo efecto.

public bool OnePagePerSheet { get; set; }
