## PdfSaveOptions.PageCount property

Obtiene o establece el número de páginas a guardar.

```csharp
public int PageCount { get; set; }
```

### Observaciones

El valor predeterminado es System.Int32.MaxValue, lo que significa que se representarán todas las páginas..

### Ejemplos

El siguiente ejemplo muestra cómo convertir un rango de páginas (3 y 4) en un archivo de Microsoft Excel a PDF.

```csharp
//Abrir un archivo de Excel
Workbook wb = new Workbook("Book1.xlsx");

PdfSaveOptions options = new PdfSaveOptions();

//Imprime solo la página 3 y la página 4 en el PDF de salida
//Índice de la página de inicio (índice basado en 0)
options.PageIndex = 3;
//Número de páginas a imprimir
options.PageCount = 2;

//Guardar el archivo PDF
wb.Save("output.pdf", options);
```
