## PdfSaveOptions.CheckFontCompatibility property

Indica si se debe comprobar la compatibilidad de fuentes para cada carácter del texto.

```csharp
public bool CheckFontCompatibility { get; set; }
```

### Observaciones

El valor predeterminado es verdadero. Deshabilitar esta propiedad puede brindar un mejor rendimiento. Pero cuando la fuente de texto/carácter predeterminada o especificada no se puede usar para representarlo, caracteres ilegibles (como bloques) pueden aparecer en el pdf generado. Para tal situación, el usuario debe mantener esta propiedad como verdadera para que la fuente alternativa pueda buscarse y usarse para representar el texto en su lugar;
