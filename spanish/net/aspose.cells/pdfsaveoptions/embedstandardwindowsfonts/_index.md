## PdfSaveOptions.EmbedStandardWindowsFonts property

True para incrustar fuentes de tipo verdadero. Afecta solo a los caracteres ASCII 32-127. Las fuentes para códigos de caracteres superiores a 127 siempre están incrustadas. Las fuentes siempre están incrustadas para PDF/A-1a, PDF/A-1b estándar. El valor predeterminado es verdadero.

public bool EmbedStandardWindowsFonts { get; set; }
