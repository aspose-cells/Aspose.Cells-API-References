## PdfSaveOptions.CheckWorkbookDefaultFont property

Cuando los caracteres en Excel son Unicode y no están configurados con la fuente correcta en el estilo de celda, Pueden aparecer como bloque en pdf,imagen. Establezca esto en verdadero para intentar usar la fuente predeterminada del libro de trabajo para mostrar estos caracteres primero.

```csharp
public bool CheckWorkbookDefaultFont { get; set; }
```

### Observaciones

El valor predeterminado es verdadero.
