## Range.Intersect method

Devuelve un[`Range`](../../range) objeto que representa la intersección rectangular de dos rangos.

```csharp
public Range Intersect(Range range)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| range | Range | El rango de intersección. |

### Valor_devuelto

Devuelve un[`Range`](../../range) objeto

### Observaciones

Si los dos rangos no se cruzan, devuelve nulo.

### Ejemplos

```csharp

[C#]

// Instanciando un objeto Workbook
Workbook workbook = new Workbook();
// Obtenga las primeras celdas de la hoja de trabajo.
Cells cells = workbook.Worksheets[0].Cells;
Range range1 = cells.CreateRange("A1:A5");
Range range2 = cells.CreateRange("A3:A10");
//Obtenga el rango de intersección de los dos rangos.
Range intersectRange = range1.Intersect(range2);
//Guardar el archivo Excel
workbook.Save("book1.xlsm");

 [Visual Basic]

'Crear una instancia de un objeto Workbook

```
