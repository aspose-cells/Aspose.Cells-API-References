## Copy(Range) {#copy}

Copia datos (incluidas fórmulas), formato, objetos de dibujo, etc. de un rango de origen.

```csharp
public void Copy(Range range)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| range | Range | Fuente[`Range`](../../range) objeto. |

### Ejemplos

```csharp

[C#]

// Instanciando un objeto Workbook
Workbook workbook = new Workbook();
// Obtenga las primeras celdas de la hoja de trabajo.
Cells cells = workbook.Worksheets[0].Cells;
Range range1 = cells.CreateRange("A1:A5");
Range range2 = cells.CreateRange("A6:A10");
//Copiar el rango.
range1.Copy(range2);
//Guardar el archivo Excel
workbook.Save("book1.xlsm");

 [Visual Basic]

'Crear una instancia de un objeto Workbook

'Obtenga las primeras celdas de la hoja de trabajo.

Range range2 = cells.CreateRange("A6:A10")
//Copiar el rango
range1.Copy(range2)
'Guarde el archivo de Excel
workbook.Save("book1.xlsm")
```
