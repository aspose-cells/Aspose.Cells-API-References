### Valor_devuelto

si esta celda debe mantenerse en el modelo de celdas de la hoja actual. Por lo general, debe ser falso para que todas las celdas no se mantengan en la memoria después de ser procesadas y luego se guarde la memoria. Para algún propósito especial, como las necesidades del usuario para acceder algunas celdas después de que se haya procesado todo el libro de trabajo, el usuario puede hacer que este método se vuelva verdadero para mantener esas celdas especiales en el modelo de Celdas y acceder a ellas más tarde mediante API como Celdas [fila, columna]. Sin embargo, manteniendo los datos de las celdas en Celdas El modelo requerirá más memoria y si se mantienen todas las celdas, la lectura del archivo de plantilla en el modo LightCells será igual a la lectura normal.

### Observaciones

Se llamará después de que se hayan leído los datos de una celda.
