## FormatConditionCollection.Add method

Agrega una condición de formato y un rango de celda afectado a FormatConditions FormatConditions puede contener hasta tres formatos condicionales. No se permiten referencias a las otras hojas en las fórmulas de formato condicional.

```csharp
public int[] Add(CellArea cellArea, FormatConditionType type, OperatorType operatorType, 
    string formula1, string formula2)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| cellArea | CellArea | Rango de celdas con formato condicional. |
| type | FormatConditionType | Tipo de formato condicional. Podría ser uno de los miembros de FormatConditionType. |
| operatorType | OperatorType | Operador de comparación. Podría ser uno de los miembros de OperatorType. |
| formula1 | String | El valor o la expresión asociada con el formato condicional. |
| formula2 | String | El valor o la expresión asociada con el formato condicional |

[0]: Índice de objeto de condición de formato; [1] Índice de rango de celda afectado.
