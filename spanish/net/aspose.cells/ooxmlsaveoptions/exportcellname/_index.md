## OoxmlSaveOptions.ExportCellName property

Indica si exportar el nombre de la celda al archivo Excel2007 .xlsx (.xlsm, .xltx, .xltm). Si SQL Server DTS puede acceder al archivo de salida, este valor debe ser verdadero. Establecer el valor en falso aumentará considerablemente el rendimiento y reducirá el tamaño del archivo al crear un archivo grande. El valor predeterminado es verdadero.
