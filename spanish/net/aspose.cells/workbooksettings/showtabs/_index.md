## WorkbookSettings.ShowTabs property

Obtener o establecer un valor si se muestran las pestañas del Libro de trabajo.

```csharp
public bool ShowTabs { get; set; }
```

### Observaciones

El valor por defecto es verdadero.

### Ejemplos

El siguiente código oculta las pestañas de hoja y los botones de desplazamiento de pestañas para la hoja de cálculo.
