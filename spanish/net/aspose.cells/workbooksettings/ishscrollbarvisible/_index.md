## WorkbookSettings.IsHScrollBarVisible property

Obtiene o establece un valor que indica si la hoja de cálculo generada contendrá una barra de desplazamiento horizontal.

```csharp
public bool IsHScrollBarVisible { get; set; }
```

### Observaciones

El valor por defecto es verdadero.

### Ejemplos

El siguiente código hace que la barra de desplazamiento horizontal sea invisible para la hoja de cálculo.

```csharp
[C#]
// Ocultar la barra de desplazamiento horizontal del archivo de Excel.
settings.IsHScrollBarVisible = false;

[Visual Basic]
' Oculte la barra de desplazamiento horizontal del archivo de Excel.
settings.IsHScrollBarVisible = False
```
