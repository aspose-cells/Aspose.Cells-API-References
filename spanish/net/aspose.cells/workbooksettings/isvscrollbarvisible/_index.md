## WorkbookSettings.IsVScrollBarVisible property

Obtiene o establece un valor que indica si la hoja de cálculo generada contendrá una barra de desplazamiento vertical.

```csharp
public bool IsVScrollBarVisible { get; set; }
```

### Observaciones

El valor por defecto es verdadero.

### Ejemplos

El siguiente código hace que la barra de desplazamiento vertical sea invisible para la hoja de cálculo.

```csharp
[C#]
// Ocultar la barra de desplazamiento vertical del archivo de Excel.
settings.IsVScrollBarVisible = false;

[Visual Basic]
' Oculte la barra de desplazamiento vertical del archivo de Excel.
settings.IsVScrollBarVisible = False
```
