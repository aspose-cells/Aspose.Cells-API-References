## ExportTableOptions.SkipErrorValue property

Indica si omite un valor no válido para la columna. Por ejemplo, si el tipo de columna es decimal, el valor es mayor que decimal. MaxValue y esta propiedad es verdadera, no volveremos a generar una excepción. El valor predeterminado es falso.
