## TextCrossType enumeration

Enumera el tipo de texto que se muestra cuando el ancho del texto es mayor que el ancho de la celda.

```csharp
public enum TextCrossType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| Default | `1` | Mostrar texto como en Microsoft Excel. |
| CrossKeep | `2` | Muestra todo el texto cruzando otras celdas y conserva el texto de las celdas cruzadas. |
| CrossOverride | `3` | Muestra todo el texto cruzando otras celdas y anula el texto de las celdas cruzadas. |
