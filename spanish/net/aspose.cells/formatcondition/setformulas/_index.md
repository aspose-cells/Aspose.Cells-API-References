## FormatCondition.SetFormulas method

Establece el valor o la expresión asociada a esta condición de formato.

```csharp
public void SetFormulas(string formula1, string formula2, bool isR1C1, bool isLocal)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| formula1 | String | El valor o expresión asociada con esta condición de formato. Si el valor de entrada comienza con '=', entonces se tomará como fórmula. De lo contrario, se tomará como valor simple (texto, número, bool). Para el valor de texto que comienza con '=', el usuario puede ingresarlo como fórmula en formato: "=\"=...\"". |
| formula2 | String | El valor o la expresión asociada con esta condición de formato. El formato de entrada es el mismo con formula1 |
| isR1C1 | Boolean | Si la fórmula es la fórmula R1C1. |
| isLocal | Boolean | Si la fórmula tiene formato de configuración regional. |
