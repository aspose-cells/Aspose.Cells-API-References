## FormulaSettings.CalculationMode property

Obtiene o establece el modo para el cálculo del libro de trabajo en ms excel.

```csharp
public CalcModeType CalculationMode { get; set; }
```

### Observaciones

Esta propiedad es solo para guardar la configuración en el archivo de hoja de cálculo resultante para que otras aplicaciones (como MS Excel) puedan actuar en consecuencia al cargar y manipular el archivo resultante. Para tener en cuenta el rendimiento de la mayoría de las aplicaciones de los usuarios, no calculamos ninguna fórmula en el libro de trabajo automáticamente, sin importar qué modo se haya establecido para esta propiedad. Si el usuario necesita calcular fórmulas, llame siempre a métodos en diferentes objetos de acuerdo con el requisito: [`CalculateFormula`](../../workbook/calculateformula) ,[`CalculateFormula`](../../worksheet/calculateformula) , [`Calculate`](../../cell/calculate) , ...etc.
