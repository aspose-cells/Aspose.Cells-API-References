## FormulaSettings.CalculateOnOpen property

Indica si se requiere que la aplicación realice un cálculo completo cuando se abre el libro.

```csharp
public bool CalculateOnOpen { get; set; }
```

### Observaciones

Esta propiedad es solo para guardar la configuración en el archivo de hoja de cálculo resultante para que otras aplicaciones (como MS Excel) puedan actuar en consecuencia al cargar el archivo resultante. Por consideraciones de rendimiento para la mayoría de las aplicaciones de los usuarios, no calculamos ninguna fórmula en el libro de trabajo automáticamente, sin importar qué valor se haya establecido para esta propiedad.
