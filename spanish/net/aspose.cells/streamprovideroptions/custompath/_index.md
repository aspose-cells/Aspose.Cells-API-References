## StreamProviderOptions.CustomPath property

La ruta personalizada del usuario (URL) guardada en el archivo html generado para la fuente referida. Si el usuario no lo define, se usará DefaultPath. Por ejemplo, el usuario guardará los datos de la hoja en d:/sheet001.htm, la URL utilizada en el archivo html principal debe ser "d:/sheet001.htm" o otra ruta relativa válida a la que se pueda acceder mediante el archivo html principal.

public string CustomPath { set; }
