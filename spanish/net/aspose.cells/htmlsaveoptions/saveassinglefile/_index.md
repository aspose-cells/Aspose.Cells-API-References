## HtmlSaveOptions.SaveAsSingleFile property

Indica si guardar el html como archivo único. El valor predeterminado es falso.

```csharp
public bool SaveAsSingleFile { get; set; }
```

### Observaciones

Si hay varias hojas de trabajo u otros recursos requeridos, como imágenes en el libro de trabajo, comúnmente esas hojas de trabajo y otros recursos deben guardarse en archivos separados. En algunos escenarios, es posible que el usuario necesite obtener solo un archivo resultante, por ejemplo, para facilitar la transferencia. Si es así, el usuario puede establecer esta propiedad como verdadera.
