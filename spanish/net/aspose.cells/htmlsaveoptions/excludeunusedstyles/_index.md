## HtmlSaveOptions.ExcludeUnusedStyles property

Indica si se excluyen estilos no utilizados. Para los archivos html generados, la exclusión de estilos no utilizados puede reducir el tamaño del archivo sin afectar los efectos visuales. Entonces, el valor predeterminado de esta propiedad es verdadero. Si el usuario necesita mantener todos los estilos en el libro de trabajo para el html generado (como el escenario en el que el usuario necesita restaurar el libro de trabajo del html generado más adelante), establezca esta propiedad como falsa .

```csharp
public bool ExcludeUnusedStyles { get; set; }
```
