## Style.InvariantCustom property

Obtiene la cadena de patrón independiente de la referencia cultural para el formato de número. Si no se ha establecido ningún formato de número para este objeto, se devolverá nulo. Si el formato de número está integrado, se devolverá la cadena de patrón correspondiente al número integrado.

```csharp
public string InvariantCustom { get; }
```

### Observaciones

Para los formatos numéricos integrados, el contenido del patrón devuelto aún depende de la cultura, como, para algunas configuraciones regionales, devuelve "m/d/y" y para otras configuraciones regionales devuelve "d/m/y". La diferencia de[`CultureCustom`](../culturecustom) es (eso es también lo que significa independiente de la cultura): los especificadores de formato y los separadores se mantienen como estándar, como '/' siempre se usará como separador de fecha y hora e "y" siempre se usará como el número de parte del "año" no importa qué otro carácter especial se use para la configuración regional específica.
