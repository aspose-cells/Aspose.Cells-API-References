## Workbook.DefaultStyle property

Obtiene o establece el valor predeterminado[`Style`](../../style) objeto del libro.

```csharp
public Style DefaultStyle { get; set; }
```

### Observaciones

La propiedad DefaultStyle es útil para implementar un estilo para todo el libro de trabajo.

### Ejemplos

El siguiente código crea e instancia un nuevo libro de trabajo y establece un valor predeterminado[`Style`](../../style) a ella.

```csharp
[C#]
Workbook workbook = new Workbook();
Style defaultStyle = workbook.DefaultStyle;
defaultStyle.Font.Name = "Tahoma";
workbook.DefaultStyle = defaultStyle;

[Visual Basic]
Dim workbook as Workbook = new Workbook()
Dim defaultStyle as Style = workbook.DefaultStyle
defaultStyle.Font.Name = "Tahoma"
workbook.DefaultStyle = defaultStyle
```
