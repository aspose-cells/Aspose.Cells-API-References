### Ejemplos

El siguiente código exportó los datos vinculados por el primer XmlMap.

```csharp
Workbook wb = new Workbook("Book1.xlsx");

//Asegúrese de que el archivo xlsx de origen contenga un XmlMap.
XmlMap xmlMap = wb.Worksheets.XmlMaps[0];

wb.ExportXml(xmlMap.Name, "output.xml");
```
