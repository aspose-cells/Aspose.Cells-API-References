## Validation.GetListValue method

Obtener el valor de la lista de validación para la celda especificada.

```csharp
public object GetListValue(int row, int column)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| row | Int32 | El índice de fila. |
| column | Int32 | El índice de la columna. |

### Valor_devuelto

El valor para generar la lista de esta validación para la celda especificada. Si la lista hace referencia a un rango, el valor devuelto será un[`ReferredArea`](../../referredarea) objeto; De lo contrario, el valor devuelto puede ser nulo, objeto[] u objeto simple.

### Observaciones

Solo para la validación cuyo tipo es Lista y se ha aplicado a la celda dada, de lo contrario, se devolverá nulo.
