## TxtSaveOptions.TrimLeadingBlankRowAndColumn property

Indica si las filas y columnas en blanco iniciales deben recortarse como lo hace MS Excel. El valor predeterminado es verdadero.

```csharp
public bool TrimLeadingBlankRowAndColumn { get; set; }
```

### Observaciones

Lo mismo con la regla en MS Excel, una fila/columna no se tomará como en blanco si tiene un estilo personalizado, incluso si no contiene datos de celda. Al guardar con el modo LightCells, esta opción no tiene efecto. El usuario debe controlar el rango de salida mediante la implementación de[`LightCellsDataProvider`](../lightcellsdataprovider) o especificando[`ExportArea`](../exportarea)
