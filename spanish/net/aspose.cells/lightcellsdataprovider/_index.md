### Observaciones

Al guardar un libro de trabajo en este modo,[`StartSheet`](./startsheet) se verificará al guardar cada hoja de trabajo en el libro de trabajo. Para una hoja, si[`StartSheet`](./startsheet) da verdadero, entonces todos los datos y propiedades de las filas/celdas de esta hoja que se van a guardar serán proporcionados por la implementación de esta interfaz. En primer lugar,[`NextRow`](./nextrow) se llamará para guardar el siguiente índice de fila. Si se devuelve un índice de fila válido (el índice de fila debe estar en orden ascendente para que se guarden las filas), entonces se proporcionará un objeto Fila que representa esta fila para implementación para establecer sus propiedades por[`StartRow`](./startrow) . Para una fila,[`NextCell`](./nextcell)se comprobará en primer lugar. Si se devuelve un índice de columna válido (el índice de columna debe estar en orden ascendente para que se guarden todas las celdas de una fila), entonces se proporcionará un objeto de celda que representa esta celda para que la implementación establezca sus datos y propiedades mediante[`StartCell`](./startcell) . Después de configurar los datos de esta celda, esta celda se guardará directamente en el archivo de hoja de cálculo generado y la siguiente celda se verificará y procesará.
