## LightCellsDataProvider.StartRow method

Comienza a guardar datos de una fila.

```csharp
public void StartRow(Row row)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| row | Row | Objeto de fila para implementación para completar datos. Su índice de fila es el valor devuelto de la última llamada de[`NextRow`](../nextrow) . Si la fila se ha inicializado en el modelo de celdas internas, se usará el objeto de fila existente. De lo contrario, se usará un objeto Fila temporal para la implementación para completar los datos. |

### Observaciones

Se llamará al comienzo de guardar una fila y los datos de sus celdas. Si la fila actual tiene algunas propiedades personalizadas, como altura, estilo, etc., la implementación de debe establecer esas propiedades en el objeto Fila dado aquí.
