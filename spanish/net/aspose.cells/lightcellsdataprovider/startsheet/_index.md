## LightCellsDataProvider.StartSheet method

Comienza a guardar una hoja de trabajo.

```csharp
public bool StartSheet(int sheetIndex)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| sheetIndex | Int32 | índice de la hoja actual que se va a guardar. |

### Valor_devuelto

verdadero si este proveedor proporcionará datos para la hoja dada; falso si la hoja dada debe usar su modelo de datos normal (Celdas).

### Observaciones

Se llamará al comienzo de guardar una hoja de trabajo durante el guardado de un libro de trabajo. Si el proveedor necesita consultar

```csharp
sheetIndex
```

later en el método startRow(Row) o startCell(Cell), es decir, si el proceso necesita saber qué hoja de cálculo se está procesando, la implementación debe conservar el

```csharp
sheetIndex
```

valor aquí.
