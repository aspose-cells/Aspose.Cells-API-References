## FormulaParseOptions.Parse property

Si analiza la fórmula dada. El valor predeterminado es verdadero. Si es falso, la cadena de fórmula dada se mantendrá como está para la celda hasta que el usuario llame a otros métodos para analizarlos o los datos de fórmula analizados sean requeridos por otras operaciones, como el cálculo de fórmulas.
