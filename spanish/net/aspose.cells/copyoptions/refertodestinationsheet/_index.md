## CopyOptions.ReferToDestinationSheet property

Al copiar el rango en el mismo archivo y el gráfico hace referencia a la hoja de origen, Falso significa que la fuente de datos del gráfico copiado no cambiará. Verdadero significa que la fuente de datos del gráfico copiado hace referencia a la hoja de destino.

```csharp
public bool ReferToDestinationSheet { get; set; }
```

### Observaciones

El valor por defecto es falso, funciona como MS Excel.
