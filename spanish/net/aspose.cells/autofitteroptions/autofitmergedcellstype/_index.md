## AutoFitterOptions.AutoFitMergedCellsType property

Obtiene y establece el tipo de altura de fila de ajuste automático de las celdas combinadas.

```csharp
public AutoFitMergedCellsType AutoFitMergedCellsType { get; set; }
```

### Observaciones

El valor predeterminado de Excel es ignorar las celdas combinadas al ajustar la altura de la fila, por lo que Aspose.Cells funciona como el valor predeterminado de MS Excel. Establezca este tipo para cambiar la forma de ajustar automáticamente la altura de la fila de las celdas combinadas.
