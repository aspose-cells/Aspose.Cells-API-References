## PivotPageFields.AddIdentify method

Establece qué etiqueta de elemento en cada campo de página usar para identificar el rango de datos. pageItemIndex.Length debe ser igual a PageFieldCount, así que agregue el campo de página primero.

```csharp
public void AddIdentify(int rangeIndex, int[] pageItemIndex)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| rangeIndex | Int32 | El índice de rango de datos de consolidación. |
| pageItemIndex | Int32[] | El índice de elementos de página en cada campo de página. pageItemIndex[2] = 1 significa que el segundo elemento en el tercer campo se usará para identificar este rango. pageItemIndex[1] = -1 significa que no hay ningún elemento en el segundo campo para usar identifique este rango y MS creará automáticamente un elemento "en blanco" en el segundo campo para identificar este rango. |
