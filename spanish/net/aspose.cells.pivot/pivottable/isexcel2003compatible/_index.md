## PivotTable.IsExcel2003Compatible property

Especifica si la tabla dinámica es compatible con Excel 2003 al actualizar la tabla dinámica, si es verdadero, una cadena debe tener 255 caracteres o menos, por lo que si la cadena tiene más de 255 caracteres, se truncará. si es falso, una cadena no tendrá la restricción mencionada anteriormente. El valor predeterminado es verdadero.

public bool IsExcel2003Compatible { get; set; }
