## GridDesktop.StartFormatPainter method

Notifica a GridDesktop para iniciar FormatPainter.

```csharp
public void StartFormatPainter(bool formatOnce)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| formatOnce | Boolean | si formatOnce es verdadero, FormatPainter puede usar una vez, si formatOnce es falso, FormatPainter puede usar varias veces, hasta que invoque EndFormatPainter(). |

### Ejemplos

```csharp
[C#]
gridDesktop1.StartFormatPainter(true);

[Visual Basic]
```
