## Worksheet.Name property

Obtiene o establece el nombre de la hoja de cálculo.

```csharp
public string Name { get; set; }
```

### Observaciones

La longitud máxima del nombre de la hoja es 31. Y no puede asignar el mismo nombre (sin distinción entre mayúsculas y minúsculas) a dos hojas de trabajo. Por ejemplo, no puede establecer "NombreHoja1" en la primera hoja de trabajo y establecer "NOMBREHOJA1" en la segunda hoja de trabajo.
