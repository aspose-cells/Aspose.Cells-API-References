## Insert(int) {#insert_1}

Inserta una hoja de trabajo en WorksheetCollection en el índice especificado. El número de filas de esta hoja de trabajo es 256, el número de columnas es 32. El nombre de la hoja de trabajo será generado por el método Add().

```csharp
public void Insert(int index)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| index | Int32 | El índice de base cero en el que se debe insertar la hoja de cálculo. |

### Valor_devuelto

Se agrega el objeto Return Worksheet.
