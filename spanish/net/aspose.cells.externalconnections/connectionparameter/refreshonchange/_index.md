## ConnectionParameter.RefreshOnChange property

Indicador que indica si la consulta debe actualizarse automáticamente cuando cambia el contenido de una celda que proporciona el valor del parámetro. Si es verdadero, los datos externos se actualizan usando el nuevo valor del parámetro cada vez que hay un cambio. Si es falso, los datos externos solo se actualizan cuando lo solicita el usuario, o algún otro evento activa la actualización (por ejemplo, libro de trabajo abierto).

```csharp
public bool RefreshOnChange { get; set; }
```
