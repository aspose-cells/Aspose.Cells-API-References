## ExternalConnection.OnlyUseConnectionFile property

Indica si la aplicación de hoja de cálculo debe usar siempre y solo la información de conexión en el archivo de conexión externo indicado por el atributo odcFile cuando se actualiza la conexión. Si es falso, la aplicación de hoja de cálculo debe seguir el procedimiento indicado por el atributo reconnectionMethod

public bool OnlyUseConnectionFile { get; set; }
