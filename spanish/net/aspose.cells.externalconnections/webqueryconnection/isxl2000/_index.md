## WebQueryConnection.IsXl2000 property

Este indicador existe para la compatibilidad con versiones anteriores de archivos de hojas de cálculo existentes más antiguos, y se establece en verdadero si esta consulta web se actualizó en una aplicación de hoja de cálculo más reciente o igual a Microsoft Excel 2000. Este es un atributo opcional que se puede ignorar.

public bool IsXl2000 { get; set; }
