## Chart.RotationAngle property

Representa la rotación de la vista de gráfico 3D (la rotación del área de trazado alrededor del eje z, en grados).

```csharp
public int RotationAngle { get; set; }
```

### Observaciones

El valor de esta propiedad debe ser de 0 a 360, excepto para gráficos de barras 3D, donde el valor debe ser de 0 a 44. El valor predeterminado es 20. Se aplica solo a gráficos 3D.
