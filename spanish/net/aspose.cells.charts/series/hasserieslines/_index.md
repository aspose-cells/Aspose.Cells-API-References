## Series.HasSeriesLines property

Verdadero si un gráfico de columnas apiladas o un gráfico de barras tiene líneas de serie o si un gráfico circular o un gráfico circular de barras tiene líneas conectoras entre las dos secciones. Solo se aplica a gráficos de columnas apiladas, gráficos de barras, gráficos circulares o gráficos circulares de barras.
