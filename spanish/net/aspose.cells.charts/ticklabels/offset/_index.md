## TickLabels.Offset property

Obtiene y establece la distancia de las etiquetas desde el eje.

```csharp
public int Offset { get; set; }
```

### Observaciones

La distancia predeterminada es 100 por ciento, que representa el espacio predeterminado entre las etiquetas de los ejes y la línea del eje. El valor puede ser un porcentaje entero de 0 a 1000, en relación con el tamaño de fuente de la etiqueta del eje.
