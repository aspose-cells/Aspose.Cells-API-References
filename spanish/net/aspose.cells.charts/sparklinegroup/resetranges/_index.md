## SparklineGroup.ResetRanges method

Restablece el rango de datos y el rango de ubicación del grupo minigráfico. Este método borrará los elementos minigráficos originales del grupo y creará nuevos elementos minigráficos para los nuevos rangos.

```csharp
public void ResetRanges(string dataRange, bool isVertical, CellArea locationRange)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| dataRange | String | Especifica el nuevo rango de datos del grupo minigráfico. |
| isVertical | Boolean | Especifica si trazar los minigráficos del nuevo rango de datos por fila o por columna. |
| locationRange | CellArea | Especifica dónde se colocarán los minigráficos. |
