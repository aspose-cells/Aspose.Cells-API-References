## GridCell.SetFontColor method

Establece el color de la fuente en la celda. Para mejorar el rendimiento, implemente el método "SetFontColor", no implemente la propiedad "FontColor".

```csharp
public void SetFontColor(Color color)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| color | Color | color de fuente a configurar. |

### Ejemplos

```csharp
[C#]
GridCell cell = gridDesktop1.GetActiveWorksheet().Cells[0, 0];
Font font = new Font("Courier New", 8, FontStyle.Italic);
cell.SetFont(font);
Color color = cell.GetFontColor();
color = Color.Black;
cell.SetFontColor(color);

[Visual Basic]

Dim font As Font =  New Font("Courier New",8,FontStyle.Italic) 
cell.SetFont(font)
Dim color As Color =  cell.GetFontColor()

```
