## GridCell.Protected property

Indica si la celda está protegida. Si el valor es "verdadero", el usuario no puede modificar la celda a través de la interfaz de usuario. Este atributo no tiene nada que ver con la propiedad Style.CellLocked y no se guardará en el archivo cuando datos de cuadrícula exportados. El valor predeterminado es "falso".

public bool Protected { get; set; }
