## SheetRender.PageScale property

Obtiene la escala de página calculada de la hoja. Devuelve la escala establecida si[`Zoom`](../../../aspose.cells/pagesetup/zoom) Está establecido. De lo contrario, devuelve la escala calculada según[`FitToPagesWide`](../../../aspose.cells/pagesetup/fittopageswide) y[`FitToPagesTall`](../../../aspose.cells/pagesetup/fittopagestall) .

```csharp
public double PageScale { get; }
```

### Ejemplos

```csharp
Workbook wb = new Workbook("Book1.xlsx");

SheetRender sheetRender = new SheetRender(wb.Worksheets[0], new ImageOrPrintOptions());

//Obtiene la escala de página calculada de la hoja.
double pageScale = sheetRender.PageScale;
```
