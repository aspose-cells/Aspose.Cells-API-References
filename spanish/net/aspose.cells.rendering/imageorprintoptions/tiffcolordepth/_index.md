## ImageOrPrintOptions.TiffColorDepth property

Obtiene o establece la profundidad de bits para que se aplique solo al guardar páginas en el`Pelea` formato.

```csharp
public ColorDepth TiffColorDepth { get; set; }
```

### Observaciones

Tiene efecto solo cuando se guarda en TIFF. Si TiffCompression se establece en CCITT3, CCITT4, esto no tendrá efecto, la profundidad de bits de la imagen tiff generada será siempre 1.
