## GridTableItemStyle.RotationAngle property

Obtiene o establece el atributo Rotación.

```csharp
public short RotationAngle { get; set; }
```

### Observaciones

0: No rotado.

255: de arriba a abajo.

-90: Hacia abajo.

90: Hacia arriba.

Puede establecer 255 o un valor en el rango de -90 a 90.
