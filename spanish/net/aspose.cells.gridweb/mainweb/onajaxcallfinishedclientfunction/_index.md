## MainWeb.OnAjaxCallFinishedClientFunction property

Obtiene o establece el nombre de la función del lado del cliente que se llamará cuando finalice ajaxcall. La función del cliente debe declararse así: función GridAjaxcallFinished() { alert(this.id+"llamada ajax finalizada"); } Nota: puede usar el puntero "este" en la función del cliente para señalar el control de cuadrícula que activa el evento.
