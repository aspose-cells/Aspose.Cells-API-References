## MainWeb.OnCellErrorClientFunction property

Obtiene o establece el nombre de la función del lado del cliente que se llamará cuando falle la validación de una celda. La función del cliente debe declararse así: función MyOnCellError(celda) { alerta(GridWeb1.getCellValueByCell(celda)); } Nota: puede usar el puntero "este" en la función del cliente para señalar el control de cuadrícula que activa el evento.

public string OnCellErrorClientFunction { get; set; }
