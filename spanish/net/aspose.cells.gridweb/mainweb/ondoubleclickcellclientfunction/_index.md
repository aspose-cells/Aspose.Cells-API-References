## MainWeb.OnDoubleClickCellClientFunction property

Obtiene o establece la función del lado del cliente que se llamará cuando se haga doble clic en una celda. La función del cliente debe declararse así: función MyOnDoubleClickCell(celda) { GridWeb1.setCellValueByCell(celda, "prueba"); } Nota: puede usar el puntero "este" en la función del cliente para señalar el control de cuadrícula que activa el evento.
