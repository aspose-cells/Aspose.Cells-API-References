## MainWeb.OnCellUpdatedClientFunction property

Obtiene o establece el nombre de la función del lado del cliente que se llamará cuando se actualice el valor de una celda. La función del cliente debe declararse así: función MyOnCellUpdated(celda) { alert(this.getCellValueByCell(cell)); } Nota: puede usar el puntero "este" en la función del cliente para señalar el control de cuadrícula que activa el evento.
