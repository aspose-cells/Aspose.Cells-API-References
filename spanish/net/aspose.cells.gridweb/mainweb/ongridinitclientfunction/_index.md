## MainWeb.OnGridInitClientFunction property

Obtiene o establece el nombre de la función del lado del cliente que se llamará cuando se inicialice la cuadrícula. La función del cliente debe declararse así: función MyOnGridInit(cuadrícula) { alert("El grid se inicializa: " + grid.id); } Nota: puede usar el puntero "este" en la función del cliente para señalar el control de cuadrícula que activa el evento.
