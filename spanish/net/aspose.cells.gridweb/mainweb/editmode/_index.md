## MainWeb.EditMode property

Obtiene o establece el modo de edición del control.

```csharp
public bool EditMode { get; set; }
```

### Observaciones

Cuando se establece en verdadero, las celdas del control son editables y se muestran los íconos "guardar" y "deshacer" . Cuando se establece en falso, las celdas del control son de solo lectura y no se muestran los íconos "guardar" y "deshacer".
