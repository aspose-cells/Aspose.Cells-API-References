## MainWeb.MinRow property

Obtiene o establece el índice mínimo de fila de visualización (basado en cero) de la hoja web. El control usa el valor más pequeño de MinRow y la fila mínima de los datos de la hoja.

```csharp
public int MinRow { get; set; }
```

### Ejemplos

```csharp
[C#]
	// Crea una "ventana de visualización" de 4x4.
	GridWeb1.MinRow = 2;
	GridWeb1.MaxRow = 5;
	GridWeb1.MinColumn = 3;
	GridWeb1.MaxColumn = 6;

[Visual Basic]
	' Creates a 4x4 "display window".
	GridWeb1.MinRow = 2
	GridWeb1.MaxRow = 5
	GridWeb1.MinColumn = 3
	GridWeb1.MaxColumn = 6
```
