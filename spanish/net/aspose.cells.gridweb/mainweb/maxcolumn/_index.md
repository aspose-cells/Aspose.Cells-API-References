## MainWeb.MaxColumn property

Obtiene o establece el índice máximo de la columna de visualización (basado en cero) de la hoja web. El control usa el valor mayor de MaxColumn y la columna máxima de datos de la hoja.

```csharp
public int MaxColumn { get; set; }
```

### Ejemplos

```csharp
[C#]
	// Crea una "ventana de visualización" de 4x4.
	GridWeb1.MinRow = 2;
	GridWeb1.MaxRow = 5;
	GridWeb1.MinColumn = 3;
	GridWeb1.MaxColumn = 6;

[Visual Basic]
	' Creates a 4x4 "display window".
	GridWeb1.MinRow = 2
	GridWeb1.MaxRow = 5
	GridWeb1.MinColumn = 3
	GridWeb1.MaxColumn = 6
```
