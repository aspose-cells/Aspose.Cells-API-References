## MainWeb.LoadCustomData event

Se activa cuando el Modo de sesión del control está configurado en Personalizado y necesita cargar datos de hoja. Puede manejar este evento en el modo Sesión personalizada para cargar datos de hoja desde un archivo o una base de datos.
