## MainWeb.OnPageChangeClientFunction property

Obtiene o configura la función del lado del cliente para que se llame después de cambiar el índice de la página. Solo tiene efecto cuando EnablePaging es verdadero. La función del cliente debe declararse así: función MyOnPageChange(índice) { console.log("la página actual es:"+índice); } Nota: puede usar el puntero "este" en la función del cliente para señalar el control de cuadrícula que activa el evento.

```csharp
public string OnPageChangeClientFunction { get; set; }
```
