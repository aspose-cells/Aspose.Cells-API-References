## Config.IgnoreEmptyContent property

Establece si mostrar el rango máximo que incluye datos, estilo, celdas combinadas y formas. el valor predeterminado es verdadero. Si la última fila o columna contiene celdas sin valor y fórmula pero tiene un estilo personalizado entonces no mostraremos esto fila/columna cuando este valor es true
