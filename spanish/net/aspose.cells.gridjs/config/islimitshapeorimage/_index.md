## Config.IslimitShapeOrImage property

Establece si se limita el recuento total de formas/imágenes de visualización dentro de una hoja de trabajo; si se establece en verdadero, GridJs limitará el tamaño total de forma/imagen de visualización dentro de una hoja de trabajo a MaxShapeOrImageCount el valor predeterminado es true
