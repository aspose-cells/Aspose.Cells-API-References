## GridValidation.ClientValidationFunction property

Obtiene o establece el nombre de la función javascript de validación del cliente.

```csharp
public string ClientValidationFunction { get; set; }
```

### Observaciones

Utilice la propiedad ClientValidationFunction para especificar el nombre de la función de validación del cliente. La función debe declararse con esta formación: función customValicationFunction(origen, valor) El parámetro "fuente" es el objeto de la celda. El parámetro "valor" es el valor de cadena de una celda que se va a comprobar. La función debería devolver verdadero si el valor es válido.
