## GridCell.Formula property

Obtiene o establece una fórmula delCell .

```csharp
public string Formula { get; set; }
```

### Observaciones

Una cadena de fórmula siempre comienza con un signo igual (=). Y utilice siempre comas (,) como delimitador de parámetros, como "=SUM(A1, E1, H2)".

El usuario puede configurar cualquier fórmula en el archivo del diseñador del libro de trabajo. Aspose.Cells mantendrá todas las fórmulas. Si el usuario usa esta propiedad para establecer una fórmula en una celda, se admite la mayor parte de las funciones integradas del Libro de trabajo . Y viene más. Si tiene alguna necesidad especial de las funciones integradas de Workbook, , háganoslo saber.

### Ejemplos

```csharp
[C#]
	cell.Formula = "=SUM(A1:C3) + E6*2";
[Visual Basic]
	cell.Formula = "=SUM(A1:C3) + E6*2"
```
