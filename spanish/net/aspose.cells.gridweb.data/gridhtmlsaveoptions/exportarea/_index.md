## GridHtmlSaveOptions.ExportArea property

Obtiene o establece el área de celda de exportación de la hoja de trabajo activa actual. Si establece este atributo, se omitirá el área de impresión de la hoja de trabajo activa actual. Solo se exportará el área especificada al guardar el archivo en html.
