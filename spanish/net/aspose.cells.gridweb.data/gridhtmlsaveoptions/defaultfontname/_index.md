## GridHtmlSaveOptions.DefaultFontName property

Especifique el nombre de la fuente predeterminada para exportar html, la fuente predeterminada se usará cuando la fuente del estilo no exista, Si esta propiedad es nula, Aspose.Cells usará la fuente universal que tiene la misma familia que la fuente original, el valor predeterminado es nulo.

public string DefaultFontName { get; set; }
