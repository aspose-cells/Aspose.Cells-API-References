## HyperlinkActionType enumeration

El hipervínculo tiene dos tipos de acción: UrlLink y CellCommand. El hipervínculo de tipo UrlLink es un ancla en una celda, haga clic en él para navegar a otra página o se abrirá una nueva ventana del navegador. El hipervínculo de tipo CellCommand es un botón de comando en una celda, haga clic en él generará un evento del lado del servidor.

```csharp
public enum HyperlinkActionType
```

### Valores

| Nombre | Valor | Descripción |
| --- | --- | --- |
| UrlLink | `0` | URL de enlace. Un ancla en una celda. |
| CellCommand | `1` | Botón de comando de celda. |
| SimpleImage | `2` | solo imagen simple |
