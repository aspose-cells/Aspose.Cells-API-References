## Sort(int, int, int, int, int, bool, bool, bool) {#sort}

Ordena los datos ascendentes/descendentes de arriba a abajo en un rango de una hoja de trabajo por índice de columna especificado. Ordena los datos ascendentes/descendentes de izquierda a derecha en un rango de una hoja de trabajo por índice de fila especificado.

```csharp
public void Sort(int startRow, int startColumn, int rows, int columns, int index, bool isAsending, 
    bool isCaseSensitive, bool islefttoright)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| startRow | Int32 | El número de fila de la primera celda para ordenar. |
| startColumn | Int32 | El número de columna de la primera celda a ordenar. |
| rows | Int32 | Número de filas a importar. |
| columns | Int32 | Número de columnas a importar. |
| index | Int32 | El índice de columna que especifica la columna de ordenación. si la Orientación es de arriba a abajo, representa el índice de columna que especifica la columna de ordenación. si la Orientación es de izquierda a derecha, representa el índice de fila que especifica la ordenación fila. |
| isAsending | Boolean | si el orden de clasificación está enviando. |
| isCaseSensitive | Boolean | si la ordenación distingue entre mayúsculas y minúsculas. |
| islefttoright | Boolean | si la orientación de clasificación es de izquierda a derecha |

### Ejemplos

```csharp

 [C#]

 GridWeb1.WebWorksheets[0].Cells.Sort(1,0,25,6,3,true,true,false);
```
