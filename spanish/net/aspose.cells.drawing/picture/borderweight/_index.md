## Picture.BorderWeight property

Obtiene o establece el peso de la línea del borde de una imagen en unidades de pt.

```csharp
public double BorderWeight { get; set; }
```

### Ejemplos

```csharp

[C#]
// Instanciando un objeto Workbook
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
// Agregar una imagen en la ubicación de una celda cuyos índices de fila y columna son 1 en la hoja de trabajo. Es celda "B2"
int imgIndex = worksheet.Pictures.Add(1, 1, "example.jpeg");
//Obtener el objeto de imagen insertado
Picture pic = worksheet.Pictures[imgIndex];
//Establecer el color del borde de la imagen
pic.BorderLineColor = Color.Red;
//Establecer el ancho del borde de la imagen
pic.BorderWeight = 3;
//Guardar el archivo de Excel.
workbook.Save("result.xlsx");
```
