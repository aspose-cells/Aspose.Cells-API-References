## Shape.SetLinkedCell method

Establece el rango vinculado al valor del control.

```csharp
public void SetLinkedCell(string formula, bool isR1C1, bool isLocal)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| formula | String | El rango vinculado al valor del control. |
| isR1C1 | Boolean | Si la fórmula debe formatearse como R1C1. |
| isLocal | Boolean | Si la fórmula debe formatearse según la configuración regional. |

### Ejemplos

```csharp

[C#]
//Después de ejecutar el código siguiente, se crea un objeto ScrollBar en el archivo generado. A medida que arrastra el control deslizante, el valor se muestra en la celda A12.

//Controles ActiveX
//Aspose.Cells.Drawing.Shape scrollBar = book.Worksheets[0].Shapes.AddActiveXControl( Aspose.Cells.Drawing.ActiveXControls.ControlType.ScrollBar,2, 0, 2, 0, 30, 130);

//Controles de formulario
Aspose.Cells.Drawing.Shape scrollBar = book.Worksheets[0].Shapes.AddScrollBar(2, 0, 2, 0, 130, 30);

//Establece el rango vinculado al valor del control.
scrollBar.SetLinkedCell("$A$12", false, true);

```
