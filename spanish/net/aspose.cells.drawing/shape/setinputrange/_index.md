## Shape.SetInputRange method

```csharp
public void SetInputRange(string formula, bool isR1C1, bool isLocal)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| formula | String | El rango utilizado para llenar el control. |
| isR1C1 | Boolean | Si la fórmula debe formatearse como R1C1. |
| isLocal | Boolean | Si la fórmula debe formatearse según la configuración regional. |
