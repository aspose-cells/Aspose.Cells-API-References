## MsoFillFormat.SetOneColorGradient method

Establece el relleno especificado en un degradado de un color.

```csharp
public void SetOneColorGradient(Color color, double degree, GradientStyleType style, int variant)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| color | Color | Un color degradado. |
| degree | Double | El grado de gradiente. Puede ser un valor de 0,0 (oscuro) a 1,0 (claro). |
| style | GradientStyleType | Estilo de sombreado degradado. |
| variant | Int32 | La variante degradada. Puede ser un valor de 1 a 4, correspondiente a una de las cuatro variantes en la pestaña Degradado en el cuadro de diálogo Efectos de relleno. Si el estilo es GradientStyle.FromCenter, el argumento Variant solo puede ser 1 o 2. |
