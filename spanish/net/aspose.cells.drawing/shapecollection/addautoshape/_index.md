## ShapeCollection.AddAutoShape method

Agrega una autoforma a la hoja de trabajo.

```csharp
public Shape AddAutoShape(AutoShapeType type, int upperLeftRow, int top, int upperLeftColumn, 
    int left, int height, int width)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| type | AutoShapeType | Tipo de forma automática. |
| upperLeftRow | Int32 | Índice de la fila superior izquierda. |
| top | Int32 | Representa el desplazamiento vertical de Shape desde su fila izquierda, en unidades de píxel. |
| upperLeftColumn | Int32 | Índice de la columna superior izquierda. |
| left | Int32 | Representa el desplazamiento horizontal de Shape desde su columna izquierda, en unidades de píxel. |
| height | Int32 | Representa la altura de Shape, en unidades de píxel. |
| width | Int32 | Representa el ancho de Shape, en unidades de píxel. |

### Valor_devuelto

Un objeto Forma.

### Observaciones

El tipo no puede ser Gráfico/Comentario/Imagen/OleObject/Polygon/DialogBox

### Ejemplos

```csharp

[C#]
//Agrega una autoforma a la hoja de trabajo.
Shape autoShape = shapes.AddAutoShape(AutoShapeType.Cube, 1, 0, 1, 0, 100, 50);
```
