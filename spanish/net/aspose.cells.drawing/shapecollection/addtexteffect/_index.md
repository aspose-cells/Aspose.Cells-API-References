## ShapeCollection.AddTextEffect method

Inserta un objeto de WordArt.

```csharp
public Shape AddTextEffect(MsoPresetTextEffect effect, string text, string fontName, int size, 
    bool fontBold, bool fontItalic, int upperLeftRow, int top, int upperLeftColumn, int left, 
    int height, int width)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| effect | MsoPresetTextEffect | El tipo de efecto de texto preestablecido de mso. |
| text | String | El texto de WordArt. |
| fontName | String | El nombre de la fuente. |
| size | Int32 | El tamaño de fuente |
| fontBold | Boolean | Indica si la fuente está en negrita. |
| fontItalic | Boolean | Indica si la fuente está en cursiva. |
| upperLeftRow | Int32 | Índice de la fila superior izquierda. |
| top | Int32 | Representa el desplazamiento vertical de la forma desde su fila izquierda, en unidades de píxel. |
| upperLeftColumn | Int32 | Índice de la columna superior izquierda. |
| left | Int32 | Representa el desplazamiento horizontal de la forma desde su columna izquierda, en unidades de píxel. |
| height | Int32 | Representa la altura de la forma, en unidades de píxel. |
| width | Int32 | Representa el ancho de la forma, en unidades de píxel. |

### Valor_devuelto

Devuelve un objeto Shape que representa el nuevo objeto de WordArt.

### Ejemplos

