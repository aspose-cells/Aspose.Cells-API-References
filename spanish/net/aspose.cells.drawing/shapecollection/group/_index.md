## ShapeCollection.Group method

Agrupa las formas.

```csharp
public GroupShape Group(Shape[] groupItems)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| groupItems | Shape[] | los elementos del grupo. |

### Valor_devuelto

Devuelve la forma del grupo.

### Observaciones

La forma en groupItems no debe estar agrupada. La forma debe estar en esta colección de Formas.

### Ejemplos

```csharp

[C#]
//añadir primera forma
shapes.AddRectangle(2, 0, 2, 0, 50, 50);
//añadir segunda forma
shapes.AddRectangle(6, 0, 2, 0, 30, 30);

Shape[] shapesArr = new Shape[] { shapes[0], shapes[1] };
GroupShape groupShape = shapes.Group(shapesArr);

```
