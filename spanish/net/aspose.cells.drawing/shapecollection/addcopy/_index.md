## ShapeCollection.AddCopy method

Agrega y copia una forma a la hoja de trabajo.

```csharp
public Shape AddCopy(Shape sourceShape, int upperLeftRow, int top, int upperLeftColumn, int left)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| sourceShape | Shape | Forma de fuente. |
| upperLeftRow | Int32 | Índice de la fila superior izquierda. |
| top | Int32 | Representa el desplazamiento vertical de la casilla de verificación desde su fila izquierda, en unidades de píxel. |
| upperLeftColumn | Int32 | Índice de la columna superior izquierda. |
| left | Int32 | Representa el desplazamiento horizontal del cuadro de texto desde su columna izquierda, en unidades de píxel. |

### Valor_devuelto

El nuevo índice de objeto de forma.

### Ejemplos

```csharp

[C#]
//añadir una forma
RectangleShape rectangle = shapes.AddRectangle(2, 0, 2, 0, 130, 130);
//Copiar
shapes.AddCopy(rectangle, 7, 0, 7, 0);
```
