## ShapeCollection.CopyInRange method

Copie formas en el rango al rango de destino.

```csharp
public void CopyInRange(ShapeCollection sourceShapes, CellArea ca, int destRow, int destColumn, 
    bool isContained)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| sourceShapes | ShapeCollection | Formas de origen. |
| ca | CellArea | El rango de la fuente. |
| destRow | Int32 | El índice de la fila de destino del rango de destino. |
| destColumn | Int32 | La columna dest del rango dest. |
| isContained | Boolean | Si solo copia las formas que están contenidas en el rango. Si es verdadero, solo copia las formas en el rango. De lo contrario, funciona como MS Office. |

### Ejemplos

```csharp

[C#]
//añadir una forma
shapes.AddRectangle(2, 0, 2, 0, 130, 130);
CellArea area2 = new CellArea();
area2.StartColumn = 1;
area2.StartRow = 1;
area2.EndColumn = 5;
area2.EndRow = 11;

//Copiar
shapes.CopyInRange(shapes, area2, 12, 1, false);

```
