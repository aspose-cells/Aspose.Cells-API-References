## SetTwoColorGradient(Color, Color, GradientStyleType, int) {#settwocolorgradient_1}

Establece el relleno especificado en un degradado de dos colores. Solo se aplica a Excel 2007.

```csharp
public void SetTwoColorGradient(Color color1, Color color2, GradientStyleType style, int variant)
```

| Parámetro | Escribe | Descripción |
| --- | --- | --- |
| color1 | Color | Un color degradado. |
| color2 | Color | Dos degradados de color. |
| style | GradientStyleType | Estilo de sombreado degradado. |
| variant | Int32 | La variante degradada. Puede ser un valor de 1 a 4, correspondiente a una de las cuatro variantes en la pestaña Degradado en el cuadro de diálogo Efectos de relleno. Si el estilo es GradientStyle.FromCenter, el argumento Variant solo puede ser 1 o 2. |
