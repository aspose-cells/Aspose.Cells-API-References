## PivotPageFields.AddIdentify method

Définit l'étiquette d'élément dans chaque champ de page à utiliser pour identifier la plage de données. Le pageItemIndex.Length doit être égal à PageFieldCount, veuillez donc ajouter le champ de page en premier.

```csharp
public void AddIdentify(int rangeIndex, int[] pageItemIndex)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| rangeIndex | Int32 | Index de plage de données de consolidation. |
| pageItemIndex | Int32[] | L'index d'élément de page dans le champ de chaque page. pageItemIndex[2] = 1 signifie que le deuxième élément du troisième champ doit être utilisé pour identifier cette plage. pageItemIndex[1] = -1 signifie qu'aucun élément du deuxième champ n'est utilisé pour identifiez cette plage et MS créera automatiquement un élément "vide" dans le deuxième champ pour identifier cette plage. |
