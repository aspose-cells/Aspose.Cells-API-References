## Config.IgnoreEmptyContent property

Définit s'il faut afficher la plage maximale qui inclut les données, le style, les cellules fusionnées et les formes. la valeur par défaut est true .si la dernière ligne ou colonne contient des cellules sans valeur ni formule mais avec un style personnalisé alors nous ne le montrerons pas ligne/colonne lorsque cette valeur est true
