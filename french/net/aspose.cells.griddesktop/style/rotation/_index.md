## Style.Rotation property

Représente l'angle de rotation du texte.

```csharp
public int Rotation { get; set; }
```

### Remarques

0 : Pas de rotation.

255 : De haut en bas.

-90 : Vers le bas.

90 : vers le haut.

Vous pouvez définir 255 ou une valeur comprise entre -90 et 90.
