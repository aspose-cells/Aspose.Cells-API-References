## Style.Indent property

Représente le niveau m_IndentLevel pour la cellule ou la plage. Ne peut être qu'un entier de 0 à 15.

```csharp
public int Indent { get; set; }
```

### Remarques

Si le type d'alignement horizontal du texte est défini sur une valeur autre que gauche ou droite, le niveau m_IndentLevel sera remis à zéro.
