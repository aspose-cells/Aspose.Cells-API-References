### Remarques

La valeur par défaut est une chaîne vide. Si SourceFullName n'est pas une chaîne vide, l'image est liée. Si SourceFullName n'est pas une chaîne vide, mais Data est null, alors l'image est liée et non stockée dans le fichier.
