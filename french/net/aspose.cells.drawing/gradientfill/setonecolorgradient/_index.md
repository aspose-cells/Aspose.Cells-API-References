## GradientFill.SetOneColorGradient method

Définit le remplissage spécifié sur un dégradé d'une seule couleur. S'applique uniquement à Excel 2007.

```csharp
public void SetOneColorGradient(Color color, double degree, GradientStyleType style, int variant)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| color | Color | Une couleur dégradée. |
| degree | Double | Le degré de gradient. Peut être une valeur comprise entre 0,0 (sombre) et 1,0 (clair). |
| style | GradientStyleType | Style d'ombrage dégradé. |
| variant | Int32 | La variante dégradée. Il peut s'agir d'une valeur comprise entre 1 et 4, correspondant à l'une des quatre variantes de l'onglet Dégradé de la boîte de dialogue Effets de remplissage. Si style est GradientStyle.FromCenter, l'argument Variant ne peut être que 1 ou 2. |
