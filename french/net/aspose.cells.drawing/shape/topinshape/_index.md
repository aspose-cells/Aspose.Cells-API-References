## Shape.TopInShape property

Représente le décalage vertical de la forme par rapport à la bordure supérieure de la forme parent, en unité de 1/4000 de hauteur de la forme parent.

```csharp
public int TopInShape { get; set; }
```

### Remarques

S'applique uniquement lorsque cette forme se trouve dans le groupe ou le graphique.

### Exemples

```csharp

[C#]
if (shape.TopInShape == 8000)
    shape.TopInShape = 4000;
```
