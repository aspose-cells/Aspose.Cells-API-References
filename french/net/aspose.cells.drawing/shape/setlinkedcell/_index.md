## Shape.SetLinkedCell method

Définit la plage liée à la valeur du contrôle.

```csharp
public void SetLinkedCell(string formula, bool isR1C1, bool isLocal)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| formula | String | La plage liée à la valeur du contrôle. |
| isR1C1 | Boolean | Indique si la formule doit être formatée en tant que R1C1. |
| isLocal | Boolean | Indique si la formule doit être formatée par les paramètres régionaux. |

