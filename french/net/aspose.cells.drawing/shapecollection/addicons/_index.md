## ShapeCollection.AddIcons method

Ajoute une image svg.

```csharp
public Picture AddIcons(int upperLeftRow, int top, int upperLeftColumn, int left, int height, 
    int width, byte[] imageByteData, byte[] compatibleImageData)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| upperLeftRow | Int32 | Index de ligne en haut à gauche. |
| top | Int32 | Représente le décalage vertical de la forme par rapport à sa ligne de gauche, en unité de pixel. |
| upperLeftColumn | Int32 | Index de la colonne en haut à gauche. |
| left | Int32 | Décalage horizontal de la forme par rapport à sa colonne de gauche, en pixels. |
| height | Int32 | La hauteur de la forme, en unité de pixel. |
| width | Int32 | La largeur de la forme, en unité de pixel. |
| imageByteData | Byte[] | Les données d'octet d'image. |
| compatibleImageData | Byte[] | Conversion des données d'image à partir de svg afin d'être compatible avec Excel 2016 ou versions antérieures. |
