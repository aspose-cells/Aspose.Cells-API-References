## ShapeCollection.AddAutoShape method

Ajoute une forme automatique à la feuille de calcul.

```csharp
public Shape AddAutoShape(AutoShapeType type, int upperLeftRow, int top, int upperLeftColumn, 
    int left, int height, int width)
```

| Paramètre | Taper | La description |

| type | AutoShapeType | Type de forme automatique. |
| upperLeftRow | Int32 | Index de ligne en haut à gauche. |
| top | Int32 | Représente le décalage vertical de Shape par rapport à sa ligne de gauche, en pixels. |

| left | Int32 | Représente le décalage horizontal de Shape par rapport à sa colonne de gauche, en pixels. |
