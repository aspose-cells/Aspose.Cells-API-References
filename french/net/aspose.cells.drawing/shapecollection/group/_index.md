## ShapeCollection.Group method

Regroupez les formes.

```csharp
public GroupShape Group(Shape[] groupItems)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| groupItems | Shape[] | les éléments du groupe. |

### Return_Value

Renvoie la forme du groupe.

### Remarques

La forme dans les groupItems ne doit pas être groupée. La forme doit être dans cette collection Shapes.

### Exemples

```csharp

[C#]
// ajoute la première forme
shapes.AddRectangle(2, 0, 2, 0, 50, 50);
// ajouter une deuxième forme
shapes.AddRectangle(6, 0, 2, 0, 30, 30);

Shape[] shapesArr = new Shape[] { shapes[0], shapes[1] };
GroupShape groupShape = shapes.Group(shapesArr);

```
