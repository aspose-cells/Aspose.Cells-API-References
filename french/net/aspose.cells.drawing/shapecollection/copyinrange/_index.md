## ShapeCollection.CopyInRange method

Copiez les formes de la plage vers la plage de destination.

```csharp
public void CopyInRange(ShapeCollection sourceShapes, CellArea ca, int destRow, int destColumn, 
    bool isContained)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| sourceShapes | ShapeCollection | Formes sources. |
| ca | CellArea | La plage source. |
| destRow | Int32 | Index de ligne dest de la plage dest. |
| destColumn | Int32 | La colonne dest de la plage dest. |
| isContained | Boolean | Copie uniquement les formes contenues dans la plage. Si vrai, copie uniquement les formes de la plage. Sinon, cela fonctionne comme MS Office. |

### Exemples

```csharp

[C#]
// ajouter une forme
shapes.AddRectangle(2, 0, 2, 0, 130, 130);
CellArea area2 = new CellArea();
area2.StartColumn = 1;
area2.StartRow = 1;
area2.EndColumn = 5;
area2.EndRow = 11;

//copie
shapes.CopyInRange(shapes, area2, 12, 1, false);

```
