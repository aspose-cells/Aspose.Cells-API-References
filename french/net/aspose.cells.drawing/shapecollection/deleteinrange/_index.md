## ShapeCollection.DeleteInRange method

Supprimer les formes de la plage. Les formes de commentaire ne seront pas supprimées.

```csharp
public void DeleteInRange(CellArea ca)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| ca | CellArea | La plage. Si les formes sont contenues dans la plage, elles seront supprimées. |

### Exemples

```csharp

[C#]
// ajoute la première forme
shapes.AddRectangle(2, 0, 2, 0, 50, 50);
// ajouter une deuxième forme
shapes.AddRectangle(6, 0, 2, 0, 30, 30);

CellArea area3 = new CellArea();
area3.StartColumn = 0;
area3.StartRow = 5;
area3.EndColumn = 5;
area3.EndRow = 8;

//del
shapes.DeleteInRange(area3);
```
