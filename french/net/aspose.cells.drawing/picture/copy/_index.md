## Picture.Copy method

Copiez l'image.

```csharp
public void Copy(Picture source, CopyOptions options)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| source | Picture | L'image source. |
| options | CopyOptions | Les options de copie. |

### Exemples

```csharp

[C#]
//Instanciation d'un objet Workbook
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
//insérer la première image
int imgIndex1 = worksheet.Pictures.Add(1, 1, "1.png");
// Récupère l'objet image inséré
Picture pic1 = worksheet.Pictures[imgIndex1];
//insérer une deuxième image
int imgIndex2 = worksheet.Pictures.Add(1, 9, "2.jpeg");
// Récupère l'objet image inséré
Picture pic2 = worksheet.Pictures[imgIndex2];
// Copiez l'image 1 sur l'image 2. Vous obtiendrez deux objets de l'image 1 superposés.
CopyOptions opt = new CopyOptions();
pic2.Copy(pic1, opt);
// Enregistrez le fichier excel.
workbook.Save("result.xlsx");
```
