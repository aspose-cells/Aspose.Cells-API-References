## Picture.SignatureLine property

Obtient et définit la ligne de signature

```csharp
public SignatureLine SignatureLine { get; set; }
```

### Exemples

```csharp

[C#]
//Instanciation d'un objet Workbook
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
// Ajout d'une image à l'emplacement d'une cellule dont les indices de ligne et de colonne sont 1 dans la feuille de calcul. C'est la cellule "B2"
int imgIndex = worksheet.Pictures.Add(1, 1, "example.jpeg");
// Récupère l'objet image inséré
Picture pic = worksheet.Pictures[imgIndex];
// Créer un objet de ligne de signature
SignatureLine s = new SignatureLine();
s.Signer = "Simon Zhao";
s.Title = "Development Lead";
s.Email = "Simon.Zhao@aspose.com";
// Affecte l'objet ligne de signature à Picture.
pic.SignatureLine = s;
// Enregistrez le fichier excel.
workbook.Save("result.xlsx");
```
