## ToggleButtonActiveXControl.IsTripleState property

Indique comment le contrôle spécifié affichera les valeurs nulles.

```csharp
public bool IsTripleState { get; set; }
```

### Remarques

**Paramètre**

**La description**

Vrai

Le contrôle passera par les états pour les valeurs Oui, Non et Null. Le contrôle apparaît estompé (grisé) lorsque sa propriété Value est définie sur Null.

Faux

(Par défaut) La commande passera par les états pour les valeurs Oui et Non. Les valeurs nulles s'affichent comme si elles n'étaient pas des valeurs.
