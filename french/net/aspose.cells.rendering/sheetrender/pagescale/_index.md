## SheetRender.PageScale property

Obtient l'échelle de page calculée de la feuille. Renvoie l'échelle définie si[`Zoom`](../../../aspose.cells/pagesetup/zoom) est défini. Sinon, renvoie l'échelle calculée selon[`FitToPagesWide`](../../../aspose.cells/pagesetup/fittopageswide) et[`FitToPagesTall`](../../../aspose.cells/pagesetup/fittopagestall) .

```csharp
public double PageScale { get; }
```

### Exemples

```csharp
Workbook wb = new Workbook("Book1.xlsx");

SheetRender sheetRender = new SheetRender(wb.Worksheets[0], new ImageOrPrintOptions());

// Obtient l'échelle de page calculée de la feuille.
double pageScale = sheetRender.PageScale;
```
