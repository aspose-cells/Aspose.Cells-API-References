## WebQueryConnection.EditWebPage property

L'URL de la page Web destinée à l'utilisateur affichant les données de la requête Web. Cette URL est persistante dans le cas où sourceData="true" et l'URL ont été redirigées pour référencer un fichier XML. Ensuite, la page destinée à l'utilisateur peut être affichée dans l'interface utilisateur et les données XML peuvent être récupérées dans les coulisses.
