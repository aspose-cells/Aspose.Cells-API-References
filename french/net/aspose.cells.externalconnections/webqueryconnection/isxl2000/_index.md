## WebQueryConnection.IsXl2000 property

Cet indicateur existe pour la rétrocompatibilité avec les anciens fichiers de feuille de calcul existants et est défini sur vrai si cette requête Web a été actualisée dans une application de feuille de calcul plus récente ou égale à pour Microsoft Excel 2000. Il s'agit d'un attribut facultatif qui peut être ignoré.

public bool IsXl2000 { get; set; }
