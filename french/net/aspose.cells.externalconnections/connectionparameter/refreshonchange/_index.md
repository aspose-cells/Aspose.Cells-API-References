## ConnectionParameter.RefreshOnChange property

Indicateur indiquant si la requête doit s'actualiser automatiquement lorsque le contenu d'une cellule qui fournit la valeur du paramètre change. Si la valeur est true, les données externes sont actualisées à l'aide de la nouvelle valeur de paramètre à chaque modification. Si false, les données externes ne sont actualisées qu'à la demande de l'utilisateur, ou d'autres événements déclenchent l'actualisation (par exemple, classeur ouvert).

public bool RefreshOnChange { get; set; }
