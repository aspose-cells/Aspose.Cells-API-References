## GridCell.GetFontColor method

Obtient la couleur de la police de la cellule. Lorsque vous modifiez la couleur, vous devez appeler la méthode "SetFontColor", pour définir la couleur de la police sur la cellule.

```csharp
public Color GetFontColor()
```

### Return_Value

renvoie la couleur de la police de la cellule.

### Exemples

```csharp
[C#]
GridCell cell = gridDesktop1.GetActiveWorksheet().Cells[0, 0];
Font font = new Font("Courier New", 8, FontStyle.Italic);
cell.SetFont(font);
Color color = cell.GetFontColor();
color = Color.Black;
cell.SetFontColor(color);

[Visual Basic]

cell.SetFont(font)

```
