## GridCell.Protected property

Indique si la cellule est protégée. Si la valeur est "true", l'utilisateur ne peut pas modifier la cellule via l'interface utilisateur. Cet attribut n'a rien à voir avec la propriété Style.CellLocked et ne sera pas enregistré dans le fichier lorsque données de grille exportées. La valeur par défaut est "false".

public bool Protected { get; set; }
