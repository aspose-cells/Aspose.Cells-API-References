## GridCell.Formula property

Obtient ou définit une formule duCell .

```csharp
public string Formula { get; set; }
```

### Remarques

Une chaîne de formule commence toujours par un signe égal (=). Et veuillez toujours utiliser la virgule (,) comme séparateur de paramètres, comme "=SOMME(A1, E1, H2)".

L'utilisateur peut définir n'importe quelle formule dans le fichier du concepteur de classeur. Aspose.Cells conservera toutes les formules. Si l'utilisateur utilise cette propriété pour définir une formule dans une cellule, la majeure partie des fonctions intégrées du classeur est prise en charge. Et plus est à venir. Si vous avez un besoin particulier pour les fonctions intégrées de Workbook, veuillez nous le faire savoir.

### Exemples

```csharp
[C#]
	cell.Formula = "=SUM(A1:C3) + E6*2";
[Visual Basic]
	cell.Formula = "=SUM(A1:C3) + E6*2"
```
