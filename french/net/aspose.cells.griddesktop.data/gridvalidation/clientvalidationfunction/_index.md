### Remarques

Utilisez la propriété ClientValidationFunction pour spécifier le nom de la fonction de validation client. La fonction doit être déclarée comme suit : fonction customValicationFunction(source, valeur) Le paramètre "source" est l'objet cellule. Le paramètre "valeur" est la valeur de chaîne d'une cellule à vérifier. La fonction doit renvoyer vrai si la valeur est valide.
