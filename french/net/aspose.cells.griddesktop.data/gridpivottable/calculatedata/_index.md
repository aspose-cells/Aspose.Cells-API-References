## GridPivotTable.CalculateData method

Calcule les données du tableau croisé dynamique en cellules.

```csharp
public void CalculateData()
```

### Remarques

Cell.Value dans la plage de pivot n'a pas pu renvoyer le résultat correct si la méthode n'a pas été appelée. Cette méthode calcule les données avec un cache de pivot interne, pas la source de données d'origine. Donc, si la source de données est modifiée, veuillez appeler RefreshData () méthode d'abord.
