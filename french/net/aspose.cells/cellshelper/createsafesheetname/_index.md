## CreateSafeSheetName(string) {#createsafesheetname}

Vérifie le nom de feuille donné et en crée un valide si nécessaire. Si le nom de feuille donné est conforme aux règles du nom de feuille Excel, puis le renvoie. Sinon, la chaîne sera tronquée si la longueur dépasse la limite et les caractères invalides seront remplacés par ' ', puis renvoie la valeur de chaîne reconstruite.

```csharp
public static string CreateSafeSheetName(string nameProposal)
```

| nameProposal | String | nom de feuille à utiliser |
