## Workbook.UpdateLinkedDataSource method

Si ce classeur contient des liens externes vers d'autres sources de données, Aspose.Cells tentera de récupérer les dernières données.

```csharp
public void UpdateLinkedDataSource(Workbook[] externalWorkbooks)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| externalWorkbooks | Workbook[] | Les classeurs externes sont référencés par ce classeur. S'il est nul, nous ouvrirons directement les fichiers liés externes.. Si ce n'est pas nul, nous vérifierons d'abord si le lien externe dans le tableau ; sinon, nous ouvrirons les fichiers externes liés à nouveau. |

### Remarques

Si la méthode n'est pas appelée avant de calculer les formules, Aspose.Cells utilisera les informations précédentes (mises en cache dans le fichier) ; Veuillez définir CellsHelper.StartupPath,CellsHelper.AltStartPath,CellsHelper.LibraryPath. Et veuillez définir Workbook.FilePath si ce classeur provient d'un flux, sinon Aspose.Cells ne pouvait parfois pas obtenir le chemin complet du lien externe.
