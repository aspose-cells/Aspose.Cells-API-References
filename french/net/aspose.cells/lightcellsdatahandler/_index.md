### Remarques

Lors de la lecture d'un classeur par ce mode,[`StartSheet`](./startsheet) sera vérifié lors de la lecture de chaque feuille de calcul dans le classeur. Pour une feuille, si[`StartSheet`](./startsheet) donne vrai, alors toutes les données et propriétés des lignes/cellules de cette feuille seront vérifiées et traitées par l'implémentation de cette interface. Pour chaque ligne,[`StartRow`](./startrow) sera appelée pour vérifier si elle doit être traitée. Si une ligne doit être traitée, les propriétés de cette ligne seront d'abord lues et l'utilisateur pourra accéder à ses propriétés en[`ProcessRow`](./processrow). si les cellules de la ligne doivent également être traitées, alors[`ProcessRow`](./processrow) devrait retourner true puis[`StartCell`](./startcell) sera appelé be pour chaque cellule existante dans cette ligne pour vérifier si une cellule doit être traitée. Si une cellule doit être traitée, alors[`ProcessCell`](./processcell) sera appelée pour traiter la cellule par l'implémentation de cette interface.
