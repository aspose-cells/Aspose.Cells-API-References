## LightCellsDataHandler.ProcessCell method

Commence à traiter une cellule.

```csharp
public bool ProcessCell(Cell cell)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| cell | Cell | Objet de cellule en cours de traitement actuellement |

### Return_Value

si cette cellule doit être conservée dans le modèle de cellules de la feuille actuelle. Généralement, il doit être faux afin que toutes les cellules ne soient pas conservées en mémoire après avoir été traitées, puis la mémoire soit sauvegardée. Pour un usage particulier tel que l'utilisateur doit accéder certaines cellules plus tard après le traitement de l'ensemble du classeur, l'utilisateur peut faire en sorte que cette méthode retourne true pour conserver ces cellules spéciales dans le modèle Cells et y accéder ultérieurement par des API telles que Cells[row, column]. Cependant, conserver les données des cellules dans Cells le modèle nécessitera plus de mémoire et si toutes les cellules sont conservées, la lecture du modèle file en mode LightCells deviendra identique à la lecture normale.

### Remarques

Il sera appelé après la lecture des données d'une cellule.
