## Style.IndentLevel property

Représente le niveau de retrait de la cellule ou de la plage. Ne peut être qu'un entier de 0 à 250.

```csharp
public int IndentLevel { get; set; }
```

### Remarques

Si le type d'alignement horizontal du texte est défini sur une valeur autre que gauche ou droite, le niveau d'indentation sera remis à zéro.
