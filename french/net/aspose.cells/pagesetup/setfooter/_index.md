## PageSetup.SetFooter method

Définit un script mettant en forme le pied de page d'un fichier Excel.

```csharp
public void SetFooter(int section, string footerScript)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| section | Int32 | 0 : Section gauche, 1 : Section centrale, 2 : Section droite. |
| footerScript | String | Script de format de pied de page. |

### Remarques

Commandes de script :

**Commande**

**La description**

&amp;P

Numéro de la page actuelle

&amp;N

Nombre de pages

&amp;RÉ

Date actuelle

&amp;T

Heure actuelle

&amp;UN

Nom de la feuille

&amp;F

Nom de fichier sans chemin

&amp;"&lt;Nom de la police&gt;"

Nom de la police, par exemple : &amp;"Arial"

&amp;"&lt;Nom de police&gt;, &lt;Style de police&gt;"

Nom et style de police, par exemple : &amp;"Arial,Gras"

&amp;&lt;Taille de la police&gt;

Taille de police. Si cette commande est suivie d'un nombre en clair à imprimer dans l'en-tête, il sera séparé de la hauteur de la police par un espace.

&amp;K&lt;RRGGBB&gt;

Couleur de la police, par exemple (RED) : &amp;KFF0000

&amp;G

Script d'image

Par exemple : "&amp;Arial,Bold&amp;8Footer Note"
