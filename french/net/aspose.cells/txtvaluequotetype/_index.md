## TxtValueQuoteType enumeration

Spécifie le type d'utilisation des guillemets pour les valeurs dans les fichiers au format texte.

```csharp
public enum TxtValueQuoteType
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Normal | `0` | Toutes les valeurs contenant des caractères spéciaux tels que guillemets, caractères de séparation seront citées. Idem avec le comportement de ms excel pour l'exportation de fichier texte. |
| Always | `1` | Toutes les valeurs seront toujours citées. |
| Minimum | `2` | Citez uniquement les valeurs lorsque cela est nécessaire. Par exemple, si une valeur contient des guillemets mais que le guillemet n'est pas au début de cette valeur, cette valeur ne sera pas citée. |
| Never | `3` | Toutes les valeurs ne seront pas citées. Le fichier texte exporté avec ce type peut ne pas être relu correctement car les guillemets nécessaires sont absents. |
