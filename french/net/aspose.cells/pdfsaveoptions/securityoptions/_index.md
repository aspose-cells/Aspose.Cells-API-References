## PdfSaveOptions.SecurityOptions property

Définissez ces options, lorsque la sécurité est nécessaire dans le résultat xls2pdf.

```csharp
public PdfSecurityOptions SecurityOptions { get; set; }
```

### Exemples

Le code suivant définit l'autorisation d'impression haute résolution pour le pdf de sortie.

```csharp
Workbook wb = new Workbook();
wb.Worksheets[0].Cells["A1"].Value = "Aspose";

PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();


PdfSecurityOptions pdfSecurityOptions = new PdfSecurityOptions();

//définir le mot de passe du propriétaire
pdfSecurityOptions.OwnerPassword = "YourOwnerPassword";

//envoie ton mot de passe utilisé
pdfSecurityOptions.UserPassword = "YourUserPassword";

//définir l'autorisation d'impression
pdfSecurityOptions.PrintPermission = true;

// définir une haute résolution pour l'impression
pdfSecurityOptions.FullQualityPrintPermission = true;


pdfSaveOptions.SecurityOptions = pdfSecurityOptions;

wb.Save("output.pdf", pdfSaveOptions);
```
