## PdfSaveOptions.CheckFontCompatibility property

Indique s'il faut vérifier la compatibilité des polices pour chaque caractère du texte.

```csharp
public bool CheckFontCompatibility { get; set; }
```

### Remarques

La valeur par défaut est true. Désactiver cette propriété peut donner de meilleures performances. Mais lorsque la police par défaut ou spécifiée du texte/caractère ne peut pas être utilisée pour le rendre, des caractères illisibles (tels que des blocs) peuvent apparaître dans le pdf généré. Dans une telle situation, l'utilisateur doit conserver cette propriété sur true afin que une police alternative puisse être recherchée et utilisée pour afficher le texte à la place ;
