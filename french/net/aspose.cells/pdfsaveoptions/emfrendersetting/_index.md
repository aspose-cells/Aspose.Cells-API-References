## PdfSaveOptions.EmfRenderSetting property

Paramètre pour le rendu du métafichier Emf.

```csharp
public EmfRenderSetting EmfRenderSetting { get; set; }
```

### Remarques

Les métafichiers EMF identifiés comme "EMF+ Dual" peuvent contenir à la fois des enregistrements EMF+ et des enregistrements EMF. L'un ou l'autre type d'enregistrement peut être utilisé pour rendre l'image, uniquement les enregistrements EMF+ ou uniquement les enregistrements EMF. LorsqueEmfPlusPrefer est défini, les enregistrements EMF+ seront analysés lors du rendu en pdf, sinon seuls les enregistrements EMF seront analysés. La valeur par défaut estEmfOnly .
