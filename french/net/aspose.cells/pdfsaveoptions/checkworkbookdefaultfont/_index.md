## PdfSaveOptions.CheckWorkbookDefaultFont property

Lorsque les caractères dans Excel sont Unicode et ne sont pas définis avec la police correcte dans le style de cellule, Ils peuvent apparaître sous forme de bloc dans pdf,image. Définissez ceci sur true pour essayer d'utiliser la police par défaut du classeur pour afficher ces caractères en premier.

```csharp
public bool CheckWorkbookDefaultFont { get; set; }
```

### Remarques

La valeur par défaut est true.
