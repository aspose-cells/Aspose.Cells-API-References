## Worksheet.Name property

Obtient ou définit le nom de la feuille de calcul.

```csharp
public string Name { get; set; }
```

### Remarques

La longueur maximale du nom de la feuille est de 31. Et vous ne pouvez pas attribuer le même nom (insensible à la casse) à deux feuilles de calcul. Par exemple, vous ne pouvez pas définir "SheetName1" sur la première feuille de calcul et définir "SHEETNAME1" sur la deuxième feuille de calcul.
