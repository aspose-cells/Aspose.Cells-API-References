## AbstractCalculationMonitor.OnCircular method

Implémentez cette méthode pour faire des affaires lors du calcul de formules avec des références circulaires.

```csharp
public virtual bool OnCircular(IEnumerator circularCellsData)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| circularCellsData | IEnumerator | IEnumerator avec des éléments CalculationCell représentant des cellules qui dépendent de références circulaires. |

### Return_Value

Indique si le moteur de formule doit calculer ces cellules en circulaire après cet appel. True pour laisser le moteur de formule continuer à faire le calcul pour elles. False pour laisser le moteur de formule simplement marquer ces cellules comme calculées.

### Remarques

Dans l'implémentation, l'utilisateur peut également définir la valeur attendue comme résultat calculé pour une partie/toutes ces cellules afin que le moteur de formule ne les calcule pas de manière récursive.
