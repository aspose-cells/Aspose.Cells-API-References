## DirectoryType enumeration

Représente le type de répertoire du nom de fichier.

```csharp
public enum DirectoryType
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Volume | `0` | Représente une lettre de lecteur MS-DOS. Il est suivi de la lettre du lecteur. Ou des noms de fichiers UNC, tels que \\server\share\myfile.xls |
| SameVolume | `1` | Indique que le classeur source se trouve sur le même lecteur que le classeur dépendant (la lettre de lecteur est omise) |
| DownDirectory | `2` | Indique que le classeur source se trouve dans un sous-répertoire du répertoire actuel. |
| UpDirectory | `3` | Indique que le classeur source se trouve dans le répertoire parent du répertoire actuel. |
