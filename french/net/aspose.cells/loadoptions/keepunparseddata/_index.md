## LoadOptions.KeepUnparsedData property

Indique si les données non analysées sont conservées en mémoire pour le classeur lorsqu'il est chargé à partir du fichier de modèle. La valeur par défaut est true.

```csharp
public bool KeepUnparsedData { get; set; }
```

### Remarques

Pour les scénarios où l'utilisateur n'a besoin que de lire une partie du contenu du fichier de modèle et n'a pas besoin de sauvegarder le classeur, définir cette propriété sur false peut améliorer les performances, en particulier lors de son utilisation avec une sorte de LoadFilter,
