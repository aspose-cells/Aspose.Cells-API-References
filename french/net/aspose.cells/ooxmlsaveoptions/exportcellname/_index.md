## OoxmlSaveOptions.ExportCellName property

Indique si le nom de la cellule est exporté vers un fichier Excel2007 .xlsx (.xlsm, .xltx, .xltm). Si le fichier de sortie est accessible par SQL Server DTS, cette valeur doit être true. La définition de la valeur sur false augmentera considérablement les performances et réduira la taille du fichier lors de la création d'un fichier volumineux. La valeur par défaut est true.
