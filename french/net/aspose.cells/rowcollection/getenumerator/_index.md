### Exemples

```csharp
[C#]
Workbook workbook = new Workbook("template.xlsx");
Cells cells = workbook.Worksheets[0].Cells;

IEnumerator en = cells.Rows.GetEnumerator();
while (en.MoveNext())
{
    Row row = (Row)en.Current;
    Console.WriteLine(row.Index + ": " + row.Height);
}
```
