## WorkbookSettings.IsHScrollBarVisible property

Obtient ou définit une valeur indiquant si la feuille de calcul générée contiendra une barre de défilement horizontale.

```csharp
public bool IsHScrollBarVisible { get; set; }
```

### Remarques

La valeur par défaut est true.

### Exemples

Le code suivant rend la barre de défilement horizontale invisible pour la feuille de calcul.

```csharp
[C#]
// Masque la barre de défilement horizontale du fichier Excel.
settings.IsHScrollBarVisible = false;

[Visual Basic]
' Masquez la barre de défilement horizontale du fichier Excel.
```
