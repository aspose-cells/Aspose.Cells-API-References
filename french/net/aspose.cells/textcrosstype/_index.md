## TextCrossType enumeration

Énumère l'affichage du type de texte lorsque la largeur du texte est supérieure à la largeur de la cellule.

```csharp
public enum TextCrossType
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Default | `1` | Afficher le texte comme dans Microsoft Excel. |
| CrossKeep | `2` | Afficher tout le texte en croisant les autres cellules et conserver le texte des cellules barrées. |
| CrossOverride | `3` | Afficher tout le texte en croisant d'autres cellules et remplacer le texte des cellules barrées. |
| StrictInCell | `4` | Afficher uniquement le texte dans la largeur de la cellule. |
