
| Paramètre | Taper | La description |
| --- | --- | --- |
| addInFile | String | le fichier contient les fonctions addin |
| functionName | String | le nom de la fonction complémentaire |
| lib | Boolean | si le fichier addin donné se trouve dans le répertoire ou le sous-répertoire de la bibliothèque de compléments de classeur.
