## LightCellsDataProvider.StartRow method

Commence à enregistrer les données d'une ligne.

```csharp
public void StartRow(Row row)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| row | Row | Objet de ligne à implémenter pour remplir les données. Son index de ligne est la valeur renvoyée du dernier appel de[`NextRow`](../nextrow) . Si la ligne a été initialisée dans le modèle de cellules internes, l'objet de ligne existant sera utilisé. Sinon, un objet de ligne temporaire sera utilisé pour l'implémentation afin de remplir les données. |

### Remarques

Il sera appelé au début de l'enregistrement d'une ligne et de ses données de cellules. Si la ligne actuelle a des propriétés personnalisées telles que la hauteur, le style, etc., l'implémentation doit définir ces propriétés sur l'objet Row donné ici.
