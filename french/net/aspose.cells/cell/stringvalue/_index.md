## Cell.StringValue property

Obtient la valeur de chaîne contenue dans la cellule. Si le type de cette cellule est une chaîne, renvoyez la valeur de chaîne elle-même. Pour les autres types de cellules, la valeur de chaîne formatée (formatée avec le style spécifié de cette cellule) sera renvoyée. La valeur de cellule formatée est identique à ce que vous peut obtenir d'Excel lors de la copie d'une cellule sous forme de texte (comme copier une cellule dans un éditeur de texte ou exporter vers csv).

public string StringValue { get; }
