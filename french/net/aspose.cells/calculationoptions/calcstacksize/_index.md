## CalculationOptions.CalcStackSize property

Spécifie la taille de la pile pour le calcul récursif des cellules.

```csharp
public int CalcStackSize { get; set; }
```

### Remarques

Lorsqu'un grand nombre de cellules doivent être calculées de manière récursive dans l'arbre de dépendance, StackOverflowException peut être provoqué dans le processus de calcul. Si c'est le cas, l'utilisateur doit spécifier une valeur plus petite pour cette propriété. Pour une telle situation, l'utilisateur doit déterminer le valeur appropriée pour cette propriété en fonction des formules et des données réelles. Une valeur trop petite peut entraîner une dégradation des performances pour le calcul de la formule.
