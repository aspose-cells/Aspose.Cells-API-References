## FormatCondition.Formula1 property

Obtient et définit la valeur ou l'expression associée à la mise en forme conditionnelle.

```csharp
public string Formula1 { get; set; }
```

### Remarques

Veuillez ajouter toutes les zones avant de définir la formule. Pour définir la formule de cette condition, si la valeur d'entrée commence par '=', elle sera prise comme formule. Sinon, elle sera prise comme valeur simple (texte, nombre, bool ). Pour une valeur de texte commençant par '=', l'utilisateur peut la saisir sous forme de formule au format : "=\"=...\"".
