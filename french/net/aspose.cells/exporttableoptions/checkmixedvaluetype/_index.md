## ExportTableOptions.CheckMixedValueType property

False, Aspose.Cells définira le type de DataColumn par le type de valeur de la première ligne pour les performances. True, Aspose.Cells vérifiera si le type de valeur dans la colonne est mélangé avant de définir le type de DataColumn Et le type de valeur sont mixte, le type de DataColumn sera string.
