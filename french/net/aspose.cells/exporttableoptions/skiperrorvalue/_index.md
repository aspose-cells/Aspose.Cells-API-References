## ExportTableOptions.SkipErrorValue property

Indique si la valeur non valide est ignorée pour la colonne. Par exemple, si le type de colonne est décimal, la valeur est supérieure à décimal.MaxValue et cette propriété est vraie, nous ne lancerons plus d'exception. La valeur par défaut est fausse.
