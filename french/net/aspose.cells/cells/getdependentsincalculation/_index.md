## Cells.GetDependentsInCalculation method

Obtient toutes les cellules dont le résultat calculé dépend d'une cellule spécifique.

```csharp
public IEnumerator GetDependentsInCalculation(int row, int column, bool recursive)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| row | Int32 | Index de ligne de la cellule spécifique |
| column | Int32 | Index de colonne de la cellule spécifique. |
| recursive | Boolean | Si renvoie les dépendants qui ne font pas directement référence à la cellule spécifique mais font référence à d'autres feuilles de cette cellule. |

### Return_Value

Énumérateur pour énumérer toutes les personnes à charge (objets Cellule)

### Remarques

Pour utiliser cette méthode, assurez-vous que le classeur a été défini avec la valeur true for [`EnableCalculationChain`](../../formulasettings/enablecalculationchain) et a été entièrement calculé avec ce paramètre. S'il n'y a pas de référence de formule à cette cellule, null sera renvoyé. Pour plus de détails et d'exemples, veuillez consulter[`GetDependentsInCalculation`](../../cell/getdependentsincalculation)
