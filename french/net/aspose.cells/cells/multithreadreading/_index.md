## Cells.MultiThreadReading property

Obtient ou définit si le modèle de données des cellules doit prendre en charge la lecture multi-thread. La valeur par défaut de cette propriété est false.

```csharp
public bool MultiThreadReading { get; set; }
```

### Remarques

S'il existe plusieurs threads pour lire simultanément des objets Row/Cell dans cette collection, cette propriété doit être définie sur true, sinon un résultat inattendu peut être produit. La prise en charge de la lecture multi-thread peut dégrader les performances d'accès aux objets Row/Cell à partir de cette collection. Veuillez noter que certaines fonctionnalités ne peuvent pas prendre en charge la lecture multi-thread, telles que le formatage des valeurs (par[`StringValue`](../../cell/stringvalue) ,[`DisplayStringValue`](../../cell/displaystringvalue) .etc.). Ainsi, même si cette propriété est définie sur true, ces API peuvent toujours donner un résultat inattendu pour la lecture multi-thread.
