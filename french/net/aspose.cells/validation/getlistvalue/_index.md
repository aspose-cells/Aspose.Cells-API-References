## Validation.GetListValue method

Obtenir la valeur de la liste de validation pour la cellule spécifiée.

```csharp
public object GetListValue(int row, int column)
```

| Paramètre | Taper | La description |
| --- | --- | --- |
| row | Int32 | L'index de ligne. |
| column | Int32 | L'indice de colonne. |

### Return_Value

La valeur pour produire la liste de cette validation pour la cellule spécifiée. Si la liste fait référence à une plage, la valeur renvoyée sera un[`ReferredArea`](../../referredarea) object; Sinon, la valeur renvoyée peut être null, object[] ou simple object.

### Remarques

Uniquement pour la validation dont le type est List et a été appliqué à une cellule donnée, sinon null sera renvoyé.
