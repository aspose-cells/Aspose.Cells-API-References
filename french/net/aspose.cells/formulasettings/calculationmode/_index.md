## FormulaSettings.CalculationMode property

Obtient ou définit le mode de calcul du classeur dans ms excel.

```csharp
public CalcModeType CalculationMode { get; set; }
```

### Remarques

Cette propriété sert uniquement à enregistrer les paramètres dans le fichier de feuille de calcul résultant afin que d'autres applications (telles que MS Excel) puissent agir en conséquence lors du chargement et de la manipulation du fichier résultant. Pour des considérations de performances pour la plupart des applications utilisateur, nous ne calculons aucune formule dans le classeur automatiquement, quel que soit le mode défini pour cette propriété. Si l'utilisateur a besoin de calculer des formules, veuillez toujours appeler des méthodes sur différents objets en fonction des besoins : [`CalculateFormula`](../../workbook/calculateformula) ,[`CalculateFormula`](../../worksheet/calculateformula) , [`Calculate`](../../cell/calculate) , ...etc.
