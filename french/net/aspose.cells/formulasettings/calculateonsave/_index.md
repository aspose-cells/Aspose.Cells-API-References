## FormulaSettings.CalculateOnSave property

Indique s'il faut recalculer le classeur avant d'enregistrer le document, en mode de calcul manuel.

```csharp
public bool CalculateOnSave { get; set; }
```

### Remarques

Cette propriété sert uniquement à enregistrer les paramètres dans le fichier de feuille de calcul résultant afin que d'autres applications (telles que MS Excel) puissent agir en conséquence lors du chargement et de la manipulation du fichier résultant. Pour des considérations de performances pour la plupart des applications des utilisateurs, nous ne calculons aucune formule dans le classeur automatiquement, quelle que soit la valeur définie pour cette propriété.
