## FormulaSettings.CalculateOnOpen property

Indique si l'application doit effectuer un calcul complet lorsque le classeur est ouvert.

```csharp
public bool CalculateOnOpen { get; set; }
```

### Remarques

Cette propriété sert uniquement à enregistrer les paramètres dans le fichier de feuille de calcul résultant afin que d'autres applications (telles que MS Excel) puissent agir en conséquence lors du chargement du fichier résultant. Pour des considérations de performances pour la plupart des applications des utilisateurs, nous ne calculons aucune formule dans le classeur automatiquement, quelle que soit la valeur définie pour cette propriété.
