## MainWeb.LoadCustomData event

Se déclenche lorsque le SessionMode du contrôle est défini sur Personnalisé et doit charger des données de feuille. Vous pouvez gérer cet événement en mode Session personnalisée pour charger des données de feuille à partir d'un fichier ou d'une base de données.

```csharp
public event WorkbookEventHandler LoadCustomData;
```
