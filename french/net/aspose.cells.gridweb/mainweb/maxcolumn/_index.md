## MainWeb.MaxColumn property

Obtient ou définit l'index de colonne d'affichage maximal (basé sur zéro) de la feuille Web. Le contrôle utilise la valeur la plus élevée de MaxColumn et la colonne maximale des données de feuille.

```csharp
public int MaxColumn { get; set; }
```

### Exemples

```csharp
[C#]
	// Crée une "fenêtre d'affichage" 4x4.
	GridWeb1.MinRow = 2;
	GridWeb1.MaxRow = 5;
	GridWeb1.MinColumn = 3;
	GridWeb1.MaxColumn = 6;

[Visual Basic]
	' Creates a 4x4 "display window".
	GridWeb1.MinRow = 2
	GridWeb1.MaxRow = 5
	GridWeb1.MinColumn = 3

```
