## MainWeb.OnGridInitClientFunction property

Obtient ou définit le nom de la fonction côté client à appeler lorsque la grille est initialisée. La fonction client doit être déclarée comme ceci : fonction MyOnGridInit(grille) { alert("La grille est initialisée : " + grid.id); } Remarque : Vous pouvez utiliser le pointeur "this" dans la fonction client pour pointer le contrôle de grille qui déclenche l'événement.
