## MainWeb.MinRow property

Obtient ou définit l'index de ligne d'affichage minimum (basé sur zéro) de la feuille Web. Le contrôle utilise la plus petite valeur de MinRow et la ligne min des données de la feuille.

```csharp
public int MinRow { get; set; }
```

### Exemples

```csharp
[C#]
	// Crée une "fenêtre d'affichage" 4x4.
	GridWeb1.MinRow = 2;
	GridWeb1.MaxRow = 5;
	GridWeb1.MinColumn = 3;
	GridWeb1.MaxColumn = 6;

[Visual Basic]
```
