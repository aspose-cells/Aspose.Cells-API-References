## MainWeb.EnableDoubleClickEvent property

Obtient ou définit s'il faut activer l'événement de double-clic côté client.

```csharp
public bool EnableDoubleClickEvent { get; set; }
```

### Remarques

Lorsqu'il est défini sur true, le contrôle recevra des événements de double-clic côté client et vous pouvez gérer ces événements côté serveur. Sinon, le contrôle ignorera tous les événements de double-clic .
