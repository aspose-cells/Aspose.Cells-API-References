## GridHtmlSaveOptions.ExportImagesAsBase64 property

Spécifie si les images sont enregistrées au format Base64 au format HTML, MHTML ou EPUB.

```csharp
public bool ExportImagesAsBase64 { get; set; }
```

### Remarques

Lorsque cette propriété est définie sur true, les données d'image sont exportées directement sur les éléments img et des fichiers séparés ne sont pas créés.
