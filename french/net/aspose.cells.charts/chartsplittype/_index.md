## ChartSplitType enumeration

Représente la manière dont les deux sections d'un camembert ou d'une barre de camembert sont divisées.

```csharp
public enum ChartSplitType
```

### Valeurs

| Nom | Évaluer | La description |
| --- | --- | --- |
| Position | `0` | Représente les points de données à diviser entre le pie et le deuxième graphique en plaçant la dernière Split Position des points de données dans le deuxième graphique |
| Value | `1` | Représente les points de données à diviser entre le camembert et le deuxième graphique en plaçant les points de données avec une valeur inférieure à la position de fractionnement dans le deuxième graphique. |
| PercentValue | `2` | Représente les points de données qui doivent être répartis entre le camembert et le deuxième graphique en plaçant les points avec un pourcentage inférieur au pourcentage de position de fractionnement dans le deuxième graphique. |

| Auto | `4` | Représente les points de données qui doivent être divisés à l'aide du mécanisme par défaut pour ce type de graphique. |
