## TickLabels.Offset property

Obtient et définit la distance des étiquettes à partir de l'axe.

```csharp
public int Offset { get; set; }
```

### Remarques

La distance par défaut est de 100 %, ce qui représente l'espacement par défaut entre les étiquettes d'axe et la ligne d'axe. La valeur peut être un pourcentage entier compris entre 0 et 1 000, par rapport à la taille de police de l'étiquette d'axe.
