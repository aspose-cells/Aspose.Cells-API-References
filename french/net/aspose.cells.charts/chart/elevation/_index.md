## Chart.Elevation property

Représente l'élévation de la vue cartographique 3D, en degrés.

```csharp
public int Elevation { get; set; }
```

### Remarques

L'élévation du graphique est la hauteur à laquelle vous visualisez le graphique, en degrés. La valeur par défaut est 15 pour la plupart des types de graphiques. La valeur de cette propriété doit être comprise entre -90 et 90, sauf pour les graphiques à barres 3D, où elle doit être comprise entre 0 et 44.
