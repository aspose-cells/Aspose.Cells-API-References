##  FormattingType dénombrement
Représente le type de formatage appliqué à un objet [Area](/cells/fr/python-net/aspose.cells.drawing/area) ou à un objet [Line](/cells/fr/python-net/aspose.cells.drawing/line).



Le type FormattingType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| AUTOMATIC | Représente le type de formatage automatique.|
| CUSTOM | Représente le type de formatage personnalisé.|
| NONE | Représente aucun type de formatage.|
