##  TxtValueQuoteType dénombrement
Spécifie le type d'utilisation de guillemets pour les valeurs dans les fichiers au format texte.



Le type TxtValueQuoteType expose les membres suivants :

###  Des champs
| Champ| Description|
| :- | :- |
| NORMAL | Toutes les valeurs qui contiennent des caractères spéciaux tels que des guillemets, des caractères de séparation seront citées.<br/> Idem avec le comportement de ms excel pour exporter des fichiers texte.|
| ALWAYS | Toutes les valeurs seront toujours citées.|
| MINIMUM | Citez uniquement les valeurs lorsque cela est nécessaire. Par exemple, si une valeur contient un guillemet mais que le guillemet ne se trouve pas au début de cette valeur, cette valeur ne sera pas citée.|
| NEVER | Toutes les valeurs ne seront pas entre guillemets. Le fichier texte exporté avec ce type peut ne pas être relu correctement car les guillemets nécessaires sont absents.|
