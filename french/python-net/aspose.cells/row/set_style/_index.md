##  set_style(style) {#Style}
Définit le style de cette ligne.



```python
def set_style(self, style):
    ...
```


| Paramètre| Taper| Description|
| :- | :- | :- |
| style | [Style](/cells/fr/python-net/aspose.cells/style) | le style à utiliser comme style par défaut pour les cellules de cette ligne.|
###  Remarques

Cette méthode définit uniquement le style donné comme style par défaut pour cette ligne,
sans modifier les paramètres de style des cellules existantes de cette ligne.
Pour mettre à jour les paramètres de style des cellules existantes vers le style spécifié en même temps,
veuillez utiliser [Row.apply_style(style, flag)](/cells/fr/python-net/aspose.cells/row/apply_style)
