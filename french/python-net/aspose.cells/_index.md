 Le**Aspose.Cells** namespace fournit des classes fondamentales et de base pour générer, convertir, modifier, rendre et imprimer des feuilles de calcul sans utiliser Microsoft Excel.
Par exemple, la classe [Workbook](/cells/fr/python-net/aspose.cells/workbook) est l'entrée principale pour créer ou charger puis manipuler un fichier tableur,
La classe [Cells](/cells/fr/python-net/aspose.cells/cells) est la classe principale pour manipuler toutes les données et propriétés d'une feuille, etc.
