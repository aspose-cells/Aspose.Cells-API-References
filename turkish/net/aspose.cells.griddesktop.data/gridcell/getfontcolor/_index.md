## GridCell.GetFontColor method

Hücre yazı tipi rengini alır. Rengi değiştirirken, yazı tipi rengini hücreye ayarlamak için "SetFontColor" yöntemini, çağırmalısınız.

```csharp
public Color GetFontColor()
```

### Geri dönüş değeri

hücre yazı tipi rengini döndürür.

### Örnekler

```csharp
[C#]
GridCell cell = gridDesktop1.GetActiveWorksheet().Cells[0, 0];
Font font = new Font("Courier New", 8, FontStyle.Italic);
cell.SetFont(font);
Color color = cell.GetFontColor();
color = Color.Black;
cell.SetFontColor(color);

[Visual Basic]
```
