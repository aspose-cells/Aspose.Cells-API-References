## GridCell.SetFont method

Yazı tipini hücreye ayarlar. Performansı artırmak için "SetFont" yöntemini uygulayın, "Font" özelliğini uygulamayın.

```csharp
public void SetFont(Font font)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| font | Font | yazı tipi ayarlanacak. |

### Örnekler

```csharp
[C#]
GridCell cell = gridDesktop1.GetActiveWorksheet().Cells[0, 0];
Font font = new Font("Courier New", 8, FontStyle.Italic);
cell.SetFont(font);
Color color = cell.GetFontColor();
color = Color.Black;
cell.SetFontColor(color);
```
