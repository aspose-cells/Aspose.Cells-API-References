## Cell.GetDependentsInCalculation method

Hesaplanan sonucu bu hücreye bağlı olan tüm hücreleri alır.

```csharp
public IEnumerator GetDependentsInCalculation(bool recursive)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| recursive | Boolean | Bu hücreye doğrudan başvuruda bulunmayan, ancak bu hücrenin diğer yapraklarına başvuruda bulunan bağımlıları döndürüp döndürmeyeceği |

### Geri dönüş değeri

Tüm bağımlıları numaralandırmak için Numaralandırıcı (Hücre nesneleri)

### Notlar

Bu yöntemi kullanmak için, lütfen çalışma kitabının için gerçek değere ayarlandığından emin olun.[`EnableCalculationChain`](../../formulasettings/enablecalculationchain) ve bu ayarla tamamen hesaplanmıştır. Bu hücreye formül başvurusu yoksa null döndürülür.

### Örnekler

```csharp
[C#]

Workbook workbook = new Workbook();
Cells cells = workbook.Worksheets[0].Cells;
cells["A1"].Formula = "=B1+SUM(B1:B10)+[Book1.xls]Sheet1!B2";
cells["A2"].Formula = "=IF(TRUE,B2,B1)";
workbook.Settings.FormulaSettings.EnableCalculationChain = true;
workbook.CalculateFormula();
IEnumerator en = cells["B1"].GetDependentsInCalculation(false);
Console.WriteLine("B1's calculation dependents:");
while(en.MoveNext())
{
    Cell c = (Cell)en.Current;
    Console.WriteLine(c.Name);
}
en = cells["B2"].GetDependentsInCalculation(false);
Console.WriteLine("B2's calculation dependents:");
while(en.MoveNext())
{
    Cell c = (Cell)en.Current;
    Console.WriteLine(c.Name);
}
```
