## OoxmlSaveOptions.ExportCellName property

Hücre adının Excel2007 .xlsx (.xlsm, .xltx, .xltm) dosyasına aktarılıp aktarılmayacağını gösterir. Çıktı dosyasına SQL Server DTS tarafından erişilebiliyorsa, bu değerin true olması gerekir. Değerin false olarak ayarlanması, performansı büyük ölçüde artırır ve büyük dosya oluştururken dosya boyutunu azaltır. Varsayılan değer true'dur.

public bool ExportCellName { get; set; }
