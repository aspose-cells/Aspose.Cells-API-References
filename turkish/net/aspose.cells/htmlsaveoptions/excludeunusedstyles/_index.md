## HtmlSaveOptions.ExcludeUnusedStyles property

Kullanılmayan stilleri hariç tutup tutmadığını gösterme. Oluşturulan html dosyaları için, kullanılmayan stilleri hariç tutmak, görsel efektleri etkilemeden dosya boyutunu küçültebilir. Bu nedenle, bu özelliğin varsayılan değeri true'dur. Kullanıcının oluşturulan html için çalışma kitabındaki tüm stilleri tutması gerekiyorsa (kullanıcı 'nin çalışma kitabını daha sonra oluşturulan html'den geri yüklemesi gerektiği senaryosu gibi), lütfen bu özelliği false olarak ayarlayın. .

public bool ExcludeUnusedStyles { get; set; }
