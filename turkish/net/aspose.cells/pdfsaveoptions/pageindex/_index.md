## PdfSaveOptions.PageIndex property

Kaydedilecek ilk sayfanın 0 tabanlı dizinini alır veya ayarlar.

```csharp
public int PageIndex { get; set; }
```

### Notlar

Varsayılan 0.

### Örnekler

Aşağıdaki örnek, bir Microsoft Excel dosyasındaki bir dizi sayfanın (3 ve 4) nasıl PDF'ye dönüştürüleceğini gösterir.

```csharp
//Bir Excel dosyası aç
Workbook wb = new Workbook("Book1.xlsx");

PdfSaveOptions options = new PdfSaveOptions();

//Çıktı PDF'sinde yalnızca Sayfa 3 ve Sayfa 4'ü yazdırın
//Başlangıç sayfası dizini (0 tabanlı dizin)
options.PageIndex = 3;
//Yazdırılacak sayfa sayısı
options.PageCount = 2;

//PDF dosyasını kaydedin
wb.Save("output.pdf", options);
```
