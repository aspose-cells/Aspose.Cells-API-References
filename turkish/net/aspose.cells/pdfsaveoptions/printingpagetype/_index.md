## PdfSaveOptions.PrintingPageType property

Hangi sayfaların yazdırılmayacağını belirtir.

```csharp
public PrintingPageType PrintingPageType { get; set; }
```

### Notlar

Sayfadaki içerik seyrekse, çıktı pdf dosyasında bazı sayfalar tamamen boş olacaktır. Bu boş sayfaları istemiyorsanız, bunları atlamak için bu seçeneği kullanabilirsiniz.

### Örnekler

Aşağıdaki kod, boş sayfaları veya yalnızca hücre arka planı, kenarlıklar gibi bazı stil içeriği içeren sayfaları atlar.

```csharp
Workbook wb = new Workbook("Book1.xlsx");

PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();

//boş sayfaları yoksay
pdfSaveOptions.PrintingPageType = PrintingPageType.IgnoreBlank;

wb.Save("output_ignore_blank_page.pdf", pdfSaveOptions);


//boş sayfaları yoksay and pages which only contains some style content like cell background
pdfSaveOptions.PrintingPageType = PrintingPageType.IgnoreStyle;

wb.Save("output_ignore_blank_and_style_page.pdf", pdfSaveOptions);
```
