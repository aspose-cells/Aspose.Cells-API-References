## StreamProviderOptions.CustomPath property

Yönlendirilen kaynak için oluşturulan html dosyasına kaydedilen kullanıcı özel yolu (URL). Kullanıcı tarafından tanımlanmadıysa, VarsayılanYol kullanılacaktır. Örneğin, sayfa verileri kullanıcı tarafından d:/sheet001.htm'ye kaydedilecektir, ana html dosyasında kullanılan url "d:/sheet001.htm" veya ana html dosyası tarafından erişilebilen diğer geçerli göreli yol.
