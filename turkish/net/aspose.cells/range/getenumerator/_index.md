## Range.GetEnumerator method

Bu Aralıktaki hücreler için numaralandırıcıyı alır.

```csharp
public IEnumerator GetEnumerator()
```

### Geri dönüş değeri

hücre numaralandırıcı

### Notlar

Döndürülen Numaralandırıcı tarafından öğeler arasında gezinirken, collection hücreleri değiştirilmemelidir (yeni Hücre/Satırın başlatılmasına veya mevcut Hücre/Satırın silinmesine neden olacak işlemler gibi). Aksi takdirde, numaralandırıcı tüm hücreleri doğru şekilde geçemeyebilir. (bazı öğeler tekrar tekrar geçilebilir veya atlanabilir).

### Örnekler

```csharp
[C#]
Workbook workbook = new Workbook("template.xlsx");
Cells cells = workbook.Worksheets[0].Cells;

IEnumerator en = cells.CreateRange("B2:C3").GetEnumerator();
while (en.MoveNext())
{
    Cell cell = (Cell)en.Current;
    Console.WriteLine(cell.Name + ": " + cell.Value);
}
```
