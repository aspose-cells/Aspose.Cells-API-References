## Area.InvertIfNegative property

Özellik doğruysa ve grafik noktasının değeri negatif bir sayıysa, ön plan rengi ve arka plan rengi değiştirilir.

```csharp
public bool InvertIfNegative { get; set; }
```

### Örnekler

```csharp

[C#]

//Bir Çalışma Kitabı nesnesini başlatma
Workbook workbook = new Workbook();
//Workbook nesnesine yeni bir çalışma sayfası ekleme
int sheetIndex = workbook.Worksheets.Add();
//Yeni eklenen çalışma sayfasının sayfa indeksini geçerek referansının alınması
Worksheet worksheet = workbook.Worksheets[sheetIndex];
//"A1" hücresine örnek değer ekleme
worksheet.Cells["A1"].PutValue(50);
//"A2" hücresine örnek değer ekleme
worksheet.Cells["A2"].PutValue(-100);
//"A3" hücresine örnek değer ekleme
worksheet.Cells["A3"].PutValue(150);
//Çalışma sayfasına grafik ekleme
int chartIndex = worksheet.Charts.Add(ChartType.Column, 5, 0, 15, 5);
//Yeni eklenen grafiğin örneğine erişim
Chart chart = worksheet.Charts[chartIndex];
//"A1" hücresinden "A3"e kadar olan grafiğe NSeries (grafik veri kaynağı) ekleme
chart.NSeries.Add("A1:A3", true);
chart.NSeries[0].Area.InvertIfNegative = true;
// 1. NSeries alanının ön plan rengini ayarlama
chart.NSeries[0].Area.ForegroundColor = Color.Red;
//1. NSeries alanının arka plan rengini ayarlama.
//İkinci grafik noktasının görüntülenen alan rengi, arka plan rengi olacaktır.
chart.NSeries[0].Area.BackgroundColor = Color.Yellow;
//Excel dosyasını kaydetme
workbook.Save("book1.xls");
```
