## ShapeCollection.AddIcons method

svg görüntüsü ekler.

```csharp
public Picture AddIcons(int upperLeftRow, int top, int upperLeftColumn, int left, int height, 
    int width, byte[] imageByteData, byte[] compatibleImageData)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| upperLeftRow | Int32 | Sol üst satır dizini. |
| top | Int32 | Piksel biriminde, şeklin sol satırından dikey kaymasını temsil eder. |
| upperLeftColumn | Int32 | Sol üst sütun dizini. |
| left | Int32 | Şeklin sol sütunundan piksel birimi cinsinden yatay kayması. |
| height | Int32 | Piksel birimi cinsinden şeklin yüksekliği. |
| width | Int32 | Piksel birimi cinsinden şeklin genişliği. |
| imageByteData | Byte[] | Görüntü bayt verileri. |
| compatibleImageData | Byte[] | Excel 2016 veya daha düşük sürümlerle uyumlu olması için svg'den dönüştürülmüş görüntü verileri. |

### Örnekler

```csharp

using (FileStream fs = new FileStream("icon.svg", FileMode.Open))
```
