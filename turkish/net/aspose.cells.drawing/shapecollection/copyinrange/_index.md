## ShapeCollection.CopyInRange method

Aralıktaki şekilleri hedef aralığa kopyalayın.

```csharp
public void CopyInRange(ShapeCollection sourceShapes, CellArea ca, int destRow, int destColumn, 
    bool isContained)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| sourceShapes | ShapeCollection | Kaynak şekiller. |
| ca | CellArea | Kaynak aralığı. |
| destRow | Int32 | Dest aralığının dest satır dizini. |
| destColumn | Int32 | Hedef aralığının hedef sütunu. |
| isContained | Boolean | Yalnızca aralıktaki şekilleri kopyalayıp kopyalayamayacağı. true ise, yalnızca aralıktaki şekilleri kopyalar. Aksi takdirde MS Office olarak çalışır. |

### Örnekler

```csharp

[C#]
//şekil ekle
shapes.AddRectangle(2, 0, 2, 0, 130, 130);
CellArea area2 = new CellArea();
area2.StartColumn = 1;
area2.StartRow = 1;
area2.EndColumn = 5;
area2.EndRow = 11;

//kopyala
shapes.CopyInRange(shapes, area2, 12, 1, false);

```
