```csharp
public void SetInputRange(string formula, bool isR1C1, bool isLocal)
```

| Parametre | Tip | Tanım |
| --- | --- | --- |
| formula | String | Kontrolü doldurmak için kullanılan aralık. |
| isR1C1 | Boolean | Formülün R1C1 olarak biçimlendirilmesi gerekip gerekmediği. |
