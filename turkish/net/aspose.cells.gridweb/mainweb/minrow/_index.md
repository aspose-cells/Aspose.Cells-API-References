## MainWeb.MinRow property

Web sayfasının minimum görüntü satırı indeksini (sıfır tabanlı) alır veya ayarlar. Kontrol, MinRow'un daha küçük değerini ve sayfa verilerinin min satırını kullanır.

```csharp
public int MinRow { get; set; }
```

### Örnekler

```csharp
[C#]
	// 4x4 "görüntü penceresi" oluşturur.
	GridWeb1.MinRow = 2;
	GridWeb1.MaxRow = 5;
	GridWeb1.MinColumn = 3;
	GridWeb1.MaxColumn = 6;

[Visual Basic]
	' Creates a 4x4 "display window".
```
