## MainWeb.MaxColumn property

Web sayfasının maksimum görüntüleme sütunu dizinini (sıfır tabanlı) alır veya ayarlar. Denetim, MaxColumn ve sayfa verilerinin maksimum sütununun daha büyük değerini kullanır.

```csharp
public int MaxColumn { get; set; }
```

### Örnekler

```csharp
[C#]
	// 4x4 "görüntü penceresi" oluşturur.
	GridWeb1.MinRow = 2;
	GridWeb1.MaxRow = 5;
	GridWeb1.MinColumn = 3;
	GridWeb1.MaxColumn = 6;

[Visual Basic]
	' Creates a 4x4 "display window".
	GridWeb1.MinRow = 2
```
