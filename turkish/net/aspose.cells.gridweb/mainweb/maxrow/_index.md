## MainWeb.MaxRow property

Web sayfasının maksimum görüntüleme satırı indeksini (sıfır tabanlı) alır veya ayarlar. Kontrol, MaxRow ve sayfa verilerinin maksimum satırının daha büyük değerini kullanır.

```csharp
public int MaxRow { get; set; }
```

### Örnekler

```csharp
[C#]
	// 4x4 "görüntü penceresi" oluşturur.
	GridWeb1.MinRow = 2;
	GridWeb1.MaxRow = 5;
	GridWeb1.MinColumn = 3;
	GridWeb1.MaxColumn = 6;
```
