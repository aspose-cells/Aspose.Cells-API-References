##  add_list_box(upper_left_row, top, upper_left_column, left, height, width) {#int-int-int-int-int-int}

###  İadeler

```python
def add_list_box(self, upper_left_row, top, upper_left_column, left, height, width):

```


| parametreler| Tip| Tanım|
| :- | :- | :- |
| upper_left_row | int | Sol üst sıra dizini.|
| top | int | Piksel birimi cinsinden, ListBox öğesinin sol satırından dikey uzaklığını temsil eder.|
| upper_left_column | int | Sol üst sütun dizini.|
| left | int | Piksel birimi cinsinden, ListBox öğesinin sol sütunundan yatay uzaklığını temsil eder.|
| height | int | Piksel birimi cinsinden ListBox yüksekliğini temsil eder.|
| width | int | Piksel biriminde ListBox genişliğini temsil eder.|
