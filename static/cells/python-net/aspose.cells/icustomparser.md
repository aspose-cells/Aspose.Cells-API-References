## ICustomParser class
Allows users to add their custom value parser for parsing string values to other proper cell value object.
The ICustomParser type exposes the following members:
### Methods
| Method | Description |
| :- | :- |
| [`parse_object(self, value)`](/cells/python-net/aspose.cells/icustomparser/parse_object/#system.string) | Parses given string to proper value object. |
| [`get_format(self)`](/cells/python-net/aspose.cells/icustomparser/get_format/#) | Gets the formatting pattern corresponding to the parsed value by last invocation of [`ICustomParser.parse_object`](/cells/python-net/aspose.cells/icustomparser/parse_object). |
