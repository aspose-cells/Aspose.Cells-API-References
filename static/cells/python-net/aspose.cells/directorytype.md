## DirectoryType enumeration
Represents the directory  type of the file name.
The DirectoryType type exposes the following members:
### Fields
| Field | Description |
| :- | :- |
| VOLUME | Represents an MS-DOS drive letter. It is followed by the drive letter.
| SAME_VOLUME | Indicates that the source workbook is on the same drive as the dependent workbook (the drive letter is omitted) |
| DOWN_DIRECTORY | Indicates that the source workbook is in a subdirectory of the current directory. |
| UP_DIRECTORY | Indicates that the source workbook is in the parent directory of the current directory. |
