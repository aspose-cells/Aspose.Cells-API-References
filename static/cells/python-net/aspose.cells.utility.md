##aspose.cells.utility
The **Aspose.Cells.Utility**  namespace contains utility classes processing JSON formats for Excel Spreadsheet files.

| :- | :- |
| [`ConversionUtility`](/cells/python-net/aspose.cells.utility/conversionutility) | Represents utility to convert files to other formats. |

| [`JsonLayoutOptions`](/cells/python-net/aspose.cells.utility/jsonlayoutoptions) | Represents the options of json layout type. |
| [`JsonUtility`](/cells/python-net/aspose.cells.utility/jsonutility) | Represents the utility class of processing json. |
