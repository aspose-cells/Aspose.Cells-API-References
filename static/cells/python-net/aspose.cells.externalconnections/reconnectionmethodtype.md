## ReConnectionMethodType enumeration
Specifies what the spreadsheet application should do when a connection fails.
The ReConnectionMethodType type exposes the following members:
### Fields
| Field | Description |
| :- | :- |
| REQUIRED | On refresh use the existing connection information and if it ends up being invalid
| ALWAYS | On every refresh get updated connection information from the external connection file,
| NEVER | Never get updated connection information from the external connection file
