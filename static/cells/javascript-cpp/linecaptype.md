## LineCapType enumeration
Represents the caps of a line
### Values
| Name | Value | Description |
| --- | --- | --- |
| Square | `0` | Square protrudes by half line width. |
| Round | `1` | Rounded ends. |
| Flat | `2` | Line ends at end point. |
| None | `3` | None cap |
