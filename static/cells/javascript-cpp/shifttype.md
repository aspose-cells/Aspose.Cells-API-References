## ShiftType enumeration
Represent the shift options when deleting a range of cells.
### Values
| Name | Value | Description |
| --- | --- | --- |
| Down | `0` | Shift cells down. |
| Left | `1` | Shift cells left. |
| None | `2` | Do not shift cells. |
| Right | `3` | Shift cells right. |
| Up | `4` | Shift cells up. |
