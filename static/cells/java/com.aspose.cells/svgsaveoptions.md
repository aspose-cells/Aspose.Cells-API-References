##SvgSaveOptions
Represents Svg save options.
**Inheritance:**

Represents Svg save options. For advanced usage, please use [WorkbookRender](../../com.aspose.cells/workbookrender) or [SheetRender](../../com.aspose.cells/sheetrender).
