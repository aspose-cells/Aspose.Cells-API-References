##GridImageOrPrintOptions.AllColumnsInOnePagePerSheet
GridImageOrPrintOptions property. If AllColumnsInOnePagePerSheet is true  all column content of one sheet will output to only one page in result. The width of paper size of pagesetup will be invalid and the other settings of pagesetup will still take effect
## GridImageOrPrintOptions.AllColumnsInOnePagePerSheet property
If AllColumnsInOnePagePerSheet is true , all column content of one sheet will output to only one page in result. The width of paper size of pagesetup will be invalid, and the other settings of pagesetup will still take effect.
