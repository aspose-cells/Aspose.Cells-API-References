##GridJsOptions.IslimitShapeOrImage
GridJsOptions property. Gets/Sets whether to limit the total display shape/image count in one worksheet if set to true GridJs will limit the total count of the display shapes or images in one worksheet to MaxShapeOrImageCount the default value is true
