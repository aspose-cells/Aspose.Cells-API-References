##Aspose.Cells.Vba
The Aspose.Cells.Vba namespace contains classes to access Visual Basic Analysis projects and modules in Excel Spreadsheet files
The **Aspose.Cells.Vba** namespace contains classes to access Visual Basic Analysis projects and modules in Excel Spreadsheet files.
