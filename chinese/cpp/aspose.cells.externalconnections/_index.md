## 类

| 类 | 描述 |
| --- | --- |
| [ConnectionParameter](./connectionparameter/) | 指定与外部数据连接使用的任何参数相关的属性，参数适用于 ODBC 和 Web 查询。 |
| [ConnectionParameterCollection](./connectionparametercollection/) | 指定 [ConnectionParameter](./connectionparameter/) 集合。 |
| [DataModelConnection](./datamodelconnection/) | 指定数据模型连接。 |
| [DBConnection](./dbconnection/) | 指定与 ODBC 或 OLE DB 外部数据连接关联的所有属性。 |
| [ExternalConnection](./externalconnection/) | 指定外部数据连接。 |
| [ExternalConnectionCollection](./externalconnectioncollection/) | 指定 [ExternalConnection](./externalconnection/) 集合。 |
| [WebQueryConnection](./webqueryconnection/) | 指定 Web 查询源的属性。Web 查询将从 HTML 表格检索数据，并且还可以提供 HTTP \"Get\" 参数，这些参数将在生成 HTML 时由 Web 服务器处理，通过包含参数和参数元素。 |
