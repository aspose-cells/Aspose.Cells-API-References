## TickMarkType枚举
表示指定轴的刻度线类型。



TickMarkType 类型公开了以下成员：

### 字段
|场地|描述|
| :- | :- |
| CROSS |刻度线类型为十字。|
| INSIDE |刻度标记类型为Inside。|
| NONE |刻度标记类型为None。|
| OUTSIDE |刻度线类型为Outside|
