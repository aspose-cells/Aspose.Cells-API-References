## treat_quote_prefix_as_value 属性

指示前导单引号是否应作为一个单元格值的一部分。
默认为 true。如果为 false，则将从相应单元格的值中删除前导单引号
和 [Style.quote_prefix](/cells/zh/python-net/aspose.cells/style#quote_prefix) 将被设置为真为单元格。
