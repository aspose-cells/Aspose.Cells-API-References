## HtmlHiddenColDisplayType枚举
表示在html中显示隐藏列的两种类型。



HtmlHiddenColDisplayType 类型公开了以下成员：

### 字段
|场地|描述|
| :- | :- |
| HIDDEN |隐藏 html 页面中的隐藏列。|
| REMOVE |删除 html 页面中的隐藏列。|
