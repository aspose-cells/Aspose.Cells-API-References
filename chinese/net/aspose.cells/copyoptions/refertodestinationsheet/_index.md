## CopyOptions.ReferToDestinationSheet property

复制同一个文件中的范围且图表引用源工作表时， False 表示复制的图表的数据源不会改变。 True 表示复制的图表的数据源引用目标工作表。

```csharp
public bool ReferToDestinationSheet { get; set; }
```

### 评论

默认值为 false，它作为 MS Excel 工作。
