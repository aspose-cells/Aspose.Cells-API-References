## TxtLoadOptions.TreatQuotePrefixAsValue property

指示是否应将前导单引号作为一个单元格值的一部分。 默认为真。如果为 false，则将从相应单元格的值 中删除前导单引号，并且[`QuotePrefix`](../../style/quoteprefix)将为单元格设置为 true。

public bool TreatQuotePrefixAsValue { get; set; }
