## TxtLoadOptions.ExtendToNextSheet property

当数据的行或列超过限制时，是否将数据扩展到下一张。 如果该属性为true，额外的数据将被扩展到当前一张之后的下一张（如果当前一张是最后一张， 将追加新的表格到当前工作簿）。 如果此属性为false，超出限制的数据将被忽略。 默认为false；

```csharp
public bool ExtendToNextSheet { get; set; }
```
