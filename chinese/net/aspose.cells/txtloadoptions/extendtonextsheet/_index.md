## TxtLoadOptions.ExtendToNextSheet property

当数据的行或列超过限制时，是否将数据扩展到下一张。 如果此属性为真，额外的数据将扩展到当前工作表后面的下一个工作表（如果当前工作表是最后一个工作表，则 新工作表将附加到当前工作簿）。 如果该属性为假，超出限制的数据将被忽略。 默认为假；

```csharp
public bool ExtendToNextSheet { get; set; }
```
