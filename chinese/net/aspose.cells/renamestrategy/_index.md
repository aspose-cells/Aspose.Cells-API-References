## RenameStrategy enumeration

列重复名称的策略选项。

```csharp
public enum RenameStrategy
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Exception | `0` | 引发异常。 |
| Digit | `1` | 以数字命名。重复的名称将变为 ...1、...2 等。 |
| Letter | `2` | 以字母命名.. 重复的名称将变为 ...A、...B 等。 |

### 评论

处理带表头的数据时，有些场景要求表头对所有列都不能重复。 比如导出数据到数据表时，要求表头作为数据表的列名， 表头重复值无效. 对于此类情况，用户可以通过指定此策略来确定如何处理。
