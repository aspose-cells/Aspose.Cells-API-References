## LightCellsDataProvider.StartRow method

```csharp
public void StartRow(Row row)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| row | Row | 用于填充数据的行对象。它的行索引是最近一次调用[`NextRow`](../nextrow)的返回值。 如果行已在内部单元格模型中初始化，则将使用现有的行对象。 否则将使用临时 Row 对象来实现填充数据。 |

### 评论

它将在保存行及其单元格数据的开始时调用。 如果当前行有一些自定义属性，例如高度、样式等， 实现应该在此处将这些属性设置为给定的 Row 对象。
