## WorkbookSettings.Region property

获取或设置工作簿的区域设置。

```csharp
public CountryCode Region { get; set; }
```

### 评论

1. Aspose.Cells 组件用于从模板文件加载的工作簿的区域设置: i)。对于 XLS 文件，有为区域设置定义的字段，并且 MS Excel 会在保存 XLS 文件时将区域设置数据保存到文件中。 因此，我们将模板文件中保存的区域用于工作簿。 如果您不想使用XLS文件中保存的区域，请在加载模板文件后将其重置为预期的区域（例如，CountryCode.Default）。 而且，我们在保存 XLS 文件时也将用户指定的值（通过这种方法）保存到文件中。 ii)。对于其他文件格式，例如 XLSX、XLSB...等，文件格式规范中没有为区域设置定义字段。 因此，我们为工作簿使用应用程序环境的区域设置。 并且，不能为具有这些文件格式的生成文件保留用户指定的值（通过此方法）。 2. MS Excel中的查看效果: 这里应用的区域设置只能在Aspose.Cells组件运行时生效，在MS Excel查看生成的文件时不生效. 即使生成的 XLS 文件中保存了指定的区域设置数据，当使用 MS Excel 查看/编辑它时， 使用 MS Excel 进行格式化的区域是始终是运行 MS Excel 的环境的默认区域设置， 不是保存在文件中的设置。这是 MS Excel 的行为，无法通过代码更改。
