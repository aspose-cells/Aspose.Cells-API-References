## WorkbookSettings.Region property

获取或设置工作簿的区域设置。

```csharp
public CountryCode Region { get; set; }
```

### 评论

1. Aspose.Cells 组件用于从模板文件加载的工作簿的区域设置： i)。对于 XLS 文件，有为区域设置定义的字段，MS Excel 在保存 XLS 文件时确实会将区域设置数据保存到文件中。 因此，我们将模板文件中保存的区域用于工作簿。 如果不要使用保存在XLS文件中的区域，请在加载模板文件后将其重置为预期的区域（例如，CountryCode.Default）。 并且，我们也将用户指定的值（通过此方法）保存到文件中保存 XLS 文件时。 ii)。对于其他文件格式，如 XLSX、XLSB...等，在文件格式规范中没有为区域设置定义字段。 所以，我们将应用程序环境的区域设置用于工作簿。 并且，用户指定的值（通过此方法）无法为生成的文件保留这些文件格式。 2. 对于 MS Excel 中的视图效果： 此处应用的区域设置只能在运行时与 Aspose.Cells 组件一起生效，而不是使用 MS Excel 查看生成的文件时。 即使生成的 XLS 文件中保存了指定的区域设置数据，使用 MS Excel 查看/编辑它时， 使用 MS Excel 执行格式化的区域始终为默认值MS Excel 运行环境的区域设置， 不是文件中保存的设置。这是 MS Excel 的行为，无法通过代码更改。
