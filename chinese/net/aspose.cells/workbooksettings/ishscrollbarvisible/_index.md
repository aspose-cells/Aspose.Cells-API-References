## WorkbookSettings.IsHScrollBarVisible property

获取或设置一个值，指示生成的电子表格是否包含水平滚动条。

```csharp
public bool IsHScrollBarVisible { get; set; }
```

### 评论

默认值为真。

### 例子

以下代码使水平滚动条对电子表格不可见。

```csharp
[C#]
 //隐藏Excel文件的水平滚动条
settings.IsHScrollBarVisible = false;

[Visual Basic]
' Hide the horizontal scroll bar of the Excel file.
settings.IsHScrollBarVisible = False
```
