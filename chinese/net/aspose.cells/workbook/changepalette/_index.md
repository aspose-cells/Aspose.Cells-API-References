## Workbook.ChangePalette method

更改指定索引中电子表格的调色板。

```csharp
public void ChangePalette(Color color, int index)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| color | Color | 颜色结构。 |
| index | Int32 | 调色板索引，0 - 55。 |

### 评论

调色板有 56 个条目，每个条目由一个 RGB 值。 如果设置的颜色不在调色板中，它不会生效。 所以如果你想设置自定义颜色，请先更改调色板。
