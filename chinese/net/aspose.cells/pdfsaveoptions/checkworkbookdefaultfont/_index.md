## PdfSaveOptions.CheckWorkbookDefaultFont property

当 Excel 中的字符为 Unicode 且未在单元格样式中设置正确的字体时， 它们可能在 pdf、图像中显示为块。 将此设置为 true 以尝试使用工作簿的默认字体首先显示这些字符。

```csharp
public bool CheckWorkbookDefaultFont { get; set; }
```

### 评论

默认为真。
