## PdfSaveOptions.PageCount property

获取或设置要保存的页数。

```csharp
public int PageCount { get; set; }
```

### 评论

默认是 System.Int32.MaxValue 这意味着所有页面都将被渲染..

### 例子

以下示例显示如何将 Microsoft Excel 文件中的一系列页面（3 和 4）呈现为 PDF。

```csharp
//打开一个Excel文件
Workbook wb = new Workbook("Book1.xlsx");

PdfSaveOptions options = new PdfSaveOptions();

//只打印输出PDF中的第3页和第4页
//起始页索引（从0开始的索引）
options.PageIndex = 3;
//要打印的页数
options.PageCount = 2;

//保存PDF文件
wb.Save("output.pdf", options);
```
