## PdfSaveOptions.CheckFontCompatibility property

表示是否检查文本中每个字符的字体兼容性。

```csharp
public bool CheckFontCompatibility { get; set; }
```

### 评论

默认值为真。 禁用此属性可能会提供更好的性能。 但是当文本/字符的默认或指定字体不能用于渲染时，生成的pdf中可能会出现 不可读的字符（例如块）。 对于这种情况，用户应该将此属性保持为真，以便 替代字体可以被搜索并用于呈现文本；
