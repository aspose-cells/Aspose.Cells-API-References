## PdfSaveOptions.PageIndex property

获取或设置要保存的第一页的从 0 开始的索引。

```csharp
public int PageIndex { get; set; }
```

### 评论

默认为 0。

### 例子

以下示例说明如何将 Microsoft Excel 文件中的一系列页面（3 和 4）呈现为 PDF。

```csharp
//打开一个Excel文件
orkbook wb = new Workbook("Book1.xlsx");

dfSaveOptions options = new PdfSaveOptions();

//在输出PDF

/起始页索引（从0开始的索引）
ptions.PageIndex = 3;
//要打印的页数
ptions.PageCount = 2;

//保存PDF文件
b.Save("output.pdf", options);
```
