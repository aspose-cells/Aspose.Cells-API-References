public enum DataBarAxisPosition

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Automatic | `0` | 根据范围内最小负值与最大正值之比，在可变位置显示轴。 正值以从左到右的方向显示。 负值以从右到左的方向显示。 当所有值为正或所有值为负时，不显示轴。 |
| Midpoint | `1` | 在单元格的中点显示轴，无论范围内的值集如何。 正值以从左到右的方向显示。 负值以从右到左的方向显示。 |
| None | `2` | 不显示坐标轴，从左到右显示正负值。 |
