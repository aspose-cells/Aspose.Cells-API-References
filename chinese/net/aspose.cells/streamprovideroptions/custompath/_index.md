## StreamProviderOptions.CustomPath property

保存在为引用源生成的 html 文件中的用户自定义路径（URL）。如果用户未定义，则使用 DefaultPath。 例如，用户将表单数据保存到 d:/sheet001.htm，主 html 文件中使用的 url 应为“d:/sheet001.htm”或主 html 文件可以访问的其他有效相对路径。
