## AbstractCalculationEngine.ProcessBuiltInFunctions property

内置引擎已经支持的内置函数是否应该由本实现检查处理。 默认为false。 如果用户需要更改某些内置函数的计算逻辑，该属性应该是设置为真。

public virtual bool ProcessBuiltInFunctions { get; }
