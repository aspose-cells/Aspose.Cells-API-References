## AbstractCalculationEngine.IsParamLiteralRequired property

表示此引擎在计算时是否需要参数的文字文本。默认值为假。

```csharp
public virtual bool IsParamLiteralRequired { get; }
```

### 评论

如果此自定义计算引擎需要参数的文字文本，则需要更多的堆栈来缓存参数的文字文本和Calculate()可以递归调用方法来计算参数的值。 通常计算公式不需要文字文本，对于大多数实现，此方法应返回 false 以获得更好的性能。
