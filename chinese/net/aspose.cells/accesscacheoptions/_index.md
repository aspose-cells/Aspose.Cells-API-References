## AccessCacheOptions enumeration

用于数据访问的缓存选项。可以结合&#x7C;多个选项一起使用的运算符。

```csharp
[Flags]
public enum AccessCacheOptions
```

| 姓名 | 价值 | 描述 |

| None | `0` | 没有任何数据访问的缓存。 |

### 评论

对于某些特征，访问大型数据集需要大量重复和复杂的操作 例如搜索、计算等，这些操作会花费很多额外的时间。 一般情况下，所有依赖数据在访问过程中都保持不变，所以可以建立一些缓存来使用 提高访问性能。 为此，我们提供了这个API，以便用户可以指定哪种数据访问需要 以通过可能的缓存机制进行优化。 &lt;br&gt;&lt;/br&gt; 请注意，对于不同的选项，可能需要不同的数据集为“只读”。 并且访问数据的性能取决于很多方面，使用缓存机制 并不能保证性能会有所提高。对于某些情况， 例如要访问的数据集很小，使用缓存可能会导致更多时间，因为 缓存本身也需要一定的额外时间。
