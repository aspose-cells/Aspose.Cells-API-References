## FormulaSettings.CalculationId property

指定用于计算工作簿中值的计算引擎的版本。

```csharp
public string CalculationId { get; set; }
```

### 评论

此属性仅用于将设置保存到生成的电子表格文件 以便其他应用程序（例如 ms excel）在加载和操作生成的文件时可能会采取相应的行动。 对于 ms excel，如果此属性的值小于与打开结果文件的应用程序关联的重新计算引擎标识符 相比，应用程序将在加载文件后立即重新计算此工作簿上所有公式 的结果。 出于对大多数用户应用程序的性能考虑，我们不计算任何公式自动工作簿， 无论为此属性设置了什么值。
