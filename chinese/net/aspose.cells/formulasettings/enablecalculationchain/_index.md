## FormulaSettings.EnableCalculationChain property

是否开启公式计算链。默认为假。

```csharp
public bool EnableCalculationChain { get; set; }
```

### 评论

当工作簿中有很多公式并且用户需要重复计算它们时 只修改其中的一小部分，启用计算链可能对性能有所帮助。 另一方面，如果启用了chain，维护chain的模型需要额外的内存， 并且它还需要更多的cpu时间来进行一些其他操作，例如更改单元格的值或公式。 将此属性从false更改为true后，计算链将在工作簿第一次计算时分析构建 ，因此第一次计算所需时间 可能比没有链的正常计算多。
