## FormulaSettings.EnableCalculationChain property

是否启用公式的计算链。默认为假。

```csharp
public bool EnableCalculationChain { get; set; }
```

### 评论

当工作簿中有很多公式，用户需要重复计算时 只修改一小部分，启用计算链可能对性能有帮助。 另一方面，如果启用了链，维护链的模型需要额外的内存， 并且它还需要更多的cpu时间来进行一些其他操作，例如更改单元格的值或公式。 将此属性从false更改为true后，将分析并构建计算链 在工作簿第一次计算的时间，所以第一次计算所需的时间 可能比没有链的正常计算要多。
