## Style.CultureCustom property

获取并设置数字格式的区域性模式字符串。 如果没有为此对象设置数字格式，则返回null。 如果内置数字格式，则返回与内置数字对应的模式字符串。

```csharp
public string CultureCustom { get; set; }
```

### 评论

对于内置数字格式，模式内容（例如，对于某些语言环境，一种内置日期格式为“m/d/y”， 但对于某些其他语言环境，它变为“d/m/y”）和格式说明符（如， 某些语言环境使用 'y' 以外的字符来表示日期格式的年份部分） 与文化相关； 对于用户指定的自定义格式，仅根据文化更改格式说明符， 其他部分的格式模式不会被修改。
