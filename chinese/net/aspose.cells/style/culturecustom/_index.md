## Style.CultureCustom property

获取和设置数字格式的文化相关模式字符串。 如果没有为此对象设置数字格式，则返回 null。 如果内置数字格式，则返回内置数字对应的模式字符串。

```csharp
public string CultureCustom { get; set; }
```

### 评论

对于内置数字格式，两种模式内容（例如，一种内置日期格式是“m /d/y" 对于某些语言环境， 但对于某些其他语言环境，它变为 "d/m/y") 和格式说明符（例如 某些语言环境正在使用'y' 以外的字符来表示日期格式的年份部分） 取决于文化； 对于用户指定的自定义格式，只有格式说明符根据文化改变， 格式化模式的其他部分不会被修改。
