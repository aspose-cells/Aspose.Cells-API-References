## ShapeCollection.AddSvg method

添加 svg 图像。

```csharp
public Picture AddSvg(int upperLeftRow, int top, int upperLeftColumn, int left, int height, 
    int width, byte[] svgData, byte[] compatibleImageData)
```

| 范围 | 类型 | 描述 |
| --- | --- | --- |
| upperLeftRow | Int32 | 左上行索引。 |
| top | Int32 | 表示形状与其左行的垂直偏移量，以像素为单位。 |
| upperLeftColumn | Int32 | 左上列索引。 |
| left | Int32 | 形状与其左列的水平偏移量，以像素为单位。 |
| height | Int32 | 形状的高度，以像素为单位。 |
| width | Int32 | 形状的宽度，以像素为单位。 |
| svgData | Byte[] | svg 图像数据。 |
| compatibleImageData | Byte[] | 从 svg 转换的图像数据，以便与 Excel 2016 或更低版本兼容。 |

### 例子

```csharp

[C#]
```
