## ChartSplitType enumeration

表示饼图或饼图的两个部分的拆分方式。

```csharp
public enum ChartSplitType
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Position | `0` | 表示数据点应在饼图 和第二个图表之间拆分，方法是将数据点的最后一个Split Position 放在第二个图表中 |
| Value | `1` | 表示数据点应在饼图 和第二张图表之间进行拆分，方法是将 值小于分割位置的数据点放在第二张图表中。 |
| PercentValue | `2` | 表示数据点应在饼图 和第二张图表之间拆分，方法是将 百分比小于拆分位置百分比的点放在 第二张图表中。 |
| Custom | `3` | 表示数据点应根据自定义拆分 值在饼图 和第二张图表之间拆分。 |
| Auto | `4` | 表示数据点应使用此图表类型的 default 机制进行拆分。 |
