## ChartSplitType enumeration

表示饼图或饼图的两个部分的拆分方式。

```csharp
public enum ChartSplitType
```

### 价值观

| 姓名 | 价值 | 描述 |
| --- | --- | --- |
| Position | `0` | 表示数据点应在饼图 和第二张图表之间进行拆分，将最后一个拆分位置 放在第二个图表中的数据点 |
| Value | `1` | 表示数据点应在饼图 和第二张图表之间通过将数据点与 值分开小于第二张图表中的拆分头寸。 |
| PercentValue | `2` | 表示数据点应在饼图 和第二张图表之间拆分，将点与 百分比相减比 第二个图表中的拆分头寸百分比。 |
| Custom | `3` | 表示数据点应根据自定义拆分 值在饼图 和第二个图表之间拆分。 |
| Auto | `4` | 表示数据点应使用此图表类型的默认 机制进行拆分。 |
