### 例子

```csharp

[C#]

//实例化一个工作簿对象
Workbook workbook = new Workbook();
 // 初始化 VBA 项目。
VbaProject vbaProject = workbook.VbaProject; 
// 添加一个新模块。
int index = vbaProject.Modules.Add(VbaModuleType.Class, "test");

VbaModule vbaModule = vbaProject.Modules[index];

//保存Excel文件
```
