## ImageOrPrintOptions.OnePagePerSheet property

如果 OnePagePerSheet 为 true ，则结果中一张表的所有内容将仅输出到一页。 pagesetup 的纸张大小会失效，pagesetup 的其他设置仍然有效。

public bool OnePagePerSheet { get; set; }
