## PivotTable.DataFields property

获取一个 PivotField 对象，该对象表示数据透视表中的所有数据字段。 只读。只有当 DataPiovtFiels 中有两个或多个数据字段时才会初始化。 它仅用于将 DataPivotField 添加到数据透视表的行/列区域。默认在行区域。

public PivotFieldCollection DataFields { get; }
