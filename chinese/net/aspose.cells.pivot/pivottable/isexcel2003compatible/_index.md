## PivotTable.IsExcel2003Compatible property

指定刷新数据透视表时数据透视表是否兼容Excel2003， 如果为true，则字符串必须小于或等于255个字符，所以如果字符串大于 255 个字符， 将被截断。如果为 false，则字符串将没有上述限制。 默认值为真。

public bool IsExcel2003Compatible { get; set; }
