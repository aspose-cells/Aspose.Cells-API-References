## ExternalConnection.OnlyUseConnectionFile property

表示刷新连接时，电子表格应用程序是否应始终且仅使用 odcFile 属性 指示的外部连接文件中的 连接信息。如果为 false，则电子表格应用程序 应遵循 reconnectionMethod 属性 指示的过程

public bool OnlyUseConnectionFile { get; set; }
