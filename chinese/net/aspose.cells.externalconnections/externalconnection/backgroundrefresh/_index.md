## ExternalConnection.BackgroundRefresh property

表示是否可以在后台（异步）刷新连接。 如果连接的首选用法是在后台异步刷新，则为 true； 如果连接的首选用法是在前台同步刷新，则为 false。

public bool BackgroundRefresh { get; set; }
