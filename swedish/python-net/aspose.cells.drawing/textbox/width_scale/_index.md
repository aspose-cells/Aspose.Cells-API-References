##  width_scale fastighet

Hämtar och ställer in breddskalan, i procentenhet av den ursprungliga bildens bredd.
Om formen inte är bild, returnerar egenskapen WidthScale endast 100;

###  Exempel

```python

if shape.width_scale == 3:
    shape.width_scale = 1
```
