 De**Aspose.Cells.Timelines** namnrymden innehåller klasser som gör det möjligt att komma åt och rita tidslinjer i Excel-kalkylbladsfiler.

###  Klasser
| Klass| Beskrivning|
| :- | :- |
| [Timeline](/cells/sv/python-net/aspose.cells.timelines/timeline) |Sammanfattande beskrivning av tidslinjevy<br/> På grund av MS Excel stöder inte Excel 2003 tidslinje|
| [TimelineCollection](/cells/sv/python-net/aspose.cells.timelines/timelinecollection) | Anger samlingen av alla tidslinjeobjekt på det angivna kalkylbladet.<br/> På grund av MS Excel stöder inte Excel 2003 tidslinje.|
