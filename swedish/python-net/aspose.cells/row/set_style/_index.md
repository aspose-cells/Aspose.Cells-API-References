##  set_style(style) {#Style}
Ställer in stilen för den här raden.



```python
def set_style(self, style):
    ...
```


| Parametrar| Typ| Beskrivning|
| :- | :- | :- |
| style | [Style](/cells/sv/python-net/aspose.cells/style) | formatet som ska användas som standardformat för celler i den här raden.|
###  Anmärkningar

Denna metod ställer bara in den givna stilen som standardstilen för den här raden,
utan att ändra stilinställningarna för befintliga celler i den här raden.
För att uppdatera stilinställningarna för befintliga celler till den angivna stilen samtidigt,
använd [Row.apply_style(style, flag)](/cells/sv/python-net/aspose.cells/row/apply_style)
