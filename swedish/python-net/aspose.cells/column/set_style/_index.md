##  set_style(style) {#Style}
Ställer in stilen för denna kolumn.



```python
def set_style(self, style):
    ...
```


| Parametrar| Typ| Beskrivning|
| :- | :- | :- |
| style | [Style](/cells/sv/python-net/aspose.cells/style) | formatet som ska användas som standardformat för celler i den här kolumnen.|
###  Anmärkningar

Denna metod ställer bara in den givna stilen som standardstil för denna kolumn,
utan att ändra stilinställningarna för befintliga celler i den här kolumnen.
För att uppdatera stilinställningarna för befintliga celler till den angivna stilen samtidigt,
använd [Column.apply_style(style, flag)](/cells/sv/python-net/aspose.cells/column/apply_style)
