## OoxmlSaveOptions.ExportCellName property

Indikerar om export av cellnamn till Excel2007 .xlsx (.xlsm, .xltx, .xltm) fil. Om utdatafilen kan nås av SQL Server DTS måste detta värde vara sant. Om du ställer in värdet på false ökar prestandan kraftigt och filstorleken minskar när du skapar stor fil. Standardvärdet är sant.
