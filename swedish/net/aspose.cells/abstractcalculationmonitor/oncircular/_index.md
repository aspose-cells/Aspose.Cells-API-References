## AbstractCalculationMonitor.OnCircular method

Implementera denna metod för att göra affärer när du beräknar formler med cirkulära referenser.

```csharp
public virtual bool OnCircular(IEnumerator circularCellsData)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| circularCellsData | IEnumerator | IEnumerator med CalculationCell-objekt som representerar celler som är beroende av cirkulära referenser. |

### Returvärde

Om formelmotorn behöver beräkna dessa celler i cirkulär efter detta anrop. Sant för att låta formelmotorn fortsätta att göra beräkningar för dem. False för att låta formelmotorn bara markera dessa celler som beräknade.

### Anmärkningar

I implementeringen kan användaren också ställa in det förväntade värdet som beräknat resultat för en del av/alla dessa celler så att formelmotorn inte kommer att beräkna dem rekursivt.
