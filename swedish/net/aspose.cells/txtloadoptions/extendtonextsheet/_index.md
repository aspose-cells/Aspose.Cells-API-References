## TxtLoadOptions.ExtendToNextSheet property

Om data utökas till nästa ark när raderna eller kolumnerna med data överskrider gränsen. Om den här egenskapen är sann kommer extra data att utökas till nästa ark efter det nuvarande (om det aktuella arket är det sista, kommer ett nytt ark att läggas till till aktuell arbetsbok). Om den här egenskapen är falsk ignoreras data som överskrider gränsen. Default is false;
