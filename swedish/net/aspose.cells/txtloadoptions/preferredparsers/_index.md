## TxtLoadOptions.PreferredParsers property

Hämtar och ställer in föredragna värdeparsers för att ladda textfil.

```csharp
public ICustomParser[] PreferredParsers { get; set; }
```

### Anmärkningar

parsers[0] är att parsern kommer att användas för den första kolumnen i textmallfilen, parsers[1] är parsern kommer att användas för den andra kolumnen, ...etc. Den sista (parsers[parsers. length-1]) kommer att användas för alla andra kolumner som börjar från parsers.length-1. Om ett objekt är null, kommer motsvarande kolumn att tolkas av standardparsern för Aspose.Cells.
