## AbstractCalculationEngine.IsParamLiteralRequired property

Indikerar om denna motor behöver den bokstavliga texten av parametern när den gör beräkningar. Standardvärdet är false.

```csharp
public virtual bool IsParamLiteralRequired { get; }
```

### Anmärkningar

Om den här anpassade beräkningsmotorn kräver parameterns bokstavliga text, kommer fler stackar att krävas för att cachelagra den bokstavliga texten för parametrar och metoden Calculate() kan anropas rekursivt för att beräkna parameterns värde. Vanligtvis behövs inte den bokstavliga texten för att beräkna formler och denna metod bör returnera false för de flesta implementeringar för att få bättre prestanda.
