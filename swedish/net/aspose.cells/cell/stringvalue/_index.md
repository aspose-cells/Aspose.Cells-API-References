## Cell.StringValue property

Hämtar strängvärdet som finns i cellen. Om typen av denna cell är sträng, returnera sedan själva strängvärdet. För andra celltyper kommer det formaterade strängvärdet (formaterat med den angivna stilen för denna cell) att returneras. Det formaterade cellvärdet är detsamma som du kan komma från excel när du kopierar en cell som text (som kopierar cell till textredigerare eller exporterar till csv).

public string StringValue { get; }
