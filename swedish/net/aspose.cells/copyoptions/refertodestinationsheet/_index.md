## CopyOptions.ReferToDestinationSheet property

När du kopierar intervallet i samma fil och diagrammet hänvisar till källbladet, betyder False att det kopierade diagrammets datakälla inte kommer att ändras. True betyder att det kopierade diagrammets datakälla refererar till målarket.

```csharp
public bool ReferToDestinationSheet { get; set; }
```

### Anmärkningar

Standardvärdet är falskt, det fungerar som MS Excel.
