## Validation.GetListValue method

Hämta värdet för lista över valideringen för den angivna cellen.

```csharp
public object GetListValue(int row, int column)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| row | Int32 | Radindex. |
| column | Int32 | Kolumnindex. |

### Returvärde

Värdet för att skapa listan över denna validering för den angivna cellen. Om listan hänvisar till ett intervall, kommer det returnerade värdet att vara en[`ReferredArea`](../../referredarea) object; Annars kan det returnerade värdet vara null, object[] eller enkelt objekt.

### Anmärkningar

Endast för validering vars typ är List och har tillämpats på en given cell, annars kommer null att returneras.
