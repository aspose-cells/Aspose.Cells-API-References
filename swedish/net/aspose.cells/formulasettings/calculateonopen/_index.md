## FormulaSettings.CalculateOnOpen property

Indikerar om programmet måste utföra en fullständig beräkning när arbetsboken öppnas.

```csharp
public bool CalculateOnOpen { get; set; }
```

### Anmärkningar

Den här egenskapen är endast till för att spara inställningarna i resulterande kalkylblad file så att andra applikationer (som ms excel) kan agera i enlighet därmed när den resulterande filen laddas. För prestandaöverväganden för de flesta användares applikationer, beräknar vi inte någon formel i arbetsbok automatiskt, oavsett vilket värde som har ställts in för den här egenskapen.
