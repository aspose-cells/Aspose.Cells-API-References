## FormulaSettings.CalculationId property

Anger versionen av beräkningsmotorn som används för att beräkna värden i arbetsboken.

```csharp
public string CalculationId { get; set; }
```

### Anmärkningar

Den här egenskapen är endast till för att spara inställningarna i resulterande kalkylblad file så att andra applikationer (som ms excel) kan agera i enlighet därmed när de laddar och manipulerar den resulterande filen. I fallet med ms excel, om värdet på den här egenskapen är mindre än omräkningsmotorns identifierare som är associerad med applikationen som öppnar den resulterande filen, kommer applikationen att räkna om resultaten av alla formler i den här arbetsboken omedelbart efter att filen har laddats. För prestandaöverväganden för de flesta användares applikationer, beräknar vi inte någon formel på arbetsboken automatiskt, oavsett vilket värde som har ställts in för den här egenskapen.
