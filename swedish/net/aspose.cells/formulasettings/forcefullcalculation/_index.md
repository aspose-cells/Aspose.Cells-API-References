## FormulaSettings.ForceFullCalculation property

Anger om alla formler beräknas varje gång en beräkning utlöses.

```csharp
public bool ForceFullCalculation { get; set; }
```

### Anmärkningar

Den här egenskapen är endast till för att spara inställningarna i resulterande kalkylblad file så att andra applikationer (som ms excel) kan agera därefter när de laddar och manipulerar den resulterande filen. För prestandaövervägande för de flesta användares applikationer, beräknar vi inte någon formel i arbetsboken automatiskt, oavsett vilket värde som har ställts in för den här egenskapen.
