## Cells.GetDependentsInCalculation method

Hämtar alla celler vars beräknade resultat beror på specifik cell.

```csharp
public IEnumerator GetDependentsInCalculation(int row, int column, bool recursive)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| row | Int32 | Radindex för den specifika cellen |
| column | Int32 | Kolumnindex för den specifika cellen. |
| recursive | Boolean | Om returnerar de beroende som inte refererar till den specifika cellen direkt utan refererar till andra blad i den cellen. |

### Returvärde

Enumerator för att räkna upp alla beroende (cellobjekt)

### Anmärkningar

För att använda den här metoden, se till att arbetsboken har ställts in med sant värde för [`EnableCalculationChain`](../../formulasettings/enablecalculationchain) och har beräknats helt med den här inställningen. Om det inte finns någon formelreferens till den här cellen kommer null att returneras. För mer information och exempel, se[`GetDependentsInCalculation`](../../cell/getdependentsincalculation)
