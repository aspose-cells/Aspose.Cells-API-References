## Cells.GetEnumerator method

Hämtar celluppräkningen.

```csharp
public IEnumerator GetEnumerator()
```

### Returvärde

Cellenumeratorn

### Anmärkningar

Vid genomkörning av element av den returnerade Enumeratorn bör inte cellsamlingen modifieras (såsom operationer som kommer att göra att ny cell/rad instansieras eller befintlig cell/rad tas bort). Annars kanske uppräknaren inte kan gå igenom alla celler korrekt. (vissa element kan korsas upprepade gånger eller hoppa över).

### Exempel

```csharp
[C#]
Workbook workbook = new Workbook("template.xlsx");
Cells cells = workbook.Worksheets[0].Cells;

IEnumerator en = cells.GetEnumerator();
while (en.MoveNext())
{
    Cell cell = (Cell)en.Current;
    Console.WriteLine(cell.Name + ": " + cell.Value);
}
```
