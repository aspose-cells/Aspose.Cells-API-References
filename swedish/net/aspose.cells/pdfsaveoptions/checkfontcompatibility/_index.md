## PdfSaveOptions.CheckFontCompatibility property

Indikerar om teckensnittskompatibilitet ska kontrolleras för varje tecken i text.

```csharp
public bool CheckFontCompatibility { get; set; }
```

### Anmärkningar

Standardvärdet är sant. Inaktivera den här egenskapen kan ge bättre prestanda. Men när standardtypsnittet eller specificerat teckensnitt för text/tecken inte kan användas för att återge det, kan oläsbara tecken (som block) förekomma i den genererade pdf-filen. För en sådan situation bör användaren behålla denna egenskap som sann så att alternativt teckensnitt kan sökas och användas för att återge texten istället;
