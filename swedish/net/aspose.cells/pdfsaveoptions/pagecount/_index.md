## PdfSaveOptions.PageCount property

Hämtar eller ställer in antalet sidor som ska sparas.

```csharp
public int PageCount { get; set; }
```

### Anmärkningar

Standard är System.Int32.MaxValue vilket innebär att alla sidor kommer att renderas..

### Exempel

Följande exempel visar hur man renderar en rad sidor (3 och 4) i en Microsoft Excel-fil till PDF.

```csharp
//Öppna en Excel-fil
Workbook wb = new Workbook("Book1.xlsx");

PdfSaveOptions options = new PdfSaveOptions();

//Skriv endast ut sida 3 och sida 4 i utdata-PDF-filen
//Startsidesindex (0-baserat index)
options.PageIndex = 3;
//Antal sidor som ska skrivas ut
options.PageCount = 2;

//Spara PDF-filen
wb.Save("output.pdf", options);
```
