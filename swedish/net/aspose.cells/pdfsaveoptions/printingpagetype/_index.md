## PdfSaveOptions.PrintingPageType property

Indikerar vilka sidor som inte kommer att skrivas ut.

```csharp
public PrintingPageType PrintingPageType { get; set; }
```

### Anmärkningar

Om innehållet i arket är sparsamt, kommer det att finnas några sidor som är helt tomma i den utgående pdf-filen. Om du inte vill ha dessa tomma sidor kan du använda det här alternativet för att utelämna dem.

### Exempel

Följande kod utesluter tomma sidor eller sidor som bara innehåller en del stilinnehåll som cellbakgrund, kanter.

```csharp
Workbook wb = new Workbook("Book1.xlsx");

PdfSaveOptions pdfSaveOptions = new PdfSaveOptions();

//ignorera tomma sidor
pdfSaveOptions.PrintingPageType = PrintingPageType.IgnoreBlank;

wb.Save("output_ignore_blank_page.pdf", pdfSaveOptions);


//ignorera tomma sidor and pages which only contains some style content like cell background
pdfSaveOptions.PrintingPageType = PrintingPageType.IgnoreStyle;

wb.Save("output_ignore_blank_and_style_page.pdf", pdfSaveOptions);
```
