## LoadFilter.StartSheet method

Förbereder filteralternativ innan ett givet kalkylblad laddas. Användarens implementering av LoadFilter kan ändra LoadDataFilterOptions here för att ange hur man laddar data för detta kalkylblad.

```csharp
public virtual void StartSheet(Worksheet sheet)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| sheet | Worksheet | Kalkylbladet som ska laddas. Det finns bara ett fåtal egenskaper som kan användas för det givna kalkylbladsobjektet here eftersom de flesta data och egenskaper inte har laddats. De tillgängliga egenskaperna är: Namn, Index, VisibilityType |
