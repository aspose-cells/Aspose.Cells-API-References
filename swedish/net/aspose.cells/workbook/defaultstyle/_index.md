## Workbook.DefaultStyle property

Hämtar eller ställer in standard[`Style`](../../style) objekt för arbetsboken.

```csharp
public Style DefaultStyle { get; set; }
```

### Anmärkningar

Egenskapen DefaultStyle är användbar för att implementera en stil för hela arbetsboken.

### Exempel

Följande kod skapar och instansierar en ny arbetsbok och anger en standard[`Style`](../../style) till den.

```csharp
[C#]
Workbook workbook = new Workbook();
Style defaultStyle = workbook.DefaultStyle;
defaultStyle.Font.Name = "Tahoma";
workbook.DefaultStyle = defaultStyle;

[Visual Basic]
Dim workbook as Workbook = new Workbook()
Dim defaultStyle as Style = workbook.DefaultStyle
defaultStyle.Font.Name = "Tahoma"
workbook.DefaultStyle = defaultStyle
```
