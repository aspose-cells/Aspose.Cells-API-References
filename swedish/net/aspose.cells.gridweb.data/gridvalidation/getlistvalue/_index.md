## GridValidation.GetListValue method

Hämta värdet för lista över valideringen för den angivna cellen.


| row | Int32 | Radindex. |

### Returvärde

Värdet för att producera listan över denna validering för den angivna cellen. Om listan hänvisar till ett område, kommer det returnerade värdet att vara ett ReferredArea-objekt; Annars kan det returnerade värdet vara null, object[] eller enkelt objekt.

### Anmärkningar

Endast för validering vars typ är List och har tillämpats på en given cell, annars kommer null att returneras.
