## PivotPageFields.AddIdentify method

Anger vilken objektetikett i varje sidfält som ska användas för att identifiera dataintervallet. pageItemIndex.Length måste vara lika med PageFieldCount, så lägg till sidfältet först.

```csharp
public void AddIdentify(int rangeIndex, int[] pageItemIndex)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| rangeIndex | Int32 | Konsolideringsdataintervallsindex. |
| pageItemIndex | Int32[] | Sidobjektindexet i varje sidfält. pageItemIndex[2] = 1 betyder det andra objektet i det tredje fältet som ska användas för att identifiera detta intervall. pageItemIndex[1] = -1 betyder inget objekt i det andra fältet att använda för att identifiera detta intervall och MS kommer automatiskt att skapa "tomt" objekt i det andra fältet för att identifiera detta intervall. |
