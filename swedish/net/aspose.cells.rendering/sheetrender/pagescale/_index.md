## SheetRender.PageScale property

Får beräknad sidskala för arket. Returnerar den inställda skalan om[`Zoom`](../../../aspose.cells/pagesetup/zoom) är inställd. I annat fall returnerar den beräknade skalan enl[`FitToPagesWide`](../../../aspose.cells/pagesetup/fittopageswide) och[`FitToPagesTall`](../../../aspose.cells/pagesetup/fittopagestall) .

```csharp
public double PageScale { get; }
```

### Exempel

```csharp
Workbook wb = new Workbook("Book1.xlsx");

SheetRender sheetRender = new SheetRender(wb.Worksheets[0], new ImageOrPrintOptions());

//Får beräknad sidskala på arket.
double pageScale = sheetRender.PageScale;
```
