## GridCell.Protected property

Indikerar om cellen är skyddad. Om värdet är "true" kan användaren inte ändra cellen via användargränssnittet. Det här attributet har inget att göra med Style.CellLocked-egenskapen och kommer inte att sparas i filen när rutnätsdata exporterad. Standardvärdet är "false".

public bool Protected { get; set; }
