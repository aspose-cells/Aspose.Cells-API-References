## ShapeCollection.AddRadioButton method

Lägger till en RadioButton i arbetsbladet.

```csharp
public RadioButton AddRadioButton(int upperLeftRow, int top, int upperLeftColumn, int left, 
    int height, int width)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| upperLeftRow | Int32 | Övre vänstra radens index. |
| top | Int32 | Representerar den vertikala förskjutningen av RadioButton från dess vänstra rad, i pixelenhet. |
| upperLeftColumn | Int32 | Övre vänstra kolumnindex. |
| left | Int32 | Representerar den horisontella förskjutningen av RadioButton från dess vänstra kolumn, i pixelenhet. |
| height | Int32 | Representerar höjden på RadioButton, i pixelenhet. |
| width | Int32 | Representerar bredden på RadioButton, i pixelenhet. |

### Returvärde

Ett RadioButton-objekt.
