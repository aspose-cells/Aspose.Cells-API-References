## ShapeCollection.AddRectangle method

Lägger till en RectangleShape till kalkylbladet.

```csharp
public RectangleShape AddRectangle(int upperLeftRow, int top, int upperLeftColumn, int left, 
    int height, int width)
```

| Parameter | Typ | Beskrivning |
| --- | --- | --- |
| upperLeftRow | Int32 | Övre vänstra radens index. |
| top | Int32 | Representerar den vertikala förskjutningen av RectangleShape från dess vänstra rad, i pixelenhet. |
| upperLeftColumn | Int32 | Övre vänstra kolumnindex. |
| left | Int32 | Representerar den horisontella förskjutningen av RectangleShape från dess vänstra kolumn, i pixelenhet. |
| height | Int32 | Representerar höjden på RectangleShape, i pixelenhet. |
| width | Int32 | Representerar bredden på RectangleShape, i pixelenhet. |

### Returvärde

Ett RectangleShape-objekt.

### Exempel

```csharp

[C#]
// lägg till en rektangel
RectangleShape rectangleShape = shapes.AddRectangle(2, 0, 2, 0, 130, 130);
```
