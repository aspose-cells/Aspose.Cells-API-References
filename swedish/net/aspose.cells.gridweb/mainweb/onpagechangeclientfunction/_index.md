## MainWeb.OnPageChangeClientFunction property

Hämtar eller ställer in klientsidans funktion så att den anropas efter att sidindex har ändrats. träder endast i kraft när EnablePaging är sant. Klientfunktionen ska deklareras så här: function MyOnPageChange(index) { console.log("nuvarande sida är:"+index); } Obs: Du kan använda "den här"-pekaren i klientfunktionen för att peka på rutnätskontrollen som utlöser händelsen.
