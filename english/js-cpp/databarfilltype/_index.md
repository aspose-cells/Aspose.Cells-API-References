## DataBarFillType enumeration
Specifies how a data bar is filled with color.
### Values
| Name | Value | Description |
| --- | --- | --- |
| Solid | `0` | The data bar is filled with solid color. |
| Gradient | `1` | The data bar is filled with a color gradient. |
