## BulletType enumeration
Represents the type of the bullet.
### Values
| Name | Value | Description |
| --- | --- | --- |
| None | `0` | No bullet. |
| Character | `1` | Character bullet. |
| Picture | `2` | Image bullet. |
| AutoNumbered | `3` | Automatic numbered bullet. |
