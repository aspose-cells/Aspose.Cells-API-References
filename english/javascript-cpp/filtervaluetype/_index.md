## FilterValueType enumeration
Represents type of filter value.
### Values
| Name | Value | Description |
| --- | --- | --- |
| IsDateTime | `4` | The filter value is datetime. Corresponding value must be [DateTimeGroupItem](../datetimegroupitem/). |
| IsString | `8` | The filter is a label. Corresponding value must be string. |
