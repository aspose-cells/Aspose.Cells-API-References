## HtmlLayoutMode enumeration
Represents the layout mode for HTML rendering.
### Values
| Name | Value | Description |
| --- | --- | --- |
| Normal | `0` | Renders content like MS Excel. |
| Print | `1` | Renders content in print layout mode. |
