## TextCapsType enumeration
This type specifies the cap types of the text.
### Remarks
Only can be applied to chart and shape.
### Values
| Name | Value | Description |
| --- | --- | --- |
| None | `0` | None caps |
| All | `1` | Apply all caps on the text. |
| Small | `2` | Apply small caps to the text. |
