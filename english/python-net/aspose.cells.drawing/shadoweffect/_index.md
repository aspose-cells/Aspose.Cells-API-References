## ShadowEffect class

This class specifies the shadow effect of the chart element or shape.

The ShadowEffect type exposes the following members:
## Properties
| Name | Description |
| :- | :- |
|preset_type|Gets and sets the preset shadow type of the shadow.|
|color|Gets and sets the color of the shadow.|
|transparency|Gets and sets the degree of transparency of the shadow. Range from 0.0 (opaque) to 1.0 (clear).|
|size|Gets and sets the size of the shadow. Range from 0 to 2.0. <br/>            Meaningless in inner shadow.|
|blur|Gets and sets the blur of the shadow. Range from 0 to 100 points.|
|angle|Gets and sets the lighting angle. Range from 0 to 359.9 degrees.|
|distance|Gets and sets the distance of the shadow. Range from 0 to 200 points.|
