## AbstractCalculationEngine class

Represents user's custom calculation engine to extend the default calculation engine of Aspose.Cells.

The AbstractCalculationEngine type exposes the following members:

|is_param_literal_required|Indicates whether this engine needs the literal text of parameter while doing calculation. Default value is false.|
|process_built_in_functions|Whether built-in functions that have been supported by the built-in engine should be checked and processed by this implementation.<br/>            Default is false.<br/>            If user needs to change the calculation logic of some built-in functions, this property should be set as true.|
