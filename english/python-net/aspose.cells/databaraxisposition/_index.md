## DataBarAxisPosition enumeration

Specifies the axis position for a range of cells with conditional formatting as data bars.

## Members
| Member name | Description |
| :- | :- |
|AUTOMATIC|Display the axis at a variable position based on the ratio of the minimum negative value to the maximum positive value in the range. <br/>            Positive values are displayed in a left-to-right direction. <br/>            Negative values are displayed in a right-to-left direction. <br/>            When all values are positive or all values are negative, no axis is displayed.|
|MIDPOINT|Display the axis at the midpoint of the cell regardless of the set of values in the range. <br/>            Positive values are displayed in a left-to-right direction. <br/>            Negative values are displayed in a right-to-left direction.|
|NONE|No axis is displayed, and both positive and negative values are displayed in the left-to-right direction.|
