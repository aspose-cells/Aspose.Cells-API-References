## ShiftType enumeration

Represent the shift options when deleting a range of cells.

## Members
| Member name | Description |
| :- | :- |
|DOWN|Shift cells down.|
|LEFT|Shift cells left.|
|NONE|Do not shift cells.|
|RIGHT|Shift cells right.|
|UP|Shift cells up.|
