## AboveAverage class

Describe the AboveAverage conditional formatting rule. <br/>            This conditional formatting rule highlights cells that<br/>            are above or below the average for all values in the range.

The AboveAverage type exposes the following members:

| Name | Description |
| :- | :- |

## Properties
| Name | Description |
| :- | :- |
|is_above_average|Get or set the flag indicating whether the rule is an "above average" rule. <br/>            'true' indicates 'above average'.<br/>            Default value is true.|
|is_equal_average|Get or set the flag indicating whether the 'aboveAverage' and 'belowAverage' criteria <br/>            is inclusive of the average itself, or exclusive of that value. <br/>            'true' indicates to include the average value in the criteria.<br/>            Default value is false.|
|std_dev|Get or set the number of standard deviations to include above or below the average in the<br/>            conditional formatting rule. <br/>            The input value must between 0 and 3 (include 0 and 3). <br/>            Setting this value to 0 means stdDev is not set.<br/>            The default value is 0.|
