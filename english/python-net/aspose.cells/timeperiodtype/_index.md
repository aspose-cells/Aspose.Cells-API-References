## TimePeriodType enumeration

Used in a FormatConditionType.TimePeriod conditional formatting rule. <br/>            These are dynamic time periods, which change based on<br/>            the date the conditional formatting is refreshed / applied.

## Members
| Member name | Description |
| :- | :- |
|TODAY|Today's date.|
|YESTERDAY|Yesterday's date.|
|TOMORROW|Tomorrow's date.|
|LAST_7_DAYS|A date in the last seven days.|
|THIS_MONTH|A date occurring in this calendar month.|
|LAST_MONTH|A date occurring in the last calendar month.|
|NEXT_MONTH|A date occurring in the next calendar month.|
|THIS_WEEK|A date occurring this week.|
|LAST_WEEK|A date occurring last week.|
|NEXT_WEEK|A date occurring next week.|
|THIS_YEAR|A date occurring this year.<br/>            Only for .ods.|
|LAST_YEAR|A date occurring last year.<br/>            Only for .ods.|
|NEXT_YEAR|A date occurring next year.<br/>            Only for .ods.|
