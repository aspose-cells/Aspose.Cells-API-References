## NegativeBarFormat class

Represents the color settings of the data bars for negative values that are defined by a data bar conditional formatting rule.

The NegativeBarFormat type exposes the following members:
## Properties
| Name | Description |
| :- | :- |
|border_color|Gets or sets a FormatColor object that you can use to specify the border color for negative data bars.|
|border_color_type|Gets whether to use the same border color as positive data bars.|
|color|Gets or sets a FormatColor object that you can use to specify the fill color for negative data bars.|
|color_type|Gets or sets whether to use the same fill color as positive data bars.|
