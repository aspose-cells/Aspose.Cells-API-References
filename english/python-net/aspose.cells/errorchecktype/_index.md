## ErrorCheckType enumeration

Represents all error check type.

## Members
| Member name | Description |
| :- | :- |
|CALC|check for calculation errors|
|EMPTY_CELL_REF|check for references to empty cells|
|TEXT_NUMBER|check the format of numeric values|
|INCONSIST_RANGE|check formulas with references to less than the entirety<br/>             of a range containing continuous data|
|INCONSIST_FORMULA|check formulas that are inconsistent with formulas in neighboring cells.|
|TEXT_DATE|check the format of date/time values|
|UNPROCTED_FORMULA|check for unprotected formulas|
|VALIDATION|whether to perform data validation|
|CALCULATED_COLUMN|Ignore errors when cells contain a value different from a calculated column formula.|
