Contains classes for work with various externalconnections in Microsoft Excel presentations.

## Classes
| Class | Description |
| :- | :- |
|[ConnectionParameter](/python-net/aspose.cells.externalconnections/connectionparameter/)|Specifies properties about any parameters used with external data connections<br/>            Parameters are valid for ODBC and web queries.|
|[ConnectionParameterCollection](/python-net/aspose.cells.externalconnections/connectionparametercollection/)|Specifies the [ConnectionParameter](/python-net/aspose.cells.externalconnections/connectionparameter/) collection|
|[DBConnection](/python-net/aspose.cells.externalconnections/dbconnection/)|Specifies all properties associated with an ODBC or OLE DB external data connection.|
|[DataModelConnection](/python-net/aspose.cells.externalconnections/datamodelconnection/)|Specifies a data model connection|
|[ExternalConnection](/python-net/aspose.cells.externalconnections/externalconnection/)|Specifies an external data connection|
|[ExternalConnectionCollection](/python-net/aspose.cells.externalconnections/externalconnectioncollection/)|Specifies the [ExternalConnection](/python-net/aspose.cells.externalconnections/externalconnection/) collection|
|[WebQueryConnection](/python-net/aspose.cells.externalconnections/webqueryconnection/)|Specifies the properties for a web query source. A web query will retrieve data from HTML tables, <br/>             and can also supply HTTP "Get" parameters to be processed by the web server in generating the HTML by <br/>             including the parameters and parameter elements.|
