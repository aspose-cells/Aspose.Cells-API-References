## ShapeDisplayType enum


Represents how the application displays shapes in this workbook. Shapes might include charts, images, and other object data that the application supports.

```cpp
enum class ShapeDisplayType
```

### Values

| Name | Value | Description |
| --- | --- | --- |
| All | 0 | <br>Indicates that all shapes be shown in the workbook. |
| Placeholders | 1 | <br>Indicates that the application show placeholders for shapes in the workbook. |
| None | 2 | <br>Indicates that all shapes be hidden in the workbook. |
