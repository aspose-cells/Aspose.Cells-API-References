## Examples


```cpp
Aspose::Cells::Startup();
//Initialize a new workbook.
Workbook book;

//Add a shape.(e.g rectangle)
Shape shape = book.GetWorksheets().Get(0).GetShapes().AddRectangle(2, 0, 2, 0, 130, 130);


//Save the excel file.
book.Save(u"exmaple.xlsx");
Aspose::Cells::Cleanup();
```
