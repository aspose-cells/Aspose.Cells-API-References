## Examples


```cpp
Aspose::Cells::Startup();

Workbook workbook;

//get ShapeCollection
ShapeCollection shapes = workbook.GetWorksheets().Get(0).GetShapes();


//Save the excel file.
workbook.Save("result.xlsx");
Aspose::Cells::Cleanup();
```
