## FillFormat::SetPresetColorGradient method


Sets the specified fill to a preset-color gradient. Only applies for Excel 2007.

```cpp
void Aspose::Cells::Drawing::FillFormat::SetPresetColorGradient(GradientPresetType presetColor, GradientStyleType style, int32_t variant)
```


| Parameter | Type | Description |
| --- | --- | --- |
| presetColor | GradientPresetType | Preset color type |
| style | GradientStyleType | Gradient shading style. |
| variant | int32_t | The gradient variant. Can be a value from 1 through 4, corresponding to one of the four variants on the Gradient tab in the Fill Effects dialog box. If style is GradientStyle.FromCenter, the Variant argument can only be 1 or 2. |
