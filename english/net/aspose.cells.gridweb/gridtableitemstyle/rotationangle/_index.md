## GridTableItemStyle.RotationAngle property

Gets or sets Rotation attribute.

```csharp
public short RotationAngle { get; set; }
```

### Remarks

0: Not rotated.

255: Top to Bottom.

-90: Downward.

90: Upward.

You can set 255 or value ranged from -90 to 90.
