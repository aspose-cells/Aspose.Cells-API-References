## MainWeb.OnDoubleClickCellClientFunction property

Gets or sets the client side function to be called when a cell is double clicked. The client function should be declared like this: function MyOnDoubleClickCell(cell) { GridWeb1.setCellValueByCell(cell, "test"); }Note: You may use the "this" pointer in the client function to point the grid control which fires the event.

public string OnDoubleClickCellClientFunction { get; set; }
