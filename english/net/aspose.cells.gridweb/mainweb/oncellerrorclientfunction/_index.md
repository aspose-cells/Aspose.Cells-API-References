## MainWeb.OnCellErrorClientFunction property

Gets or sets the client side function name to be called when a cell's validation is failed. The client function should be declared like this: function MyOnCellError(cell) { alert(GridWeb1.getCellValueByCell(cell)); }Note: You may use the "this" pointer in the client function to point the grid control which fires the event.
