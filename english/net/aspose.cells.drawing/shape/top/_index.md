## Shape.Top property

Represents the vertical offset of shape from its top row, in unit of pixels.

```csharp
public int Top { get; set; }
```

### Remarks

If the shape is in the chart, represents the vertical offset of shape from its top border.

### Examples

```csharp

[C#]
if (shape.Top == 3)
    shape.Top = 1;
```
