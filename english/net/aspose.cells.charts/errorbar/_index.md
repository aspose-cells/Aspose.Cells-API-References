## ErrorBar class

Represents error bar of data series.

```csharp
public class ErrorBar : Line
```

## Properties

| Name | Description |
| --- | --- |
| [Amount](amount) { get; set; } | Represents amount of error bar.  The amount must be greater than and equal to zero. |
| [DisplayType](displaytype) { get; set; } | Represents error bar display type. |
| [MinusValue](minusvalue) { get; set; } | Represents negative error amount when error bar type is Custom. |
| [PlusValue](plusvalue) { get; set; } | Represents positive error amount when error bar type is Custom. |

```csharp
	aseries.YErrorBar.DisplayType = ErrorBarDisplayType.Minus;

	aseries.YErrorBar.Amount = 5;
}
```
