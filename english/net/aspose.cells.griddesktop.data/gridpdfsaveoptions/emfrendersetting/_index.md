Setting for rendering Emf metafile.

```csharp
public GridEmfRenderSetting EmfRenderSetting { get; set; }
```

### Remarks

EMF metafiles identified as "EMF+ Dual" can contain both EMF+ records and EMF records. Either type of record can be used to render the image, only EMF+ records, or only EMF records.
