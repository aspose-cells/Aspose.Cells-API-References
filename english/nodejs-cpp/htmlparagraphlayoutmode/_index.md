## HtmlParagraphLayoutMode enumeration
Specifies how HTML &lt;p&gt; elements are rendered when loading HTML.
### Values
| Name | Value | Description |
| --- | --- | --- |
| Normal | `0` | Treat &lt;p&gt; inside &lt;td&gt; as inline content in a single cell. |
| AsRow | `1` | Starts a new row for each &lt;p&gt; element. |
