## DisplayDrawingObjects enumeration
Represents whether and how to show objects in the workbook.
### Values
| Name | Value | Description |
| --- | --- | --- |
| DisplayShapes | `0` | Show all objects |
| Placeholders | `1` | Show placeholders |
| Hide | `2` | Hide all shapes. |
