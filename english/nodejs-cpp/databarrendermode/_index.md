## DataBarRenderMode enumeration
Represents the DataBar render mode.
### Values
| Name | Value | Description |
| --- | --- | --- |
| BackgroundColor | `1` | Renders the DataBar using background color. |
| Image | `2` | Renders the DataBar as image. |
