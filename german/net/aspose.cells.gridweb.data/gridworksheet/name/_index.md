## GridWorksheet.Name property

Ruft den Namen des Arbeitsblatts ab oder legt ihn fest.

```csharp
public string Name { get; set; }
```

### Bemerkungen

Die maximale Länge des Blattnamens beträgt 31. Und Sie können zwei Arbeitsblättern nicht denselben Namen (ohne Berücksichtigung der Groß-/Kleinschreibung) zuweisen. Beispielsweise können Sie "SheetName1" nicht auf das erste Arbeitsblatt und "SHEETNAME1" auf das zweite Arbeitsblatt festlegen.
