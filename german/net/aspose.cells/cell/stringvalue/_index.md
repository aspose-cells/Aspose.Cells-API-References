## Cell.StringValue property

Ruft den in der Zelle enthaltenen Zeichenfolgenwert ab. Wenn der Typ dieser Zelle eine Zeichenfolge ist, geben Sie den Zeichenfolgenwert selbst zurück. Für andere Zelltypen wird der formatierte Zeichenfolgenwert (formatiert mit dem angegebenen Stil dieser Zelle) zurückgegeben. Der formatierte Zellenwert ist derselbe wie der von Ihnen kann aus Excel kommen, wenn eine Zelle als Text kopiert wird (z. B. Zelle in Texteditor kopieren oder in CSV exportieren).

public string StringValue { get; }
