## HtmlSaveOptions.DefaultFontName property

Geben Sie den Namen der Standardschriftart zum Exportieren von HTML an, die Standardschriftart wird verwendet, wenn die Schriftart des Stils nicht vorhanden ist, Wenn diese Eigenschaft null ist, verwendet Aspose.Cells eine universelle Schriftart, die dieselbe Familie wie die Originalschriftart hat, der Standardwert ist null.

public string DefaultFontName { get; set; }
