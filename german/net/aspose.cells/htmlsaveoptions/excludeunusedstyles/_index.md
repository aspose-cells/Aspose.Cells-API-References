## HtmlSaveOptions.ExcludeUnusedStyles property

Gibt an, ob nicht verwendete Stile ausgeschlossen werden. Bei den generierten HTML-Dateien kann das Ausschließen nicht verwendeter Stile die Dateigröße verkleinern , ohne die visuellen Effekte zu beeinträchtigen. Der Standardwert dieser Eigenschaft ist also true. Wenn der Benutzer alle Stile in der Arbeitsmappe für das generierte HTML beibehalten muss (z. B. das Szenario, dass user die Arbeitsmappe später aus dem generierten HTML wiederherstellen muss), setzen Sie diese Eigenschaft bitte auf false .

```csharp
public bool ExcludeUnusedStyles { get; set; }
```
