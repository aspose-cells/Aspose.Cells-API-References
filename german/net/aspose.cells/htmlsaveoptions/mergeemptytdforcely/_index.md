## HtmlSaveOptions.MergeEmptyTdForcely property

Gibt an, ob leere TD-Elemente beim Exportieren der Datei in HTML zwangsweise zusammengeführt werden. Die Größe der HTML-Datei wird erheblich reduziert, nachdem der Wert auf true gesetzt wurde. Der Standardwert ist falsch. Wenn Sie die HTML-Datei in Excel importieren oder beim Speichern der Datei in HTML perfekte Gitterlinien exportieren möchten, behalten Sie bitte den Standardwert bei.
