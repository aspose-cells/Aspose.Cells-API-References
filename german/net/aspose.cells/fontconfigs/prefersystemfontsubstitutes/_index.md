## FontConfigs.PreferSystemFontSubstitutes property

Geben Sie an, ob Systemschriftersatz zuerst verwendet werden soll oder nicht, wenn eine Schriftart nicht angezeigt wird und der Ersatz dieser Schriftart nicht festgelegt ist. Beispielsweise wird unter Ubuntu die Schriftart „Arial“ im Allgemeinen durch „Liberation Sans“ ersetzt. Der Standardwert ist „false“. .

public static bool PreferSystemFontSubstitutes { get; set; }
