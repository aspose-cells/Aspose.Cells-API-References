## WorkbookSettings.IsVScrollBarVisible property

Ruft einen Wert ab oder legt einen Wert fest, der angibt, ob die generierte Tabelle eine vertikale Bildlaufleiste enthalten wird.

```csharp
public bool IsVScrollBarVisible { get; set; }
```

### Bemerkungen

Der Standardwert ist wahr.

### Beispiele

Der folgende Code macht die vertikale Bildlaufleiste für die Tabelle unsichtbar.

```csharp
[C#]
// Vertikale Bildlaufleiste der Excel-Datei ausblenden.
settings.IsVScrollBarVisible = false;

[Visual Basic]
' Blenden Sie die vertikale Bildlaufleiste der Excel-Datei aus.
settings.IsVScrollBarVisible = False
```
