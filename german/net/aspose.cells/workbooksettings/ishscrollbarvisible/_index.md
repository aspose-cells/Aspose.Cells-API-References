## WorkbookSettings.IsHScrollBarVisible property

Ruft einen Wert ab oder legt einen Wert fest, der angibt, ob die generierte Tabelle eine horizontale Bildlaufleiste enthalten wird.

```csharp
public bool IsHScrollBarVisible { get; set; }
```

### Bemerkungen

Der Standardwert ist wahr.

### Beispiele

Der folgende Code macht die horizontale Bildlaufleiste für die Tabelle unsichtbar.

```csharp
[C#]
// Ausblenden der horizontalen Bildlaufleiste der Excel-Datei.
settings.IsHScrollBarVisible = false;

[Visual Basic]
' Blenden Sie die horizontale Bildlaufleiste der Excel-Datei aus.
settings.IsHScrollBarVisible = False
```
