
| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| addInFile | String | die Datei enthält die Zusatzfunktionen |
| functionName | String | der Add-in-Funktionsname |
| lib | Boolean | ob sich die angegebene Add-In-Datei im Verzeichnis oder Unterverzeichnis der Arbeitsmappen-Add-In-Bibliothek befindet.
