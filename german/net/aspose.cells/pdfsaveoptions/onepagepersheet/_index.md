## PdfSaveOptions.OnePagePerSheet property

Wenn OnePagePerSheet wahr ist, wird der gesamte Inhalt eines Blatts als Ergebnis nur auf einer Seite ausgegeben. Das Papierformat von pagesetup wird ungültig, und die anderen Einstellungen von pagesetup werden weiterhin wirksam.
