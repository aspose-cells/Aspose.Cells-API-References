## PdfSaveOptions.CheckFontCompatibility property

Gibt an, ob die Schriftkompatibilität für jedes Zeichen im Text überprüft werden soll.

```csharp
public bool CheckFontCompatibility { get; set; }
```

### Bemerkungen

Der Standardwert ist wahr. Deaktivieren Sie diese Eigenschaft, um eine bessere Leistung zu erzielen. Aber wenn die standardmäßige oder angegebene Schriftart von Text/Zeichen nicht zum Rendern verwendet werden kann, können unlesbare Zeichen (z. B. Blöcke) in der generierten PDF-Datei auftreten. Für solche Situationen sollte der Benutzer diese Eigenschaft auf true belassen, damit alternative Schriftarten gesucht und stattdessen zum Rendern des Textes verwendet werden können;
