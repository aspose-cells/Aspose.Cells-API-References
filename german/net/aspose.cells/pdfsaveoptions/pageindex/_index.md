## PdfSaveOptions.PageIndex property

Ruft den 0-basierten Index der ersten zu speichernden Seite ab oder legt ihn fest.

```csharp
public int PageIndex { get; set; }
```

### Bemerkungen

Standard ist 0.

### Beispiele

Das folgende Beispiel zeigt, wie ein Seitenbereich (3 und 4) in einer Microsoft Excel-Datei in PDF gerendert wird.

```csharp
//Eine Excel-Datei öffnen
Workbook wb = new Workbook("Book1.xlsx");

PdfSaveOptions options = new PdfSaveOptions();

// Nur Seite 3 und Seite 4 in der Ausgabe-PDF drucken
//Startseitenindex (0-basierter Index)
options.PageIndex = 3;
//Anzahl der zu druckenden Seiten
options.PageCount = 2;

//Speichern Sie die PDF-Datei
wb.Save("output.pdf", options);
```
