## OoxmlSaveOptions.ExportCellName property

Gibt an, ob der Zellenname in eine Excel2007-.xlsx-Datei (.xlsm, .xltx, .xltm) exportiert wird. Wenn auf die Ausgabedatei von SQL Server DTS zugegriffen werden kann, muss dieser Wert „true“ sein. Wenn Sie den Wert auf „false“ setzen, wird die Leistung erheblich gesteigert und die Dateigröße beim Erstellen großer Dateien verringert. Der Standardwert ist „true“.

public bool ExportCellName { get; set; }
