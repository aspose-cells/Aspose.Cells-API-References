## TxtLoadOptions.PreferredParsers property

Ruft bevorzugte Wertparser zum Laden von Textdateien ab und legt sie fest.

```csharp
public ICustomParser[] PreferredParsers { get; set; }
```

### Bemerkungen

parsers[0] ist der Parser, der für die erste Spalte in der Textvorlagendatei verwendet wird, parsers[1] ist der Parser, der für die zweite Spalte verwendet wird, ...etc. Der letzte (Parser[Parser. length-1]) wird für alle anderen Spalten verwendet, beginnend bei parsers.length-1. Wenn ein Element null ist, wird die entsprechende Spalte vom Standard-Parser von Aspose.Cells. analysiert.
