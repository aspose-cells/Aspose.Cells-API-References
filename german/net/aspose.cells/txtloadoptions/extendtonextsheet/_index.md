## TxtLoadOptions.ExtendToNextSheet property

Ob Daten auf das nächste Blatt erweitert werden, wenn die Zeilen oder Spalten der Daten das Limit überschreiten. Wenn diese Eigenschaft wahr ist, werden zusätzliche Daten auf das nächste Blatt hinter dem aktuellen erweitert (wenn das aktuelle Blatt das letzte ist, wird ein neues Blatt angehängt zur aktuellen Arbeitsmappe). Wenn diese Eigenschaft falsch ist, wird die Datenüberschreitung ignoriert. Standard ist falsch;

```csharp
public bool ExtendToNextSheet { get; set; }
```
