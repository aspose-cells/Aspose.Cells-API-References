## CopyOptions.ReferToSheetWithSameName property

Wenn Sie ein Arbeitsblatt in eine andere Arbeitsmappe kopieren und das Arbeitsblatt die Formeln enthält, die auf andere Arbeitsblätter in MS Excel verweisen, sollten sich die kopierten Formeln auf die Quellarbeitsmappe beziehen. Aber manchmal haben wir andere Arbeitsblätter kopiert und wir hoffen, dass die kopierten Formeln auf andere Arbeitsblätter mit dem Namen in derselben Arbeitsmappe verweisen, bitte setzen Sie diese Eigenschaft auf wahr.

```csharp
public bool ReferToSheetWithSameName { get; set; }
```

### Bemerkungen

Der Standardwert ist wahr.
