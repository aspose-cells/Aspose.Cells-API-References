## CopyOptions.ExtendToAdjacentRange property

Gibt an, ob Bereiche erweitert werden, wenn der Bereich in den benachbarten Bereich kopiert wird.

```csharp
public bool ExtendToAdjacentRange { get; set; }
```

### Bemerkungen
