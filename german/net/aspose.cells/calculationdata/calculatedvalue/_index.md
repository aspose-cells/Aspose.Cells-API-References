## CalculationData.CalculatedValue property

Ruft den berechneten Wert für diese Funktion ab oder legt ihn fest.

public object CalculatedValue { get; set; }

### Bemerkungen

Der Benutzer sollte diese Eigenschaft in seiner benutzerdefinierten Berechnungs-Engine für die Funktionen festlegen, die die Engine unterstützt, und der festgelegte Wert wird zurückgegeben, wenn diese Eigenschaft später abgerufen wird. Der festgelegte Wert kann ein beliebiger Wert dieser Objekte sein, die auf eine Zelle festgelegt werden können ( Cell.Value). Und es kann auch ein Array solcher Werte sein, oder ein Range, Name, ReferredArea. Wenn Sie diese Eigenschaft vor der Einstellung erhalten, wird die Funktion von der Standard-Berechnungsmaschine von Aspose.Cells berechnet und berechnet Wert wird zurückgegeben.
