### Beispiele

```csharp

[C#]

//Instanziieren eines Workbook-Objekts
Workbook workbook = new Workbook();
// Holen Sie sich die ersten Arbeitsblattzellen.
Cells cells = workbook.Worksheets[0].Cells;
cells["A1"].PutValue(1);
cells["A2"].PutValue(2);
Aspose.Cells.Range source = cells.CreateRange("A1:A2");
Aspose.Cells.Range target = cells.CreateRange("A3:A10");
// Füllen Sie 3,4,5 .... 10 in den Bereich A3: A10
source.AutoFill(target);
//Excel-Datei speichern
workbook.Save("book1.xlsm");
```
