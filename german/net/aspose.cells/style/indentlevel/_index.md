## Style.IndentLevel property

Repräsentiert die Einzugsebene für die Zelle oder den Bereich. Kann nur eine Ganzzahl von 0 bis 250 sein.

### Bemerkungen

Wenn der horizontale Textausrichtungstyp auf einen anderen Wert als links oder rechts eingestellt ist, wird die Einzugsebene auf Null zurückgesetzt.
