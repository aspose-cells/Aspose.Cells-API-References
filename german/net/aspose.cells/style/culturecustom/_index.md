## Style.CultureCustom property

Ruft die kulturabhängige Musterzeichenfolge für das Zahlenformat ab und legt sie fest. Wenn für dieses Objekt kein Zahlenformat festgelegt wurde, wird null zurückgegeben. Wenn das Zahlenformat eingebaut ist, wird die Musterzeichenfolge zurückgegeben, die der eingebauten Zahl entspricht.

```csharp
public string CultureCustom { get; set; }
```

### Bemerkungen

Für das integrierte Zahlenformat, sowohl der Musterinhalt (z. B. ein integriertes Datumsformat ist „m/d/y“ für einige Gebietsschemas, , aber für einige andere Gebietsschemas wird es „t/m/y“) und der Formatbezeichner (z. B. einige Gebietsschemas verwenden andere Zeichen als 'y', um den Jahresteil für die Datumsformatierung darzustellen) sind kulturabhängig; Für vom Benutzer angegebene benutzerdefinierte Formate werden nur Formatbezeichner entsprechend der Kultur geändert, andere Teile des Formatierungsmusters werden nicht geändert.
