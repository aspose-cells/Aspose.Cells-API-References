## LightCellsDataProvider.StartSheet method

Beginnt mit dem Speichern eines Arbeitsblatts.

```csharp
public bool StartSheet(int sheetIndex)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| sheetIndex | Int32 | Index des aktuellen Blattes, das gespeichert werden soll. |

### Rückgabewert

true, wenn dieser Anbieter Daten für das angegebene Blatt bereitstellt; false, wenn das angegebene Blatt sein normales Datenmodell (Cells) verwenden soll.

### Bemerkungen

Es wird zu Beginn des Speicherns eines Arbeitsblatts beim Speichern einer Arbeitsmappe aufgerufen. Wenn der Anbieter darauf verweisen muss

```csharp
sheetIndex
```

later in der startRow(Row)- oder startCell(Cell)-Methode, das heißt, wenn der Prozess wissen muss, welches Arbeitsblatt verarbeitet wird, sollte die Implementierung die beibehalten

```csharp
sheetIndex
```

Wert hier.
