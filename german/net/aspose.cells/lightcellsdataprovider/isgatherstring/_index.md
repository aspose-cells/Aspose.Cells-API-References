## LightCellsDataProvider.IsGatherString method

Überprüft, ob der aktuelle Stringwert der Zelle in einem globalen Pool gesammelt werden muss.

```csharp
public bool IsGatherString()
```

### Rückgabewert

wahr, wenn Zeichenfolgenwerte in einem globalen Pool für die resultierende Datei gesammelt werden müssen.

### Bemerkungen

Das Sammeln von String-Werten wird nur dann Vorteile bringen, wenn es viele duplizierte String-Werte für die von dieser Implementierung bereitgestellten Zellen gibt. In dieser Situation spart das Sammeln von Strings viel Speicher und erzeugt eine kleinere resultierende Datei. Wenn es viele String-Werte für die bereitgestellten Zellen gibt von LightCellsDataProvider, aber nur wenige von ihnen sind gleich, Das Sammeln von Zeichenfolgen kostet mehr Speicher und Zeit und hat keinen Vorteil für die resultierende Datei.
