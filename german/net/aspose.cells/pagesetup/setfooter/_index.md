## PageSetup.SetFooter method

Legt ein Skript fest, das die Fußzeile einer Excel-Datei formatiert.

```csharp
public void SetFooter(int section, string footerScript)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| section | Int32 | 0: Linker Abschnitt, 1: Mittlerer Abschnitt, 2: Rechter Abschnitt. |
| footerScript | String | Skript für das Fußzeilenformat. |

### Bemerkungen

Skriptbefehle:

**Befehl**

**Beschreibung**

&amp;P

Aktuelle Seitenzahl

&amp;N

Seitenzahl

&amp;D

Aktuelles Datum

&amp;T

Aktuelle Uhrzeit

&amp;EIN

Blattname

&amp;F

Dateiname ohne Pfad

&amp;"&lt;Schriftname&gt;"

Schriftartname, zum Beispiel: &amp;"Arial"

&amp;"&lt;Schriftname&gt;, &lt;Schriftstil&gt;"

Schriftname und Schriftstil, zum Beispiel: &amp;"Arial,Bold"

&amp;&lt;Schriftgröße&gt;

Schriftgröße. Folgt diesem Befehl eine einfache Zahl, die in der Kopfzeile ausgegeben werden soll, wird diese durch ein Leerzeichen von der Schrifthöhe getrennt.

&amp;K&lt;RRGGBB&gt;

Schriftfarbe, zum Beispiel (ROT): &amp;KFF0000

&amp;G

Bildskript

Zum Beispiel: „&amp;Arial,Bold&amp;8Footer Note“
