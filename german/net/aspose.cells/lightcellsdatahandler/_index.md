### Bemerkungen

Wenn Sie eine Arbeitsmappe in diesem Modus lesen,[`StartSheet`](./startsheet) wird beim Lesen jedes Arbeitsblatts in der Arbeitsmappe überprüft. Für ein Blatt, wenn[`StartSheet`](./startsheet) wahr ergibt, dann werden alle Daten und Eigenschaften von Zeilen/Zellen dieses Blattes überprüft und von der Implementierung dieser Schnittstelle verarbeitet. Für jede Reihe[`StartRow`](./startrow) wird aufgerufen, um zu prüfen, ob sie verarbeitet werden muss. Wenn eine Zeile verarbeitet werden muss, werden die Eigenschaften dieser Zeile zuerst gelesen und der Benutzer kann auf ihre Eigenschaften zugreifen[`ProcessRow`](./processrow). wenn Zeilenzellen auch verarbeitet werden müssen, dann[`ProcessRow`](./processrow) sollte true und dann zurückgeben[`StartCell`](./startcell) wird für jede vorhandene Zelle in dieser Zeile be aufgerufen, um zu prüfen, ob eine Zelle verarbeitet werden muss. Wenn eine Zelle verarbeitet werden muss, dann[`ProcessCell`](./processcell) wird aufgerufen, um die Zelle durch die Implementierung dieser Schnittstelle zu verarbeiten.
