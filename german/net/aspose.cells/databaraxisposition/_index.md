## DataBarAxisPosition enumeration

Gibt die Achsenposition für einen Bereich von Zellen mit bedingter Formatierung als Datenbalken an.

```csharp
public enum DataBarAxisPosition
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Automatic | `0` | Zeigt die Achse an einer variablen Position basierend auf dem Verhältnis des minimalen negativen Werts zum maximalen positiven Wert im Bereich an. Positive Werte werden von links nach rechts angezeigt. Negative Werte werden von rechts nach links angezeigt. Wenn alle Werte positiv oder alle Werte negativ sind, wird keine Achse angezeigt. |
| Midpoint | `1` | Zeigt die Achse am Mittelpunkt der Zelle an, unabhängig vom Wertesatz im Bereich. Positive Werte werden von links nach rechts angezeigt. Negative Werte werden von rechts nach links angezeigt. |
| None | `2` | Es wird keine Achse angezeigt und sowohl positive als auch negative Werte werden von links nach rechts angezeigt. |
