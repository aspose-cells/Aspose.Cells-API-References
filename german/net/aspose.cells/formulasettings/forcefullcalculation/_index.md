## FormulaSettings.ForceFullCalculation property

Gibt an, ob jedes Mal alle Formeln berechnet werden, wenn eine Berechnung ausgelöst wird.

```csharp
public bool ForceFullCalculation { get; set; }
```

### Bemerkungen

Diese Eigenschaft dient nur zum Speichern der Einstellungen in der resultierenden Tabellenkalkulationsdatei , damit andere Anwendungen (z. B. MS Excel) beim Laden und Bearbeiten der resultierenden Datei entsprechend handeln können. Aus Leistungsgründen berechnen wir für die Anwendungen der meisten Benutzer keine Formel automatisch in der Arbeitsmappe, unabhängig davon, welcher Wert für diese Eigenschaft festgelegt wurde.
