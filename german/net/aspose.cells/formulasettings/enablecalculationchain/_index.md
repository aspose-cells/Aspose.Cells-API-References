## FormulaSettings.EnableCalculationChain property

Ob Berechnungskette für Formeln aktiviert wird. Standard ist false.

```csharp
public bool EnableCalculationChain { get; set; }
```

### Bemerkungen

Wenn die Arbeitsmappe viele Formeln enthält und der Benutzer sie wiederholt berechnen muss und nur einen kleinen Teil davon ändern muss, kann es für die Leistung hilfreich sein, die Berechnungskette zu aktivieren. Andererseits, wenn die Kette aktiviert ist, Die Pflege des Kettenmodells erfordert zusätzlichen Speicher und es erfordert auch etwas mehr CPU-Zeit für einige andere Operationen wie das Ändern des Zellenwerts oder von Formeln. Nachdem diese Eigenschaft von "false" auf "true" geändert wurde, wird die Berechnungskette analysiert und erstellt die Zeit der ersten Berechnung für die Arbeitsmappe, daher kann die erforderliche Zeit für die erste Berechnung länger sein als die normale Berechnung ohne Kette.
