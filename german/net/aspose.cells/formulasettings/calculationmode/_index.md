## FormulaSettings.CalculationMode property

Ermittelt oder setzt den Modus für die Arbeitsmappenberechnung in MS Excel.

```csharp
public CalcModeType CalculationMode { get; set; }
```

### Bemerkungen

Diese Eigenschaft dient nur zum Speichern der Einstellungen in der resultierenden Tabellenkalkulationsdatei , damit andere Anwendungen (z. B. MS Excel) beim Laden und Bearbeiten der resultierenden Datei entsprechend handeln können. Aus Leistungsgründen für die meisten Benutzeranwendungen berechnen wir keine Formeln die Arbeitsmappe automatisch, unabhängig davon, welcher Modus für diese Eigenschaft eingestellt wurde. Wenn der Benutzer Formeln berechnen muss, rufen Sie bitte immer Methoden für verschiedene Objekte je nach Anforderung auf: [`CalculateFormula`](../../workbook/calculateformula) ,[`CalculateFormula`](../../worksheet/calculateformula) , [`Calculate`](../../cell/calculate) , ...etc.
