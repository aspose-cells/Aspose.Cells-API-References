## FormulaSettings.CalculateOnOpen property

Gibt an, ob die Anwendung beim Öffnen der Arbeitsmappe eine vollständige Berechnung durchführen muss.

```csharp
public bool CalculateOnOpen { get; set; }
```

### Bemerkungen

Diese Eigenschaft dient nur zum Speichern der Einstellungen in der resultierenden Tabellenkalkulationsdatei , damit andere Anwendungen (z. B. MS Excel) beim Laden der resultierenden Datei entsprechend handeln können. Aus Leistungsgründen für die Anwendungen der meisten Benutzer berechnen wir keine Formel in der Arbeitsmappe automatisch, unabhängig davon, welcher Wert für diese Eigenschaft festgelegt wurde.
