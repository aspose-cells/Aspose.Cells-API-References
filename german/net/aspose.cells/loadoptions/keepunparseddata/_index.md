## LoadOptions.KeepUnparsedData property

Ob die ungeparsten Daten für die Arbeitsmappe im Speicher bleiben, wenn sie aus der Vorlagendatei geladen wird. Standard ist wahr.

```csharp
public bool KeepUnparsedData { get; set; }
```

### Bemerkungen

Für Szenarien, in denen der Benutzer nur einige Inhalte aus der Vorlagendatei lesen muss und die Arbeitsmappe nicht zurückspeichern muss, kann diese Eigenschaft auf „false“ gesetzt werden, um die Leistung zu verbessern, insbesondere wenn sie zusammen mit einer Art LoadFilter verwendet wird,
