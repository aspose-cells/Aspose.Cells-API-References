## ExportDataTable(int, int, int, int) {#exportdatatable}

Exportiert Daten in die[`Cells`](../../cells) Sammlung zu aDataTable Objekt.

```csharp
public DataTable ExportDataTable(int firstRow, int firstColumn, int totalRows, int totalColumns)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| firstRow | Int32 | Die Zeilennummer der ersten Zelle, die exportiert werden soll. |
| firstColumn | Int32 | Die Spaltennummer der ersten Zelle, die exportiert werden soll. |
| totalRows | Int32 | Anzahl der zu importierenden Zeilen. |
| totalColumns | Int32 | Anzahl der zu importierenden Spalten. |

### Rückgabewert

ExportiertDataTable Objekt.

### Bemerkungen

Wenn Sie diese Methode verwenden, um einen Datenblock zu exportieren, achten Sie bitte darauf, dass die Daten in einer Spalte denselben Datentyp haben. Verwenden Sie andernfalls die[`ExportDataTableAsString`](../exportdatatableasstring)Methode statt.

### Beispiele

```csharp
[C#]


string designerFile = "List.xls";
Workbook excel = new Workbook(designerFile);
Worksheet sheet = excel.Worksheets[0];
DataTable dt = sheet.Cells.ExportDataTable(6, 1, 69, 4);

[Visual Basic]


Dim designerFile As String = "List.xls"
Dim excel As excel = New excel(designerFile)
Dim sheet As Worksheet = excel.Worksheets(0)
Dim dt As DataTable = sheet.Cells.ExportDataTable(6, 1, 69, 4)
```
