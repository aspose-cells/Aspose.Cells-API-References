## Cells.MultiThreadReading property

Ruft ab oder legt fest, ob das Zellendatenmodell Multi-Thread-Lesen unterstützen soll. Der Standardwert dieser Eigenschaft ist „false“.

```csharp
public bool MultiThreadReading { get; set; }
```

### Bemerkungen

Wenn es mehrere Threads zum gleichzeitigen Lesen von Row/Cell-Objekten in dieser Sammlung gibt, sollte diese Eigenschaft auf true gesetzt werden, andernfalls kann es zu unerwarteten Ergebnissen kommen. Die Unterstützung von Multi-Thread-Lesen kann die Leistung beim Zugriff auf Row/Cell-Objekte beeinträchtigen dieser Sammlung. Bitte beachten Sie, dass einige Funktionen das Multi-Thread-Lesen nicht unterstützen, wie das Formatieren von Werten (durch[`StringValue`](../../cell/stringvalue) ,[`DisplayStringValue`](../../cell/displaystringvalue) .etc.). Auch wenn diese Eigenschaft auf „true“ gesetzt ist, können diese APIs immer noch unerwartete Ergebnisse für das Multi-Thread-Lesen liefern.
