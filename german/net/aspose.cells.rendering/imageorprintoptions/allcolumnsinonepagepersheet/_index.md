## ImageOrPrintOptions.AllColumnsInOnePagePerSheet property

Wenn AllColumnsInOnePagePerSheet true ist, wird der gesamte Spalteninhalt eines Blatts als Ergebnis nur auf einer Seite ausgegeben. Die Breite des Papierformats von pagesetup wird ungültig, und die anderen Einstellungen von pagesetup werden weiterhin wirksam.
