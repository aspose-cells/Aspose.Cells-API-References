## SheetRender.PageScale property

Ruft den berechneten Seitenmaßstab des Blatts ab. Gibt den eingestellten Maßstab zurück, wenn[`Zoom`](../../../aspose.cells/pagesetup/zoom) eingestellt ist. Andernfalls wird die berechnete Skala gemäß zurückgegeben[`FitToPagesWide`](../../../aspose.cells/pagesetup/fittopageswide) und[`FitToPagesTall`](../../../aspose.cells/pagesetup/fittopagestall) .

```csharp
public double PageScale { get; }
```

### Beispiele

```csharp
Workbook wb = new Workbook("Book1.xlsx");

SheetRender sheetRender = new SheetRender(wb.Worksheets[0], new ImageOrPrintOptions());

// Ruft den berechneten Seitenmaßstab des Blatts ab.
double pageScale = sheetRender.PageScale;
```
