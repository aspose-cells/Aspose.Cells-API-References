### Bemerkungen

Jeder[`DocumentProperty`](../documentproperty) Objekt stellt eine benutzerdefinierte Eigenschaft eines Containerdokuments dar.

### Beispiele

```csharp

[C#]

//Instanziiere ein Workbook-Objekt
Workbook workbook = new Workbook("book1.xls");

//Eine Liste aller benutzerdefinierten Dokumenteigenschaften der Excel-Datei abrufen
CustomDocumentPropertyCollection customProperties = workbook.Worksheets.CustomDocumentProperties;

[VB.NET]

'Instanziieren Sie ein Workbook-Objekt
Dim workbook As New Workbook("book1.xls")

'Rufen Sie eine Liste aller benutzerdefinierten Dokumenteigenschaften der Excel-Datei ab
Dim customProperties As CustomDocumentPropertyCollection = workbook.Worksheets.CustomDocumentProperties

```
