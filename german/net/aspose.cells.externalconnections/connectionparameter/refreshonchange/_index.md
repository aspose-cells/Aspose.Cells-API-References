## ConnectionParameter.RefreshOnChange property

Flag, das angibt, ob die Abfrage automatisch aktualisiert werden soll, wenn sich der Inhalt einer -Zelle ändert, die den Parameterwert bereitstellt. Wenn wahr, werden externe Daten bei jeder Änderung mit dem neuen Parameterwert aktualisiert . Wenn falsch, werden externe Daten nur aktualisiert, wenn sie vom Benutzer angefordert werden, oder ein anderes Ereignis löst eine Aktualisierung aus (z. B. Arbeitsmappe geöffnet).

```csharp
public bool RefreshOnChange { get; set; }
```
