## ReConnectionMethodType enumeration

Gibt an, was die Tabellenkalkulationsanwendung tun soll, wenn eine Verbindung fehlschlägt.

```csharp
public enum ReConnectionMethodType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Required | `1` | Verwenden Sie bei der Aktualisierung die vorhandenen Verbindungsinformationen, und wenn sie ungültig werden, dann erhalten Sie aktualisierte Verbindungsinformationen, falls verfügbar aus der externen Verbindungsdatei. |
| Always | `2` | Holen Sie sich bei jeder Aktualisierung aktualisierte Verbindungsinformationen aus der externen Verbindungsdatei, falls verfügbar, und verwenden Sie diese anstelle der vorhandenen Verbindungsinformationen. In diesem Fall schlägt die Datenaktualisierung fehl, wenn die externe Verbindungsdatei nicht verfügbar ist. |
| Never | `3` | Erhalten Sie niemals aktualisierte Verbindungsinformationen von der externen Verbindungsdatei , selbst wenn sie verfügbar ist und selbst wenn die vorhandenen Verbindungsinformationen ungültig sind |
