## WebQueryConnection.IsXl2000 property

Dieses Flag dient der Abwärtskompatibilität mit älteren vorhandenen Tabellenkalkulationsdateien und wird auf gesetzt, wenn diese Webabfrage in einer Tabellenkalkulationsanwendung aktualisiert wurde, die neuer als oder gleich wie Microsoft Excel 2000 ist. Dies ist ein optionales Attribut, das ignoriert werden kann.

public bool IsXl2000 { get; set; }
