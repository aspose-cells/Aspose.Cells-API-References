## GridDesktop.EnableClipboardCopyPaste property

Gibt an, ob basierend auf der Zwischenablage kopiert/eingefügt werden soll, sodass es mit MS-EXCEL kopieren/einfügen kann. Es kopiert/fügt nur den Zellenwert ein, es kopiert keine anderen Einstellungen der Zelle wie Format, Rahmenstil usw. Der Standardwert ist false.

public bool EnableClipboardCopyPaste { get; set; }
