## Style.Color property

Ruft die Zellenschattierungsfarbe ab oder legt sie fest. Wenn das Muster keins ist, wird Color.Empty zurückgegeben; Wenn das Muster BackgroundType.Solid ist, wird ForegroundColor zurückgegeben; Andere werden BackgroundColor zurückgegeben
