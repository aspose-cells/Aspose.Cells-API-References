## Worksheet.AutoFitRowsMerged method

Passt alle Zeilen in diesem Arbeitsblatt automatisch an. Die Zeilenhöhe wird auch automatisch angepasst, wenn die Zellen in einer Zeile zusammengeführt werden.

```csharp
public void AutoFitRowsMerged()
```

### Bemerkungen

Wenn die Zellen in einer Zeile zusammengeführt werden und die Zeilen in MS Excel automatisch anpassen, MS Excel erweitert die Zeilenhöhe nicht. Aspose.Cells erweitert die Zeilenhöhe, um sie an die Daten der Zellen anzupassen.
