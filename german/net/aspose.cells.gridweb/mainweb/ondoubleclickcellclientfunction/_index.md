## MainWeb.OnDoubleClickCellClientFunction property

Ruft die Client-seitige Funktion ab oder legt sie fest, die aufgerufen werden soll, wenn auf eine Zelle doppelgeklickt wird. Die Client-Funktion sollte wie folgt deklariert werden: Funktion MyOnDoubleClickCell(Zelle) { GridWeb1.setCellValueByCell(cell, "test"); } Hinweis: Sie können den "this"-Zeiger in der Client-Funktion verwenden, um auf das Grid-Steuerelement zu zeigen, das das Ereignis auslöst.
