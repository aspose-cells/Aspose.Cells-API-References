## MainWeb.MaxColumn property

Ruft den maximalen Anzeigespaltenindex (nullbasiert) des Webblatts ab oder legt diesen fest. Das Steuerelement verwendet den größeren Wert von MaxColumn und der maximalen Spalte von Blattdaten.

```csharp
public int MaxColumn { get; set; }
```
