## MainWeb.OnContextMenuShowClientFunction property

Ruft die Client-seitige Funktion ab oder legt sie fest, die aufgerufen werden soll, wenn das Kontextmenü angezeigt wird. Die Client-Funktion sollte wie folgt deklariert werden: Funktion onContextMenuShow() { var menu = event.srcElement; menu.setItemVisibility("Löschen", "blockieren"); menu.setItemVisibility("Update", "none"); } Hinweis: Sie können den "this"-Zeiger in der Client-Funktion verwenden, um auf das Grid-Steuerelement zu zeigen, das das Ereignis auslöst.
