## MainWeb.MinRow property

Ruft den minimalen Anzeigezeilenindex (nullbasiert) des Webblatts ab oder legt diesen fest. Das Steuerelement verwendet den kleineren Wert von MinRow und die minimale Zeile der Blattdaten.

```csharp
public int MinRow { get; set; }
```

[C#]
