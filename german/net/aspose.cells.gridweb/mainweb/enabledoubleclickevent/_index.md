## MainWeb.EnableDoubleClickEvent property

Ruft ab oder legt fest, ob das kundenseitige Doppelklickereignis aktiviert werden soll.

```csharp
public bool EnableDoubleClickEvent { get; set; }
```

### Bemerkungen

Wenn es auf „true“ gesetzt ist, empfängt das Steuerelement kundenseitige Doppelklickereignisse und Sie können diese Ereignisse serverseitig behandeln. Andernfalls ignoriert das Steuerelement alle double-click -Ereignisse.
