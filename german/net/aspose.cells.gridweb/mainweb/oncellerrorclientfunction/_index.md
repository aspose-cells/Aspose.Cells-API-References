## MainWeb.OnCellErrorClientFunction property

Ruft den clientseitigen Funktionsnamen ab oder legt ihn fest, der aufgerufen werden soll, wenn die Validierung einer Zelle fehlschlägt. Die Clientfunktion sollte wie folgt deklariert werden: Funktion MyOnCellError(Zelle) { alert(GridWeb1.getCellValueByCell(cell)); } Hinweis: Sie können den "this"-Zeiger in der Client-Funktion verwenden, um auf das Grid-Steuerelement zu zeigen, das das Ereignis auslöst.
