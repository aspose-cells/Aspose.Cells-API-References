## ChartSplitType enumeration

Repräsentiert die Art und Weise, wie die zwei Abschnitte entweder eines Tortendiagramms oder eines Balkens eines Tortendiagramms geteilt werden.

```csharp
public enum ChartSplitType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Position | `0` | Stellt dar, dass die Datenpunkte zwischen dem Kreis und dem zweiten Diagramm aufgeteilt werden sollen, indem die letzte Teilungsposition der Datenpunkte in das zweite Diagramm eingefügt wird |
| Value | `1` | Stellt dar, dass die Datenpunkte zwischen dem Kreis und dem zweiten Diagramm aufgeteilt werden sollen, indem die Datenpunkte mit einem -Wert kleiner als die Teilungsposition in das zweite Diagramm eingefügt werden. |
| PercentValue | `2` | Stellt dar, dass die Datenpunkte zwischen dem Tortendiagramm und dem zweiten Diagramm aufgeteilt werden sollen, indem die Punkte mit einem -Prozentsatz kleiner als der Teilungspositionsprozentsatz in das -zweite Diagramm eingefügt werden. |
| Custom | `3` | Stellt dar, dass die Datenpunkte gemäß den benutzerdefinierten Split -Werten zwischen dem Kreis und dem zweiten Diagramm aufgeteilt werden sollen. |
| Auto | `4` | Stellt dar, dass die Datenpunkte unter Verwendung des default -Mechanismus für diesen Diagrammtyp geteilt werden sollen. |
