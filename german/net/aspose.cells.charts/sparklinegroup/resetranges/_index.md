## SparklineGroup.ResetRanges method

Setzt den Datenbereich und den Standortbereich der Sparkline-Gruppe zurück. Diese Methode löscht ursprüngliche Sparkline-Elemente in der Gruppe und erstellt neue Sparkline-Elemente für die neuen Bereiche.

```csharp
public void ResetRanges(string dataRange, bool isVertical, CellArea locationRange)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| dataRange | String | Gibt den neuen Datenbereich der Sparkline-Gruppe an. |
| isVertical | Boolean | Gibt an, ob die Sparklines aus dem neuen Datenbereich zeilen- oder spaltenweise gezeichnet werden sollen. |
| locationRange | CellArea | Gibt an, wo die Sparklines platziert werden sollen. |
