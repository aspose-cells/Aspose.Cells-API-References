## Chart.RotationAngle property

Stellt die Drehung der 3D-Diagrammansicht dar (die Drehung des Diagrammbereichs um die Z-Achse in Grad).

```csharp
public int RotationAngle { get; set; }
```

### Bemerkungen

Der Wert dieser Eigenschaft muss zwischen 0 und 360 liegen, außer bei 3D-Balkendiagrammen, wo der Wert zwischen 0 und 44 liegen muss. Der Standardwert ist 20. Gilt nur für 3D-Diagramme.
