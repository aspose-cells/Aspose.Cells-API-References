## Chart.PivotSource property

Die Quelle sind die Daten der PivotTable. Wenn PivotSource nicht leer ist, ist das Diagramm PivotChart.

```csharp
public string PivotSource { get; set; }
```

### Bemerkungen

Wenn die PivotTable „PivotTable1“ im Arbeitsblatt „Sheet1“ in der Datei „Book1.xls“. ist, könnte die PivotSource „[Book1.xls]Sheet1!PivotTable1“ sein, wenn sich das Diagramm und die PivotTable nicht in derselben Arbeitsmappe befinden. Wenn Sie diese Eigenschaft festlegen, geht die vorherige Datenquelleneinstellung verloren.
