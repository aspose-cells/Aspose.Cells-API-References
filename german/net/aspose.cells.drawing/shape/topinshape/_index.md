## Shape.TopInShape property

Repräsentiert den vertikalen Versatz der Form vom oberen Rand der übergeordneten Form, in Einheiten von 1/4000 der Höhe der übergeordneten Form.

```csharp
public int TopInShape { get; set; }
```

### Bemerkungen

Gilt nur, wenn diese Form in der Gruppe oder im Diagramm enthalten ist.

### Beispiele

```csharp

[C#]
if (shape.TopInShape == 8000)
    shape.TopInShape = 4000;
```
