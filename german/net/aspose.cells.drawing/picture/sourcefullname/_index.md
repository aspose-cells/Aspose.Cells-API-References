## Picture.SourceFullName property

Ruft den Pfad und Namen der Quelldatei für das verknüpfte Bild ab oder legt ihn fest.

```csharp
public string SourceFullName { get; set; }
```

### Bemerkungen

Der Standardwert ist ein leerer String. Wenn SourceFullName kein leerer String ist, wird das Bild verlinkt. Wenn SourceFullName kein leerer String ist, aber Data null ist, wird das Bild verlinkt und nicht in der Datei gespeichert.
