## Picture.Move method

```csharp
public void Move(int upperLeftRow, int upperLeftColumn)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| upperLeftRow | Int32 | Zeilenindex oben links. |
| upperLeftColumn | Int32 | Spaltenindex oben links. |

### Beispiele

```csharp

[C#]
//Instanziieren eines Workbook-Objekts
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
//Hinzufügen eines Bildes an der Position einer Zelle, deren Zeilen- und Spaltenindizes 1 im Arbeitsblatt sind. Es ist eine "B2"-Zelle
int imgIndex = worksheet.Pictures.Add(1, 1, "example.jpeg");
//Das eingefügte Bildobjekt abrufen
Picture pic = worksheet.Pictures[imgIndex];
//Setzen Sie den neuen Speicherort des Bildes
pic.Move(2, 4);
//Speichern Sie die Excel-Datei.
workbook.Save("result.xlsx");
```
