## Picture.Copy method

Kopieren Sie das Bild.

```csharp
public void Copy(Picture source, CopyOptions options)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| source | Picture | Das Quellbild. |
| options | CopyOptions | Die Kopieroptionen. |

### Beispiele

```csharp

[C#]
//Instanziieren eines Workbook-Objekts
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.Worksheets[0];
// erstes Bild einfügen
int imgIndex1 = worksheet.Pictures.Add(1, 1, "1.png");
//Das eingefügte Bildobjekt abrufen
Picture pic1 = worksheet.Pictures[imgIndex1];
// Zweites Bild einfügen
int imgIndex2 = worksheet.Pictures.Add(1, 9, "2.jpeg");
//Das eingefügte Bildobjekt abrufen
Picture pic2 = worksheet.Pictures[imgIndex2];
// Bild 1 auf Bild 2 kopieren. Sie erhalten zwei Bild 1-Objekte, die übereinander gelegt werden.
CopyOptions opt = new CopyOptions();
pic2.Copy(pic1, opt);
//Speichern Sie die Excel-Datei.
workbook.Save("result.xlsx");
```
