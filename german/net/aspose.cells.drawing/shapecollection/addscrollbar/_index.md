## ShapeCollection.AddScrollBar method

Fügt dem Arbeitsblatt eine Bildlaufleiste hinzu.

```csharp
public ScrollBar AddScrollBar(int upperLeftRow, int top, int upperLeftColumn, int left, int height, 
    int width)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| upperLeftRow | Int32 | Zeilenindex oben links. |
| top | Int32 | Stellt den vertikalen Versatz von ScrollBar von seiner linken Zeile in Pixeleinheiten dar. |
| upperLeftColumn | Int32 | Spaltenindex oben links. |
| left | Int32 | Stellt den horizontalen Versatz von ScrollBar von seiner linken Spalte in Pixeleinheiten dar. |
| height | Int32 | Stellt die Höhe von ScrollBar in Pixeleinheiten dar. |
| width | Int32 | Stellt die Breite von ScrollBar in Pixeleinheiten dar. |

### Rückgabewert

Ein ScrollBar-Objekt.

### Beispiele

