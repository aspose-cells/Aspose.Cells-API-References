## ShapeCollection.AddCopy method

Fügt dem Arbeitsblatt eine Form hinzu und kopiert sie.

```csharp
public Shape AddCopy(Shape sourceShape, int upperLeftRow, int top, int upperLeftColumn, int left)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| sourceShape | Shape | Quellform. |
| upperLeftRow | Int32 | Zeilenindex oben links. |
| top | Int32 | Repräsentiert den vertikalen Versatz des Kontrollkästchens von seiner linken Zeile in Pixeleinheiten. |
| upperLeftColumn | Int32 | Spaltenindex oben links. |
| left | Int32 | Repräsentiert den horizontalen Versatz des Textfelds von seiner linken Spalte in Pixeleinheiten. |

### Rückgabewert

Der neue Formobjektindex.

### Beispiele

```csharp

[C#]
// eine Form hinzufügen
RectangleShape rectangle = shapes.AddRectangle(2, 0, 2, 0, 130, 130);
//Kopieren
shapes.AddCopy(rectangle, 7, 0, 7, 0);
```
