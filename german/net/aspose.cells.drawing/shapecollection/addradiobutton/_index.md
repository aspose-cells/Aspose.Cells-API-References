## ShapeCollection.AddRadioButton method

Fügt dem Arbeitsblatt einen RadioButton hinzu.

```csharp
public RadioButton AddRadioButton(int upperLeftRow, int top, int upperLeftColumn, int left, 
    int height, int width)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| upperLeftRow | Int32 | Zeilenindex oben links. |
| top | Int32 | Stellt den vertikalen Abstand von RadioButton von seiner linken Zeile in Pixeleinheiten dar. |
| upperLeftColumn | Int32 | Spaltenindex oben links. |
| left | Int32 | Stellt den horizontalen Versatz von RadioButton von seiner linken Spalte in Pixeleinheiten dar. |
| height | Int32 | Repräsentiert die Höhe von RadioButton in Pixeleinheiten. |
| width | Int32 | Repräsentiert die Breite von RadioButton in Pixeleinheiten. |

### Rückgabewert
