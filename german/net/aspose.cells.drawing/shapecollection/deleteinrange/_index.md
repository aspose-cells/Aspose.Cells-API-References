## ShapeCollection.DeleteInRange method

Formen im Bereich löschen. Formkommentare werden nicht gelöscht.

```csharp
public void DeleteInRange(CellArea ca)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| ca | CellArea | Der Bereich. Wenn die Formen im Bereich enthalten sind, werden sie entfernt. |

### Beispiele

```csharp

[C#]
// erste Form hinzufügen
shapes.AddRectangle(2, 0, 2, 0, 50, 50);
// zweite Form hinzufügen
shapes.AddRectangle(6, 0, 2, 0, 30, 30);

CellArea area3 = new CellArea();
area3.StartColumn = 0;
area3.StartRow = 5;
area3.EndColumn = 5;
area3.EndRow = 8;

//löschen
shapes.DeleteInRange(area3);
```
