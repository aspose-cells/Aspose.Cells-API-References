## ShapeCollection.AddTextEffect method

Fügt ein WordArt-Objekt ein.

```csharp
public Shape AddTextEffect(MsoPresetTextEffect effect, string text, string fontName, int size, 
    bool fontBold, bool fontItalic, int upperLeftRow, int top, int upperLeftColumn, int left, 
    int height, int width)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| effect | MsoPresetTextEffect | Der voreingestellte mso-Texteffekttyp. |
| text | String | Der WordArt-Text. |
| fontName | String | Der Schriftartname. |
| size | Int32 | Die Schriftgröße |
| fontBold | Boolean | Gibt an, ob die Schriftart fett ist. |
| fontItalic | Boolean | Gibt an, ob die Schriftart kursiv ist. |
| upperLeftRow | Int32 | Zeilenindex oben links. |
| top | Int32 | Stellt den vertikalen Versatz der Form von der linken Zeile in Pixeleinheiten dar. |
| upperLeftColumn | Int32 | Spaltenindex oben links. |
| left | Int32 | Stellt den horizontalen Versatz der Form von der linken Spalte in Pixeleinheiten dar. |
| height | Int32 | Stellt die Höhe der Form in Pixeleinheiten dar. |
| width | Int32 | Stellt die Breite der Form in Pixeleinheiten dar. |

### Rückgabewert

Gibt ein Shape-Objekt zurück, das das neue WordArt-Objekt darstellt.

### Beispiele

```csharp

[C#]
// WordArt hinzufügen
Shape wordArt1 = shapes.AddTextEffect(MsoPresetTextEffect.TextEffect10, "WordArt", "arial", 18, false, false, 3, 0, 3, 0, 200, 50);
```
