## OnErrorActionType enumeration

Wird im CellError-Ereignis von GridWeb verwendet. Der Benutzer hat die OnErrorActionQuery.OnErrorAction festgelegt, um dem Steuerelement mitzuteilen, dass die Verarbeitung beendet oder der Fehler ignoriert werden soll.

```csharp
public enum OnErrorActionType
```

### Werte

| Name | Wert | Beschreibung |
| --- | --- | --- |
| Throw | `0` | Löst die aktuelle Ausnahme aus. |
| Stop | `1` | Stoppt die aktuelle Verarbeitung. |
| Ignore | `2` | Ignoriert den Fehler und setzt die Verarbeitung fort. |
