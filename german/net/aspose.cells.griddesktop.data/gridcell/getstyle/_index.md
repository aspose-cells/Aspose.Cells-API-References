## GridCell.GetStyle method

Ruft den Zellenstil ab. Wenn Sie den Stil ändern, sollten Sie die Methode „SetStyle“ aufrufen, um den Stil auf die Zelle festzulegen.

```csharp
public Style GetStyle()
```

### Rückgabewert

Kopie des Zellenstils zurückgeben.

### Beispiele

```csharp
[C#]
Style style = sheet.GetCell(0, 0).GetStyle();
style.CellLocked = true;
style.VAlignment = VerticalAlignmentType.Top;
...

Dim style As Style =  sheet.GetCell(0,0).GetStyle()
```
