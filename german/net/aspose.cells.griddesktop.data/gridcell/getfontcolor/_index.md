## GridCell.GetFontColor method

Ruft die Schriftfarbe der Zelle ab. Wenn Sie die Farbe ändern, sollten Sie die Methode "SetFontColor" aufrufen, um die Schriftfarbe auf die Zelle einzustellen.

```csharp
public Color GetFontColor()
```

### Rückgabewert

Schriftfarbe der Zelle zurückgeben.

### Beispiele

```csharp
[C#]
GridCell cell = gridDesktop1.GetActiveWorksheet().Cells[0, 0];
Font font = new Font("Courier New", 8, FontStyle.Italic);
cell.SetFont(font);
Color color = cell.GetFontColor();
color = Color.Black;
cell.SetFontColor(color);

[Visual Basic]
```
