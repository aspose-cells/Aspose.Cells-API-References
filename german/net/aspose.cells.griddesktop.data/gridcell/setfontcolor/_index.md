## GridCell.SetFontColor method

Legt die Schriftfarbe für die Zelle fest. Um die Leistung zu verbessern, implementieren Sie die Methode "SetFontColor", implementieren Sie die Eigenschaft "FontColor" nicht.

```csharp
public void SetFontColor(Color color)
```

| Parameter | Typ | Beschreibung |
| --- | --- | --- |
| color | Color | Schriftfarbe eingestellt werden. |

### Beispiele

```csharp
[C#]
GridCell cell = gridDesktop1.GetActiveWorksheet().Cells[0, 0];
Font font = new Font("Courier New", 8, FontStyle.Italic);
cell.SetFont(font);
Color color = cell.GetFontColor();
color = Color.Black;
cell.SetFontColor(color);
```
