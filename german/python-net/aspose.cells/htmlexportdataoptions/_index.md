##  HtmlExportDataOptions Aufzählung
Stellt die Optionen zum Exportieren von HTML-Daten dar.



Der Typ HtmlExportDataOptions macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| TABLE | Exportieren Sie die Datei in HTML, die nur den Tabellenteil enthält.|
| ALL | Exportieren Sie alle Daten in HTML.|
