##  set_style(style) {#Style}
Legt den Stil dieser Spalte fest.



```python
def set_style(self, style):
    ...
```


| Parameter| Typ| Beschreibung|
| :- | :- | :- |
| style | [Style](/cells/de/python-net/aspose.cells/style) | der Stil, der als Standardstil für Zellen in dieser Spalte verwendet werden soll.|
###  Bemerkungen

Diese Methode legt nur den angegebenen Stil als Standardstil für diese Spalte fest,
ohne die Stileinstellungen für vorhandene Zellen in dieser Spalte zu ändern.
Um die Stileinstellungen vorhandener Zellen gleichzeitig auf den angegebenen Stil zu aktualisieren,
Bitte verwenden Sie [Column.apply_style(style, flag)](/cells/de/python-net/aspose.cells/column/apply_style)
