##  get_named_ranges() {#}
Ruft alle vordefinierten benannten Bereiche in der Tabelle ab.


###  Kehrt zurück

Ein Array von Range-Objekten.
Wenn die Referenz des definierten Namens extern ist oder mehrere Bereiche hat, wird für diesen Namen kein Range-Objekt zurückgegeben.



Gibt null zurück, wenn der benannte Bereich nicht vorhanden ist.
