##  TxtLoadStyleStrategy Aufzählung
Gibt an, wie der Stil für geparste Werte angewendet wird, wenn der Zeichenfolgenwert in eine Zahl oder eine Datumszeit konvertiert wird.



Der Typ TxtLoadStyleStrategy macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| NONE | Legt keinen Stil für den geparsten Wert fest.|
| BUILT_IN |Legen Sie den Stil als integrierte Zahl/Datum/Uhrzeit fest, wenn die geparsten Werte einfache numerische/Datum/Uhrzeit-Werte sind.|
| EXACT_FORMAT | Legen Sie das genaue benutzerdefinierte Format für den geparsten Wert fest, damit der formatierte Wert mit dem ursprünglichen Eingabewert übereinstimmt.|
