##  HtmlCrossType Aufzählung
Stellt fünf Arten von HTML-Querzeichenfolgen dar.



Der Typ HtmlCrossType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| DEFAULT | Darstellung wie MS Excel, abhängig von der nächsten Zelle.<br/>Wenn die nächste Zelle null ist, kreuzt sich die Zeichenfolge oder sie wird abgeschnitten|
| MS_EXPORT | Zeigen Sie die Zeichenfolge wie MS Excel beim HTML-Export an.|
| CROSS | Zeigen Sie HTML Querzeichenfolge an, diese Leistung zum Erstellen großer HTML-Dateien ist mehr als zehnmal schneller als das Festlegen des Werts auf Standard oder FitToCell.|
| CROSS_HIDE_RIGHT | Zeigen Sie HTML Querzeichenfolge an und blenden Sie die rechte Zeichenfolge aus, wenn sich die Texte überschneiden.|
| FIT_TO_CELL | Zeigt die Zeichenfolge nur innerhalb der Breite der Zelle an.|
