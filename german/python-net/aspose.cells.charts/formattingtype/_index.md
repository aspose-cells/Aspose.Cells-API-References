##  FormattingType Aufzählung
Stellt den Formatierungstyp dar, der auf ein [Area](/cells/de/python-net/aspose.cells.drawing/area)-Objekt oder ein [Line](/cells/de/python-net/aspose.cells.drawing/line)-Objekt angewendet wird.



Der Typ FormattingType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| AUTOMATIC | Stellt den automatischen Formatierungstyp dar.|
| CUSTOM | Stellt den benutzerdefinierten Formatierungstyp dar.|
| NONE | Stellt keinen Formatierungstyp dar.|
