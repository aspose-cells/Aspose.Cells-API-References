##  input_range Eigentum

Ruft den Arbeitsblattbereich ab, der zum Füllen des angegebenen Kombinationsfelds verwendet wird, oder legt diesen fest.

###  Beispiele

```python

if shape.input_range == "$B$6:$B10":
```
