##  width_in_shape Eigentum

Stellt die Breite der Form in Einheiten von 1/4000 der übergeordneten Form dar.

###  Bemerkungen

Gilt nur, wenn sich dieses Shape in der Gruppe oder im Diagramm befindet.

###  Beispiele

```python

if shape.upper_delta_y == 2000:
    shape.upper_delta_y = 4000

```
