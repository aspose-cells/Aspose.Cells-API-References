##  ConnectionParameterType Aufzählung
Gibt den Parametertyp der externen Verbindung an



Der Typ ConnectionParameterType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| CELL | Rufen Sie bei jeder Aktualisierung den Parameterwert aus einer Zelle ab.|
| PROMPT | Fordern Sie den Benutzer bei jeder Aktualisierung zur Eingabe eines Parameterwerts auf.|
| VALUE | Verwenden Sie bei jeder Aktualisierung einen konstanten Wert für den Parameterwert.|
