##  CredentialsMethodType Aufzählung
Gibt die Anmeldedatenmethode an, die für den Serverzugriff verwendet wird.



Der Typ CredentialsMethodType macht die folgenden Member verfügbar:

###  Felder
| Feld| Beschreibung|
| :- | :- |
| INTEGRATED | Integrierte Authentifizierung|
| NONE | Keine Anmeldeinformationen|
| PROMPT | Eingabeaufforderung Anmeldeinformationen|
| STORED | Gespeicherte Anmeldeinformationen|
